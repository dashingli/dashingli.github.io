冬季护眼：南瓜最“养眼”

十一月 15, 2018 本文总阅读量次

&nbsp; &nbsp; &nbsp; 进入冬季，气温的降低，使得本来就不爱运动的那些人，更喜欢窝在家里了。周末除了睡觉就是电脑，这样下去眼睛哪受得了，很容易近视。这该怎么办呢？真不想出去走动呢？凉飕飕的……别急，小编今天就带来一个好方法：冬季护眼南瓜最“养眼”，南瓜可以帮你保护视力哦。

&nbsp; &nbsp; &nbsp; 据报道，常吃南瓜人群中，80岁以上的高龄者，仍能保持良好视力，不戴眼镜也可看报纸。
&nbsp; &nbsp; &nbsp; 南瓜有抗氧化、防衰老、护视力的作用，因为其含有丰富的维生素E和β-胡萝卜素。β-胡萝卜素在肌体内能转变为对人体具有重要生理作用的维生素A，对保护视力、预防眼疾、维持人体上皮组织的健康、促进儿童的生长发育等均有重要作用，同时对维持正常视觉和明目、促进骨骼发育及肌体抗癌防癌方面具有重要生理作用。
&nbsp; &nbsp; &nbsp;&nbsp;作为食物中维生素A的来源，南瓜食品与动物性食品(如肝脏、蛋黄等)相比，经济实惠。
&nbsp; &nbsp; &nbsp; 其实南瓜不仅可以保护视力，还有其他功效。
&nbsp; &nbsp; &nbsp; 常吃南瓜有助于防治糖尿病，因为南瓜中含有丰富的微量元素钴和果胶。钴的含量较高，是其他任何蔬菜都不可相比的，它是胰岛细胞合成胰岛素所必需的微量元素。果胶则可延缓肠道对糖和脂质吸收。
