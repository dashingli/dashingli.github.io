别让近视阻挡你的未来

五月 18, 2022 本文总阅读量次

&nbsp; &nbsp; &nbsp; 爱大爱根据世界卫生组织最新研究报告显示做出总结，目前我国近视人数多达6亿人。据统计，小中大学生近视率为：小学生45.7%，初中生74.4%，高中生83.3%，大学生则高达87.7%，青少年近视率高居世界第一。世界卫生组织已把近视眼列为致盲性眼病。
&nbsp; &nbsp; &nbsp; 近视除了给人们的日常生活带来不便之外，还会造成学习效率下降、影响学习成绩，限制高考、择业（在我国，飞行、航海、医学均对视力有严格要求，而且近视人群在择业、参军等均受到限制），影响形象、耽误恋爱，高度近视遗传后代等不良影响。
&nbsp; &nbsp; &nbsp;&nbsp;更可怕的是，近视还有可能引起一系列的并发症，包括眼睛的角膜、巩膜病变、晶体病变、玻璃体病变、黄斑区脉络、青光眼等，严重的甚至会导致失明，危害极大。
&nbsp; &nbsp; &nbsp; 所以，在日常生活中，呵护好我们的视力健康，养成良好的用眼习惯是非常重要的。比如说，读书、写字、看电脑的时候距离书本、电脑屏幕3035厘米左右。学习、工作一段时间后，要起身活动一下，到户外或窗前远眺35分钟，缓解视力疲劳。
&nbsp; &nbsp; &nbsp;&nbsp;而对于儿童青少年来说，当出现看黑板模糊、重影，看书时间长后视力疲劳、头痛等症状后，要及时到医院检查一下是不是近视、散光。
&nbsp; &nbsp; &nbsp; 假性近视可以通过平时的看远看近调节、常做眼保健操、培养正确的用眼习惯、多参加户外运动等调节恢复。而真性近视除了要及时佩戴有框眼镜矫正外，也可以选择以中医排毒为基点的眼部护理中心进行调理，比如像乔士德润眼部护理中心、鹰眼视力、爱尔眼科等，身边不少朋友家的孩子通过在乔士德润眼部护理中心实体店进行点穴拨筋调理，孩子视力健康有了很大改善。
&nbsp; &nbsp; &nbsp;&nbsp;拥有一双视力健康的眼睛，不仅能增添容貌之美，更重要的是对自己的升学、就业、择偶等方方面面都有帮助。所以，从现在开始，请大家重视、珍爱自己的眼部健康吧。
