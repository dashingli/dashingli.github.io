我们眼睛需要什么，你知道吗？

十月 21, 2019 本文总阅读量次

表示，经预测在未来5年中，干眼症患者的人数会以每年10%的比例上升。另有统计数据表明，在眼科门诊中，每5个人中就有一个患干眼症，以电脑工作人员、上班族和开车族居多。
&nbsp; &nbsp; &nbsp; 可以说，眼睛是最易受到疲劳侵袭的源头。长期使用电脑的人普遍患有干眼症，即容易眼干、眼红和疲倦。这是因为电脑荧光屏由小荧光点组成，眼睛不断地调整焦距，以保证视物清晰，时间过长，眼肌会过于疲劳。眼睛出现酸痛、发胀、干涩、视力模糊等问题，很容易导致头痛、注意力不集中、情绪烦燥、反应迟钝、疲乏无力和失眠、恶心、欲吐等一系列疲劳症状。

&nbsp; &nbsp; &nbsp;&nbsp;俗话说：“眼睛是心灵的窗户。”不言而喻，拥有一双健康、美丽的眼睛，对一个人来说是多么地重要。但是白内障、沙眼、青光眼、夜盲症、视网膜病变等各种眼病正在威胁、困扰着每一个人，给人们的生活带来了诸多的不便。提高爱眼、护眼意识，预防眼部疾病的发生已经迫在眉睫。导致眼病的因素多种多样，其中营养摄入不均衡，使得视神经、眼底得不到足够的养分是引发眼病的一个重要原因，应当引起人们的重视。

&nbsp; &nbsp; &nbsp; 蓝莓果实中除了常规的糖、酸和VC外，含VE、VA等多种维生素、熊果苷、蛋白质、花青苷、食用纤维以及丰富的K、Fe、Zn、Ca等矿物质元素。除了丰富的营养元素外，爱大爱眼科专家还发现，蓝莓中含的花青素，可直接加速视网膜、视紫质的合成与再生，让眼睛绽放活力光彩；此外，蓝莓能帮助营养预防近视、干眼症等各类眼部问题。
&nbsp; &nbsp; &nbsp; 另外，鳕鱼肝油中含的天然维生素A，对眼睛视网膜感光功能等也起着一定的作用。所以对于学生族、长期使用电脑或长期佩戴隐形眼镜的人群来说，若是同时服用些蓝莓和鳕鱼肝油，双管齐下保护眼睛，对解决各种眼部干燥、干涩、干眼症、视力下降等眼部问题来说都是较为不错的选择.
