视力不好不能做什么工作

十月 23, 2019 本文总阅读量次

&nbsp; &nbsp; &nbsp; 眼睛健康，生活快乐。但是视力不好，生活就从此黯然失色了。视力不好不能做什么工作？爱大爱眼科专家对此问题进行了总结，下面我们一起来看看。

眼科专家表示，视力不好影响非常大，尤其对于青少年影响不容忽视，具体有以下几个方面：
&nbsp; &nbsp; &nbsp; 1、视力下降，眼睛经常干涩和疲劳，影响学习、生活和工作质量。长期戴镜，会导致生活工作不便。而中高度近视，还会导致眼球突出，眼睑松弛，影响容貌，在升学、参军、考公务员和找工作都会受到限制。
&nbsp; &nbsp; &nbsp;&nbsp;2、人到老年后还会出现眼花的问题，还需要配镜解决。花眼加上近视就需要随身携带两副眼镜，造成很大的不便。而且近视还会引发白内障、青光眼等疾病，而且发病率明显高于正常人。而中高度近视，特别是高度近视容易引发玻璃体混浊、视网膜出血和脱离而致盲，造成更为严重的后果。
&nbsp; &nbsp; &nbsp; 3、近视还人遗传，而且遗传性是非常的大，尤其是中高度近视，遗传几率在80%以上。
&nbsp; &nbsp; &nbsp; 大家知道视力不好的阻碍后，就要积极的预防和保健，防止其无限制地发展。在学习中要让孩子保持正确有姿势，在生活中避免孩子长时间的看电视电脑或是玩游戏。同时也要对有近视症状的孩子进行适当的调节。
