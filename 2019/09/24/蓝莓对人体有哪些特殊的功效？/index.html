蓝莓对人体有哪些特殊的功效？

九月 24, 2019 本文总阅读量次

小编一起来看看！

&nbsp; &nbsp; &nbsp; 随着年龄增加，眼睛的抵抗力也随之变弱。蓝莓果实中的花青素可促进视网膜细胞中视紫质的再生成，可预防重度近视及视网膜剥离，而视紫质正是保证良好视力不可或缺的成分。
&nbsp; &nbsp; &nbsp;&nbsp;自由基会引发多种疾病，蓝莓有中和自由基的功效。蓝莓中的花青素能够使自由基变得稳定。另外还可与胶原蛋白形成抗氧化保护膜，帮助VC/VE吸收利用，增强抗氧化能力。
&nbsp; &nbsp; &nbsp; 蓝莓化合物可以促进记忆力反映区脑细胞的增长，改善短期记忆，提高认识能力，再通过血脑屏障，改善退行性老年痴呆，延缓和转化衰老症状。
&nbsp; &nbsp; &nbsp;&nbsp;蓝莓中有能抵御真菌感染的化合物，可以降低人体胆固醇，预防肥胖，增强血管抵抗力，减小毛细血管脆性，还可以保持血管通透，降低心血管疾病发病率。
&nbsp; &nbsp; &nbsp; 蓝莓之所以对眼睛有保护作用是因为蓝莓中含有含量非常高的花青甘色素，即花青素。花青素是一种广泛存在于植物中水溶性色素。经常使用电脑及眼睛易疲劳的人，可以适当的食用蓝莓来保护眼睛，保持眼睛的明亮健康。
