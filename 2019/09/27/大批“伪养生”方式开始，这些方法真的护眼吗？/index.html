大批“伪养生”方式开始，这些方法真的护眼吗？

九月 27, 2019 本文总阅读量次

专家表示：事实并非如此！

&nbsp; &nbsp; &nbsp;&nbsp;一般来说，当我们的眼睛疲劳出现充血症状就会滴眼药水，且滴眼药水之后，红血丝很快便得到缓解。“但是这种消退只是暂时的，几个小时之后，红血丝只会再次出现。频繁使用之后就会产生依赖。”眼表面血管的舒张收缩功能受到影响之外，更严重的会促使血压升高、心肌冠脉缺血。正确的护眼方式就是立刻放下手机和电脑，闭着眼休息10分钟，或者用毛巾眼贴冷敷一下~
&nbsp; &nbsp; &nbsp; 手机护眼模式只是将手机的冷色光变成偏黄或偏暗的暖色光，减少蓝光进入眼睛。但其实从根本上上来讲作用不大。眼睛最大的敌人就是手机电脑，一直玩不论什么方法都会伤害眼睛。我们可以配戴防蓝光的眼镜，最一劳永逸的方法莫过于少看手机了~
&nbsp; &nbsp; &nbsp; 眼保健操是防控近视的一种方法，它主要目的是让眼睛得到充分休息，通过按摩眼睛周围的穴位循环眼周血液，提高视力那必然是不可能的。眼睛一旦近视就无法恢复，同时要警惕市面上那些谣传的恢复视力的方法。眼睛近视不可逆，一定要及时呵护眼睛，保护眼睛的方法千千万，总有一款适合你
