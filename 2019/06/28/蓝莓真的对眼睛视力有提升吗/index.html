蓝莓真的对眼睛视力有提升吗

六月 28, 2019 本文总阅读量次

介绍，蓝莓是一种营养价值非常高的水果，果肉含丰富的维生素、蛋白质和矿物质等营养元素，其中蓝莓中独特而珍贵的花青苷色素在众水果中比重是非常的高，因花青素可以保护眼睛，所以蓝莓的药效大量用于眼睛保健方面。花青苷色素对人体的眼睛具有很好地保养的作用，它可以缓解眼睛疲劳、改善人的视力，同时可直接加速视网膜、视紫质的合成与再生，让眼睛更明亮光彩！

&nbsp; &nbsp; &nbsp; 现医学研究报告指明：蓝莓中的花青素可以促进视网膜细胞中的视紫质再生，预防近视又可增进视力。蓝莓中富含丰富维生素、矿物质和纤维元素等营养成分，多多食用蓝莓可以提高人体免疫力，提高人体素质。蓝莓中丰富的维生素C具有预防癌症，抵抗心脏病的功效。蓝莓果胶丰富，可以稀释人体脂肪，保护人体心脑血管的健康。蓝莓中还有丰富的抗氧化剂，可以延缓人体的衰老。
&nbsp; &nbsp; &nbsp;&nbsp;很多健康饮食杂志发表对蓝莓的赞扬：蓝莓具有极高的抗氧化活性，富含大量的营养而在欧美国家大受欢迎；在世界卫生组织公布的绿茶等十大健康食品中，蓝莓是唯一入选的水果；联合国粮农组织将蓝莓列为“人类五大健康食品之一”；美国《时代》周刊称，“蓝莓可能比其他任何一种水果或蔬菜含有抗氧化物都要多”。
&nbsp; &nbsp; &nbsp; 蓝莓果实中的花青素对眼睛有良好的保健作用，能促进视网膜上视红素的再合成，减轻眼部疲劳及提高夜间视力，是很多药物无法比拟的。天然的护眼食物，作为新时代健康饮食的时代，无疑，蓝莓是电脑族的光明之星。
&nbsp; &nbsp; &nbsp;&nbsp;蓝莓同时是老幼咸宜的健康明智之选。蓝莓中的花青素可以抗癌并减少心脏疾病，防止脑神经老化、增强人的肌体免疫力。蓝莓生吃甘甜，制作成为蓝莓汁就津甜可口，酿制作为蓝莓酱可与面包类同时，味更甜美！可谓饮食圣品。
