爱大爱教你：眼睛如何保健

十二月 2, 2019 本文总阅读量次

&nbsp; &nbsp; &nbsp; 眼睛是人类心灵的窗口，眼睛是非常重要的器官，首先看到一个人就是从双眼看起，人人都希望自己有双明亮而有神的眼睛来带我们看这个美丽的世界，醒脑明目精神好眼睛保健的方法。

眼睛如何保健，醒脑明目精神好
多吃用对眼睛对视力比较好的食物：桂圆肉、山药、动物肝脏、桑葚、红枣等等，也可用红枣、人参、枸杞、菊花、罗汉果泡茶饮用，都是具有明目养眼的作用。
眼睛喜凉怕热，如果经常用眼过多，导致眼睛出现红肿、充血、发干的情况，可以经常用流动的凉水惊醒洗脸并对眼睛部位用凉水进行拍洗的方式，可以减少眼睛疾病的发生，保护视力。另外还可以选用茶水热熏的方式，对眼睛也是有保护和舒缓作用的。
让眼球加以运动，舒缓眼睛的压力。具体做法：两脚分开与肩宽，挺胸站立，头稍仰。瞪大双眼，尽量使眼球不停转动，先从右向左转10次，再从左向右转10次。休息10分钟之后再重复上述运动，如此3遍。这个方法能起到醒脑明目之功效。
&nbsp; &nbsp; &nbsp;&nbsp;1.预防近视，不要在光线昏暗的地方下进行看书、阅读。坐姿一定要端正，眼睛与书籍一定要保持30厘米以上的距离，不要躺着或者坐车时看书。
&nbsp; &nbsp; &nbsp; 2.读书或者玩电子产品的时间不宜过长，持续五十分钟以上应该休息十分钟，并且休息期间可以做下眼保健操。
&nbsp; &nbsp; &nbsp; 3.不要用手揉双眼，不用脏毛巾擦拭双眼，保持双眼的洁净。
&nbsp; &nbsp; &nbsp;&nbsp;4.不宜直视太阳光，以免烧伤眼睛。
