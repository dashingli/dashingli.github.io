玩手机的错误姿势

七月 11, 2019 本文总阅读量次

为大家，讲解一下错误的玩手机姿势能带来的严重后果。

《一》“躺玩”太舒服易诱发结膜炎
&nbsp; &nbsp; &nbsp; 有种说法认为，躺着玩手机会影响正常的眼压。其实，无论侧躺还是仰躺，对眼压的影响都很微小，但会对眼睛的结膜造成威胁。
&nbsp; &nbsp; &nbsp; 因为长时间保持这几种姿势，容易使血液流向眼睛，导致眼部结膜血管的轻度充血，睑结膜也容易发生轻度乳头增生及形成滤泡等。起初，可能会有眦部皮肤潮红、湿润、结痂的现象，如不加以注意，则会诱发结膜组织的慢性炎性病变，导致慢性结膜炎。
《二》距离不适当易诱发近视眼
&nbsp; &nbsp; &nbsp;&nbsp;人躺在床上时，很难保证手机与眼睛之间的距离适当，尤其专注于手机游戏时，由于手机上的字和图片都较小，眼睛会不知不觉地离手机屏幕越来越近，长此以往，很容易诱发近视眼，特别是18岁以下处于生长发育期的青少年。
&nbsp; &nbsp; &nbsp; 如果长时间注视近物不休息，导致睫状肌疲劳，就会造成睫状肌的痉挛，时间长了会造成眼球长度、大小的改变，使睫状肌调节功能变差，晶状体曲度增大，改变屈光度，从而形成或加剧近视眼症状。
《三》时间没控制易诱发干眼症
&nbsp; &nbsp; &nbsp;&nbsp;因为躺着比较舒服，所以，临睡前用手机上网、读书，不知不觉就能耗到后半夜。连续几个小时盯着小小的手机屏幕，眼睛会变得很干燥，别以为这是因为自己困了，其实这是眼部肌肉功能发生了障碍。
&nbsp; &nbsp; &nbsp; 在此，爱大爱眼睛养护中心提醒，如不及时改掉躺着玩手机的习惯，就会患上干眼症。另外，长时间与屏幕近距离接触，还可能会诱发眼睛晶状体代谢异常，使晶状体浑浊，形成白内障。
