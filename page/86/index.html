高度近视患者孩子会近视

五月 28, 2019

“全面实施一对夫妇可生育两个孩子政策，积极开展应对人口老龄化行动”，自从国家全面放开二胎政策以来，很多人都在讨论到底要不要生？什么时候生？然而，从眼科学角度来看，令人关注的还是近视遗传。近视妈妈怀孕是否会遗传给孩子？遗传几率有多大？专家表示，如果父母均为高度近视，孩子高度近视发生率很可能达到85%。
近视的遗传性这么大，我们应该如何保护二胎的视力？要弄清楚这个问题，首先要知道近视的几个主要原因：
1.遗传因素：
父母高度近视的孩子比普通孩子近视风险要高。如果夫妻一方是高度近视，孩子近视的几率约50%；夫妻双方都是高度近视，孩子近视的几率高达85%以上。
2.环境因素：
缺乏自然光照射，在光线太亮、太暗、眩光、反光的地方看书，在颠簸的环境下阅读等，在不合适的环境下用眼的行为。
3.行为因素：
过少的户外运动和过多的近距离用眼、长时间看书玩手机，不正确的写字姿势，睡眠时间少等行为都会加大近视的风险。

除遗传因素不可控外，其他近视的因素都是家长们可以防范，因此想要二胎不近视，父母们在孩子出生后，要注意观察孩子的眼睛，从以下几个方面来预防近视：
Tip1：增加户外运动时间
研究表明，增加户外活动能有效降低近视眼的发病率。因此，对于想预防近视的孩子，户外活动时间至少为2小时，3小时更佳。
Tip2：建立眼屈光发育档案
屈光发育档案就是给孩子眼睛做一个屈光发育评估，看看孩子有没有近视、远视和散光等。六七岁之前的儿童都是低度远视眼，其屈光状态会随着眼球的发育而变化，所以我们常见的近视眼是完全可以提前预测的。为详细了解孩子的屈光变化情况，及时发现屈光发育异常，定期带孩子进行屈光检查，并建立完善的屈光发育档案，是非常有必要的。
爱大爱建议，36岁开始每年做一次全面视光检查并建立屈光发育档案。在孩子3岁以后，家长应每年带孩子到专业的眼科医院做12次全面视光检查，包括屈光度数、眼轴的长度、角膜的弧度，看看孩子是否属于高危近视眼人群，通过历年的检测合理干预近视的发生和发展。

近视和散光两者的区别，你知道多少？

五月 23, 2019

在现代生活中很多孩子的用眼习惯都不是很好，最后去医院检查后发现孩子近视、散光了。得了近视或散光，这可不是什么好事，近视散光度数持续加深会对孩子的日常生活产生影响。
得了近视与散光，似乎眼前的世界就被蒙上了雾气。有人或许会有所疑惑，同样是影响视力，近视和散光哪个更严重？在此爱大爱小编就来给大家讲解一下近视和散光的相关内容。
近视和散光两者的区别，你知道多少？
首先有一点需要跟大家明确的就是近视和散光谈不上哪个更严重一些，这两种眼疾都有轻重之分，不管是被确诊为得了哪种眼科疾病都必须尽早接受矫正治疗。
临床上近视眼也被称为短视眼，近视眼只能在近处看到了一些远的距离就看不清楚了。
近视也分为真性近视、假性近视和混合性近视三种类型。近视眼的人除了眼镜会有度数之外，还会出现晚上视力变差、眼前有漂浮物、闪光感等症状，并且绝大多数近视眼的人眼底都会出现病变。
散光的人眼球在不同的子午线上屈光力也是不太一样的，光线进入眼球最终形成两条焦线和相当小的弥散斑点，这种现象被称为散光，进入眼球里面的光不能在视网膜上面形成焦点。
临床上散光又分为规则散光和不规则散光两种类型，其中最大屈光力和最小屈光力不相互垂直的成为不规则散光。
规则散光又分为逆规散光、斜向散光和顺规散光三种情况。顺规散光指的是最大屈光力主子午线在90°±30°位置，逆规散光位置是在180°±30°，剩下来的就是斜向散光了。
综上所述我们晓得了无论散光还是近视都有轻重之分，不能单纯地比较这两者谁严重一些。爱大爱希望家长不论发现自己孩子近视还是散光都要尽快去医院接受治疗，并且要教孩子合理用眼，多吃对眼镜有好处的食物，如胡萝卜等。

1、西红柿改善视力
&nbsp; &nbsp; &nbsp;&nbsp;西红柿有助改善视力。维生素A和C不仅有助于提高视力而且也可以帮助预防夜盲症，而西红柿则是维生素A和C的极好来源。最近的一项研究表明，番茄的高维生素A含量可以帮助预防黄斑变性和严重眼病。此外，食用西红柿可以降低患上白内障的风险。西红柿丰富的植物化学物质、抗氧化剂番茄红素、叶黄素和玉米黄质，能保护眼睛不受光线损害。

&nbsp; &nbsp; &nbsp;&nbsp;珍珠含95％以上的碳酸钙及少量氧化镁、氧化铝等矿物质，并含有多种氨基酸、如亮氨酸、蛋氨酸、丙氨酸、甘氨酸、谷氨酸、天门冬氨酸等，珍珠性味甘咸寒，用珍珠粉配龙脑、琥珀等配成的“真珠散”可抑制白内障的形成的功效。

眼睛与内在健康息息相关

五月 16, 2019

从眼睛可以看出一个人的健康状况，机体的状况反过来也会影响眼睛的健康状况。任何一个脏腑的生理功能出现问题，都会影响眼睛的状况，因此要想让一双眼睛灵动生辉，内在机体的健康就必不可少。要保持内在的健康，首先要做到以下几个方面，下面爱大爱小编就跟您一起了解下有哪些方面。
&nbsp;　　1.充足的睡眠
&nbsp;　　睡眠是身体消除疲劳的最主要方式，眼睛也在睡眠的过程中得到充分的休息和滋养。绝大多数健康长寿的老年人都拥有一个良好而正常的睡眠。对于女性来讲，充足的睡眠还有利于皮肤美容，这是因为在睡眠过程中皮肤毛细血管循环增多，其分泌和清除过程加强，加快了皮肤的再生，所以睡眠有益于皮肤美容。
&nbsp;　　2.健康均衡的饮食
&nbsp;　　健康均衡的饮食对于人体的重要性不言而喻，健康均衡的饮食原则遵循七大营养素均衡的原则。亲近自然，均衡营养，合理取用。
&nbsp;　　3.坚持运动
&nbsp;　　坚持规律的运动，最好是每天保持一定的适度运动，可以提高机体的新陈代谢水平，增强心肺功能，提高机体的免疫状况，同时在运动过程中可以释放压力，调节精神状况。&nbsp;
4.保持良好的精神状态
&nbsp;　　特别是女性，如果抑郁悲观，可能直接影响到激素的分泌，引起激素分泌失调，而这些化学物质的分泌失常对于免疫系统也会产生影响。长期的不良情绪或者过于剧烈的精神刺激可能成为肿瘤等恶性疾病的诱发因素。乐观的生活，是健康的开始。
&nbsp;　　5.定期身体检查&nbsp;
定期做身体检查可以做到对疾病的早发现、早诊断、早治疗。尤其是进入中年的女性，定期的身体检查尤为重要。定期的眼睛检查也有利于眼部疾病的早诊断和早治疗，患有糖尿病、高血压等疾病的女性更是要定期进行眼部的检查。
&nbsp;　　总结
&nbsp;　　此外，保持良好的用眼卫生也很重要。无论是看书还是用电脑、看手机、玩ipad都不宜时间过长，每隔一个小时左右就应该停下来休息10—15分钟来极目远眺，这样可以让眼睛得到充分的休息。此外要注意不要在光线下看书、看电子产品。还有人喜欢躺在床上看书、玩电子产品，姿势不良容易导致斜视或者加重眼睛的负担。

比起近视度数，这一指标对于青少年视力保护而言更加重要

五月 13, 2019

眼轴


有些小朋友的家长在带孩子去眼科医院验光的时候，可能会遇到这样的情况：视力测量结果只有100度，却被医生诊断为高度近视。这是因为，孩子眼轴的测量值可能已经大大超出了这个年龄段的标准值。
首先，我们先来了解一下眼轴是什么。通俗来讲，眼轴就是眼球前后的直径。更加专业点说，从眼球接收光线的最表层，即从角膜－晶状体－玻璃体－视网膜（感受光线最里的一层)的距离可以被看成是物理中的光学系统的一条中轴线，这条轴线就是所谓的“眼轴”。

儿童的眼睛在发育过程中，角膜和晶状体会越来越平，而眼轴则会越来越长，这两方面共同维持人的正视状态，保证光线聚焦于视网膜。而当眼轴过长时，光线只能聚焦于视网膜前，这种状态，就是近视。因此，在其他眼部条件相似的情况下，眼轴更长，则近视得越严重。

近视是无法治愈的

很容易出现视网膜脱离、黄斑变性等并发症。

说了这么多，作为家长的你是否认识到眼轴测量的重要性了呢？赶紧行动起来，当孩子出现视力下降时，尽早带孩子到专业眼健康机构做一次全面的眼部检查，才能早发现早干预。那么问题来了，我们要如何及时发现孩子视力下降情况呢？超级视力表APP，是可以快速准确测量视力的手机软件，APP会根据测量结果自动建立视力档案，同时能够以图表地形式一目了然地呈现视力变化。赶快下载安装，每天关注视力变化吧！

权威资讯 | 我国青少年近视问题有多严重？53.6%世界第一！

五月 10, 2019


在国家卫健委29日举行的新闻发布会上，国家卫健委疾控局副局长张勇指出，2018年全国儿童青少年总体近视率为53.6%。其中，6岁儿童为14.5%，小学生为36.0%，初中生为71.6%，高中生为81.0%，近视防控任务艰巨。
2018年下半年，国家卫健委会同教育部、民政部组织开展了2018年全国儿童青少年近视调查工作。本次调查共覆盖了全国1033所幼儿园、3810所中小学校，总筛查111.74万人，包括幼儿园儿童6.92万，各年龄段中小学生104.82万。
调查显示，我国儿童青少年总体近视发病形势严峻，低年龄段近视问题比较突出。在小学和初中阶段，近视率随着年级的升高快速增长，小学阶段从一年级的15.7%增长到六年级的59.0%，初中阶段从初一年级的64.9%增长到初三年级的77.0%，小学和初中阶段是我国近视防控的重点年龄阶段。高度近视问题不容忽视。高三年级学生高度近视，也就是近视度数高于600度，在近视总数中占比达到21.9%，由于高度近视是致盲性眼病之一，容易导致一系列严重的并发症，应该引起高度警惕和重视。
另据监测发现，我国学生近视相关危害因素广泛存在。比如，67%的学生每天户外活动时间不足2小时，29%的学生不足1小时，73%的学生每天睡眠时间不达标，课后作业时间和持续近距离用眼时间过长、不科学使用电子产品等不良用眼行为普遍存在。以上因素对近视高发起到了主要的作用，是导致我国儿童青少年近视率居高不下的重要影响因素。
张勇说，国家卫健委将继续实施好2019年学生近视和健康影响因素专项监测，更加有针对性地开展近视干预工作。以高发地区和低年龄段学生为重点，以增加日间户外活动、科学使用电子产品、合理安排学业任务等为主要措施，指导学校和家长对学生实施有针对性的近视综合干预。进一步加强和规范近视医疗服务。强化近视筛查和早期发现，提高近视矫正医疗服务水平。以近视防控为突破口，全面推进学校卫生工作。把学校卫生作为公共卫生服务体系建设的重点，指导督促地方落实好保障政策和措施，疾病预防控制机构未设立学校卫生科所的地区，2019年底前要把省级专业机构建立起来。配合教育部门，研究配齐配强校医的政策措施，共同推动校医室建设，增强学校卫生健康工作能力。根据本次调查结果，各省份已经确定了本省份儿童青少年近视防控的目标和具体的综合防控措施。

世界卫生组织于4月24日发布了一项关于5岁以下儿童的身体活动、久坐行为和睡眠的新指南。指南中对5岁以下儿童的父母提出了以下的建议：

不建议屏幕时间

1~2岁的儿童应：

对于1岁儿童，不建议久坐不动的屏幕时间（如看电视或视频，玩电脑游戏）。2岁以上儿童，久坐不动的屏幕时间不应超过1小时

5岁以下幼儿每天看电子屏幕的时间不应超过1小时，而1岁以下的婴儿则最好完全不要看屏幕。

然而现在我们很多家长，为了让小孩保持安静，经常会把孩子放在沙发上看电视，而且很可能一看就是几个小时，这很有可能埋下严重的健康隐患。
首先，长时间看电子屏幕，对视力的损伤不言而喻。婴儿眼睛发育不成熟，屏幕上图像频繁跳动，会牵扯到宝宝大部分精力，减少眨眼次数，也就减少了眼睛里润滑剂和泪液的分泌，会让眼睛干涩疲劳。同时，电视的电磁波和各种彩色光线，也会对眼睛造成刺激。这很可能给今后的近视埋下伏笔。
其次，看电视不同于看书，看书可以促进发育，提高儿童的分析思考能力。而看电视属于一种被动接受的行为，会阻碍孩子体会真实的世界，限制其想象力。3岁前，大脑发育应完成约80%。经常看电视会延缓大脑发育，且这一影响是不可逆的。
最后其实也是最重要的一点，看电视或者其他的电子产品会挤压孩子的睡眠和户外运动时间。 幼年是身体和认知能力快速发育期，在此期间应养成积极活动、避免久坐和保证睡眠的习惯，有助于塑造童年、青春期和成年期的健康生活方式，避免肥胖症及相关疾病。
所以，各位宝爸宝妈们，赶快拿开孩子眼前的电子屏幕！每天从点滴做起，呵护宝宝健康发育！

视力档案功能，帮助用户直观了解视力变化

五月 6, 2019

对于超级视力表APP用户而言，初次使用时最关注的一定是用手机测量视力究竟准不准，和使用纸质视力表测出的视力值相比是否存在偏差。然而与纸质视力表“较真儿”，并不是我们开发这款APP的初衷，我们最希望用户关注的点，是长期的视力变化趋势。

APP中的视力档案功能，便是基于这一初衷而设计的，用户每一次的视力测量数据都会自动记录在内。APP还提供了一键绘图功能，可以用折线图的形式将视力变化趋势可视化地呈现出来。

另外，针对每一条视力检测记录，用户都可查看详情，同时还能通过点击“纠正检测参数”按钮，对测量模式（裸眼/矫正）和检测距离进行修正。这一设置是为了避免时因检测时选择了错误的参数，而影响到视力变化趋势整体上的准确性和连续性。
对于发育期的青少年而言，视力检测如同刷牙一样，应当每天坚持进行，这样便可以及时发现视力下降情况，将假性近视扼杀在摇篮中。

不同大小的手机屏幕，显示的视标尺寸是否一致？

五月 6, 2019


不少超级视力表APP用户在使用过程中都有疑问：使用不同屏幕尺寸的手机进行视力检测，是否能得到一样的测量结果？今天我们就为来大家解答一下这个问题。
如今的智能手机市场上，全面屏、曲面屏、2.5D屏等各种尺寸、各种形状的屏幕种类繁多，这可给从事APP开发的程序员们出了一道难题——如何让所有的手机都能显示出正常比例的软件界面。这个问题对于其他类型的手机APP来说可能还没那么难解决，他们只需让软件界面按比例缩放即可。然而对于超级视力表APP来说，在软件界面可以随屏幕大小进行缩放的同时，需要保证视标的绝对尺寸不变。也就是说，无论用户使用的是什么样的手机，显示的视标大小都应该完全一致。
为了让不同屏幕尺寸的手机都能显示符合国标尺寸的视标，超级视力表开发团队针对全系列的苹果手机和上百部安卓系统手机进行了适配和优化，确保了所有主流机型都可以完美显示出同样尺寸的视标，这里请允许我为超级视力表开发团队严谨的精神点个赞。

虽然开发团队为市面上所有的主流机型都做了适配，然而还是不能排除某些小众手机无法显示标准尺寸视标的可能性。针对这一问题，开发人员特意在超级视力表APP中加入了视标校准功能。

用户可以在APP侧栏菜单中找到这一功能。打开后，界面左侧会显示出一条长方形，用户只需用尺子量取长方形长边的长度，将数值填入输入框中，并点击校准按钮，系统便会自动进行适配，显示出标准的视标尺寸。
