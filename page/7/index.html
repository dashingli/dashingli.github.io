玩手机的错误姿势

五月 18, 2022

为大家，讲解一下错误的玩手机姿势能带来的严重后果。

《一》“躺玩”太舒服易诱发结膜炎
&nbsp; &nbsp; &nbsp; 有种说法认为，躺着玩手机会影响正常的眼压。其实，无论侧躺还是仰躺，对眼压的影响都很微小，但会对眼睛的结膜造成威胁。
&nbsp; &nbsp; &nbsp; 因为长时间保持这几种姿势，容易使血液流向眼睛，导致眼部结膜血管的轻度充血，睑结膜也容易发生轻度乳头增生及形成滤泡等。起初，可能会有眦部皮肤潮红、湿润、结痂的现象，如不加以注意，则会诱发结膜组织的慢性炎性病变，导致慢性结膜炎。
《二》距离不适当易诱发近视眼
&nbsp; &nbsp; &nbsp;&nbsp;人躺在床上时，很难保证手机与眼睛之间的距离适当，尤其专注于手机游戏时，由于手机上的字和图片都较小，眼睛会不知不觉地离手机屏幕越来越近，长此以往，很容易诱发近视眼，特别是18岁以下处于生长发育期的青少年。
&nbsp; &nbsp; &nbsp; 如果长时间注视近物不休息，导致睫状肌疲劳，就会造成睫状肌的痉挛，时间长了会造成眼球长度、大小的改变，使睫状肌调节功能变差，晶状体曲度增大，改变屈光度，从而形成或加剧近视眼症状。
《三》时间没控制易诱发干眼症
&nbsp; &nbsp; &nbsp;&nbsp;因为躺着比较舒服，所以，临睡前用手机上网、读书，不知不觉就能耗到后半夜。连续几个小时盯着小小的手机屏幕，眼睛会变得很干燥，别以为这是因为自己困了，其实这是眼部肌肉功能发生了障碍。
&nbsp; &nbsp; &nbsp; 在此，爱大爱眼睛养护中心提醒，如不及时改掉躺着玩手机的习惯，就会患上干眼症。另外，长时间与屏幕近距离接触，还可能会诱发眼睛晶状体代谢异常，使晶状体浑浊，形成白内障。

【爱大爱眼睛养护】给显示器换换位预防“眼疲劳”

五月 18, 2022

&nbsp; &nbsp; &nbsp; 爱大爱发现，球迷们因熬夜看球，常常熬得“红眼金睛”，眼睛别提有多疲劳，头晕专家说，这正是典型的“视频终端综合症”，建议球迷们熬夜前先检查一下屋子，调整下光线，最大程度地减少眼部疲劳。
&nbsp; &nbsp; &nbsp; 你家的电脑、电视放在哪里?放置显示器不仅要考虑美观，更要考虑眼睛喜不喜欢。显示屏的位置不要面向窗户或背向窗户。
&nbsp; &nbsp; &nbsp; 当显示器背向窗户时，人眼直接望向强光源，显得屏幕的光线非常暗，二来太阳光移动，易在显示器上留下阴影，这些都会加重视疲劳。而显示器正对着窗户，阳光直射荧光屏，反光太强，当然眼睛也容易疲劳。最佳的情况是显示器跟窗户(阳台)垂直。如果是电脑你可以移动显示器的角度，玩掌中游戏时，也要注意按此原则选择位置。
&nbsp; &nbsp; &nbsp;&nbsp;无论干什么事最好都别“摸黑作业”，看显示器时，这种光线最好在头顶，或者把光源放置在身旁稍后的位置，不要使光线反射入眼睛。有光源还不够，光线还得足够稳定。如果灯光闪烁不定还是赶快换掉吧，这无疑会增加眼睛的“劳动量”。
&nbsp; &nbsp; &nbsp; 如今，不少人以电脑替代电视机，连看球赛都选择网络直播。这让眼睛更容易疲劳，也是视频终端综合症多见的原因之一。
&nbsp; &nbsp; &nbsp; 显示器放置的高度，以眼睛20°俯视角为宜。放得过高看电视一段时间以后，会觉得颈部特别累。建议屏幕与眼睛的距离为50～60厘米。因为，人的位置距显示器愈近，受刺激愈大，也就愈易造成眼疲劳。
&nbsp; &nbsp; &nbsp;&nbsp;有些人看近物会戴着低度的眼镜，这是不恰当的。眼睛屈光不对，看不清眯缝着眼睛会增加眼睛的疲劳，久而久之还会造成视力的下滑。所以，有近视、远视或散光者应及时验光配戴合适的眼镜。
&nbsp; &nbsp; &nbsp;有些人对色彩要求高，特别是设计或绘图工作者往往会把屏幕的亮度与对比度调得很高，其实这会无形中增加眼睛工作的强度，没特殊需要最好别这样操作。所以调整屏幕亮度应柔和些，有一定的对比度但不能太强烈，这样看起来最不费眼。还要调整屏幕明暗度，让使字体显示清晰，看起来不费力。
&nbsp; &nbsp; &nbsp; 如果你每天都得长时间使用电脑的话，建议安装视保屏。视保屏是加在电视或者计算机显示器前的一块透明屏幕，能过滤强光、反光，能增加屏幕观看舒适度。
&nbsp; &nbsp; &nbsp; 眼睛干干的，不舒服，这是很多熬夜族的共识。这跟用眼习惯有关，也跟环境有关，环境过于干燥会让眼睛干涩。
&nbsp; &nbsp; &nbsp; 你想想看，若房间内太干燥，会加快眼内泪液的蒸发加重眼的干涩感。现在广州比较湿热，自然环境不至于干燥，如果房间内开空调让你太干燥，容易口渴嘴唇脱皮的话，不妨在室内放置一盆水。同时多饮水，房间多通风。
&nbsp; &nbsp; &nbsp;&nbsp;多眨眼是预防眼干涩非常有效的方法。在正常情况下，人的眨眼活动频率约每分钟12次。而在操作电脑时尤其是玩游戏时，大家的注意力高度集中于屏幕，眨眼频率会不自觉地显著减少。人不眨眼时，上下睑缘间眼表面暴露面积增大，如果每分钟眨眼次数不足5次，泪液蒸发过多过快，易造成干涩问题。所以，看球的人一定要有意识地提醒自己眨眼，即至少每6秒眨眼一次。
&nbsp; &nbsp; &nbsp; 视频终端综合症以眼部症状为主，但也会有全身的不适，如肩颈不适，头晕头痛等，预防措施就是多活动。一般建议，1小时左右就要休息5～10分钟。球迷们可以在中场休息时，放松一下眼睛。欧扬说，休息时应离开显示器，起身活动并眺望远处。这样的小活动不仅放松眼睛，更能放松身体。
&nbsp; &nbsp; &nbsp; 所谓“视频终端”就是我们日常生活接触的大大小小的显示器。比如，电脑屏幕、电视机、游戏机等。长时间盯着荧光屏，特别是熬夜，很容易出现眼睛疲劳或视力损害，这类症状包括充血、干涩，眼有异物感、分泌物多等。除眼睛症状外，还有一些全身症状，像头痛、头晕、失眠或做恶梦，腰背部酸痛不适，这些都是熬夜电脑族的通病。得按文中方法好好检查一下眼睛的“工作环境”合不合格，会不会无形中让眼睛疲劳作业，这是减少熬夜眼疲劳的最主要方法。

多吃鱼眼睛对近视眼视力有帮助吗？

五月 18, 2022

表示，平时宜吃些桂圆、大枣、枸杞子、黑木耳等食物，水果可选择柑桔类。另外，羊肝也是护眼佳品。
&nbsp; &nbsp; &nbsp; 民间传言，多吃鱼眼睛是可以帮助人们提高视力的。那么，多吃鱼眼睛对近视眼视力有帮助吗？这种说法有没有科学依据呢？通过以下的介绍我们一起来了解一下。

&nbsp; &nbsp; &nbsp;&nbsp;科学上证实，鱼眼中含有丰富的维生素及一些特定的脂肪酸和硒等微量元素，而硒在谷胱甘肽过氧化酶中起到重要作用，这种酶能催化并排除对眼睛有危害的自由基物质，从而起到保护眼睛细膜和提高眼睛对视象的敏锐。因此，多吃鱼眼睛对近视眼视力还是有一定的帮助。
&nbsp; &nbsp; &nbsp; 此外，鱼肉可能对眼睛的视力有一定的帮助。澳洲国立大学DrWayneSmith发表的“眼科学档案”(ArchivesOphthalmology)一篇研究报告指出，多吃鱼可以预防随年龄增长所发生的视网膜黄斑部退化。
&nbsp; &nbsp; &nbsp;&nbsp;多吃鱼眼睛对近视眼视力有帮助吗？虽然这种说法有一定的科学依据，但是保护近视眼视力，均衡饮食是关键的一步，通过饮食的帮助来提高视力还是有一定的科学依据的。叶黄素、维生索A、DHA、AA等都是有益于宝贝眼睛健康发育的重要营养元素。
&nbsp; &nbsp; &nbsp; 特别是叶黄素，叶黄素好比”隐形的太阳镜”。作为一种帮助眼睛健康的关键抗氧化剂，叶黄素高度集中在婴幼儿的视网膜黄斑区。能够有助保护婴幼儿视网膜免受蓝光伤害。为婴幼儿幼嫩的眼睛起到屏障作用。此外。叶黄素作为抗氧化剂，能够有效帮助保护视网膜免受氧化损害。
&nbsp; &nbsp; &nbsp; 当然，对视力提高有帮助，不一定以为要针对性的“狂吃”这类食物，食物需要有一个合理的水平，才能达到真正的效果。过分的饮食又很可能带来负面的影响，到时恐怕还会害了近视眼患者的视力呢。

孩子近视眼成因中医的观点

五月 18, 2022

&nbsp; &nbsp; &nbsp; 基础教育质量监测中心日前发布了《中国义务教育质量监测报告》。监测发现，学生的肥胖、近视和睡眠不足问题较为突出，四年级、八年级学生视力不良检出率分别为36.5%、65.3%。同时，3成以上学生感到很有学习压力。

&nbsp; &nbsp; &nbsp; 中国孩子近视眼高发，课业负担重是主要问题，而孩子的脾虚体质是更重要的内因，因为中医说的脾有两个特点，一个是负责“升清降浊”，另一个是主肌肉，这两点都和近视眼的发生有关系。
&nbsp; &nbsp; &nbsp;&nbsp;所谓“升清降浊”，就是维护身体营养的供应和废物的代谢。“清”指的是营养和能量，“浊”指的是代谢废物。具体到眼睛，脾气虚的时候清气不升，营养或气血不能上达到头面，视力活动需要的能量就不足，与视力相关的肌肉、血管的功能也不足，近视由此发生。
&nbsp; &nbsp; &nbsp; 中医说的“脾”是主肌肉的，其中包括肌肉的张力、运动功能。所谓近视眼，就是调节眼球的肌肉疲劳了，失去了弹性，这不仅决定于眼睛的使用强度，还决定于肌肉的活力。中医说脾气虚时，包括眼周肌肉在内的全身肌肉都会不给力。
&nbsp; &nbsp; &nbsp; 脾虚的人易疲劳，是因为身体的大肌肉群张力不足，不能持重；这种人消化不好，大便总是不成形，这是因为胃肠肌肉运动无力。全身的肌肉性质是一致的，它们的功能行使都需要借助肌肉中的线粒体，全身肌肉无力的脾虚之人，眼周的肌肉也一样，所以，在同样的用眼情况下，这类人更容易得近视。
&nbsp; &nbsp; &nbsp; 健脾的办法很多，中药中的黄芪、白术，食物中的大枣、山药都是。还有一个便捷的办法就是运动，运动时线粒体工作加强，肌肉力量提升了，对身体各个器官的供血也就加强了，中医说的脾气虚也随之减轻。特别是孩子，他们的近视眼多是生活方式错误所致，所以预防也要从这里入手，无需借助其他。而且运动的时候自然也是眼睛休息的时候，孩子会自觉地看远处，这些都是眼周肌肉放松充电的好时机。

别让近视阻挡你的未来

五月 18, 2022

&nbsp; &nbsp; &nbsp; 爱大爱根据世界卫生组织最新研究报告显示做出总结，目前我国近视人数多达6亿人。据统计，小中大学生近视率为：小学生45.7%，初中生74.4%，高中生83.3%，大学生则高达87.7%，青少年近视率高居世界第一。世界卫生组织已把近视眼列为致盲性眼病。
&nbsp; &nbsp; &nbsp; 近视除了给人们的日常生活带来不便之外，还会造成学习效率下降、影响学习成绩，限制高考、择业（在我国，飞行、航海、医学均对视力有严格要求，而且近视人群在择业、参军等均受到限制），影响形象、耽误恋爱，高度近视遗传后代等不良影响。
&nbsp; &nbsp; &nbsp;&nbsp;更可怕的是，近视还有可能引起一系列的并发症，包括眼睛的角膜、巩膜病变、晶体病变、玻璃体病变、黄斑区脉络、青光眼等，严重的甚至会导致失明，危害极大。
&nbsp; &nbsp; &nbsp; 所以，在日常生活中，呵护好我们的视力健康，养成良好的用眼习惯是非常重要的。比如说，读书、写字、看电脑的时候距离书本、电脑屏幕3035厘米左右。学习、工作一段时间后，要起身活动一下，到户外或窗前远眺35分钟，缓解视力疲劳。
&nbsp; &nbsp; &nbsp;&nbsp;而对于儿童青少年来说，当出现看黑板模糊、重影，看书时间长后视力疲劳、头痛等症状后，要及时到医院检查一下是不是近视、散光。
&nbsp; &nbsp; &nbsp; 假性近视可以通过平时的看远看近调节、常做眼保健操、培养正确的用眼习惯、多参加户外运动等调节恢复。而真性近视除了要及时佩戴有框眼镜矫正外，也可以选择以中医排毒为基点的眼部护理中心进行调理，比如像乔士德润眼部护理中心、鹰眼视力、爱尔眼科等，身边不少朋友家的孩子通过在乔士德润眼部护理中心实体店进行点穴拨筋调理，孩子视力健康有了很大改善。
&nbsp; &nbsp; &nbsp;&nbsp;拥有一双视力健康的眼睛，不仅能增添容貌之美，更重要的是对自己的升学、就业、择偶等方方面面都有帮助。所以，从现在开始，请大家重视、珍爱自己的眼部健康吧。

如今的近视发展人口有多少？

五月 18, 2022

&nbsp; &nbsp; &nbsp; 根据2016年北京大学中国健康发展研究中心发布的《国民视觉健康》白皮书显示，2012年中国5岁以上总人口中，屈光不正的患病人数在5亿左右，其中近视的总患病人数在4.5亿左右，高度近视总人口高达3000万。如果缺乏有效的政策干预，到2020年，我国5岁以上人口的近视患病率将增长到51%左右，患病人口将达7亿。
&nbsp; &nbsp; &nbsp;&nbsp;另外据教育部2015年发布的“全国学生体质与健康调研”结果报告显示，2014年中国小学生视力不良检出率达到45.71%，初中生达到74.36%，高中生达到83.28%，青少年近视率逐年迅速增加，已经高居世界第一位。按照全国学生近视率推算，全省中小学生总计1050万人近视。

&nbsp; &nbsp; &nbsp; 眼睛近视不仅影响孩子的学习，而且还会对心理有一定的影响，小学生近视的危害多，家长们一定要注意。
小学生近视的危害一：
&nbsp; &nbsp; &nbsp;&nbsp;影响到学习。如果眼睛近视的话，坐在比较靠后一点的位置的话，会看不清黑板上的字迹，上课注意力难以集中，影响学习。
小学生近视的危害二：
&nbsp; &nbsp; &nbsp; 加重眼疲劳，进而加重近视的发展。小学生年纪都是比较小的，当眼睛长期处于紧张的调节状态下，易引起假性近视，但如果还是疲劳用眼，只会加重近视的发展，成为真性近视，而此时就不得不配眼镜了。
小学生近视的危害三：
&nbsp; &nbsp; &nbsp; 影响孩子心理健康。由于孩子近视了，一些活动不适合参加，与同学们的关系相处也变得比较微妙了，而且近视的话，即使参加活动的话，有的也有一些不便，对小学生而言，会影响到他们的心理健康，容易有负面情绪。
小学生近视的危害四：
&nbsp; &nbsp; &nbsp;&nbsp;影响到身体锻炼，眼睛近视，参加一些活动也就变得少了，处于成长中的身体得不到好的锻炼，会影响到身体的发育。
小学生近视的危害五：
&nbsp; &nbsp; &nbsp; 影响到孩子的前途。有些专业对视力的要求比较高，比如宇航员，而眼睛近视的话，选择的专业就会受到限制，影响他们的前途。
小学生近视的危害六：
&nbsp; &nbsp; &nbsp;&nbsp;小学生近视的话，如果眼睛保护不好，度数增长是比较快的。由于这个年龄段，眼睛也是处于一个生长发育阶段，眼睛视力状况也是不稳定的，不合理的用眼，会加快近视的发展，特别是已经是中度近视的小学生，以后成为高度近视的可能性就比较大，而高度近视更易引发眼病。
小学生近视的危害七：
&nbsp; &nbsp; &nbsp; 有些小学生近视了，不配镜，任其发展，因此影响到眼睛的集合与调节能力，眼睛度数会上升的比较快，有些还会发展成为斜视。

乘公交车不要盯着手机看

五月 18, 2022

&nbsp; &nbsp; &nbsp; 人眼在看距离相对固定的物体时，睫状肌的收缩和伸张能保持相对稳定，眼睛不会在短时间内感到疲劳。而在公交车上，经常会出现较大的颠簸和摇晃，手机屏幕也会随车厢一起颤动，为了看清屏幕上的图像和字幕，睫状肌就要被迫不停地调节，易产生视觉疲劳。
&nbsp; &nbsp; &nbsp;&nbsp;另外，由于公交车上空间有限，在车上人多时，有些人会站在手机跟前，有些人则离得很远，为了把图像看清楚，眼睛的调节经常处于紧张状态;还有人站在与屏幕偏斜角度大的地方看，这样更容易产生视觉疲劳;有的人会低着头看手机，时间久了，除了对眼睛有伤害，颈部肌肉也会感到酸痛、劳累。

&nbsp; &nbsp; &nbsp; 以上这些不当做法，很容易使人出现眼睛干涩、酸胀、疼痛等不适症状，还会加重近视，甚至诱发青光眼急性发作。
&nbsp; &nbsp; &nbsp;&nbsp;所以，在公交车上，最好不要长时间盯着手机屏幕看。可以看一会儿，再闭上眼睛休息一会儿，或是看看车窗外的景色，让眼睛充分放松。
&nbsp; &nbsp; &nbsp; 需要提醒的是，要做到手机前护眼绝对不是用摘眼镜来实现的，而是要注意用眼习惯的点点滴滴。首先，要注意自己的用眼时间，每看一个小时左右的手机就应该闭上眼睛休息几分钟。这样可以让眼睛不至于过度疲劳。
&nbsp; &nbsp; &nbsp;&nbsp;其次，在长时间看手机屏幕的情况下，尽量强迫自己多眨眼，这样也可以让眼睛得到充分的休息。另外要让眼睛与屏幕的距离保持在60厘米左右,屏幕中心点在水平视线以下10~15度之间，此外在劳累的时候多看一些绿色植物或者用一些眼药水。

孩子看东西模糊，是近视了吗？

五月 18, 2022

眼科专家一起来了解孩子的近视问题。

据相关资料显示，我国近视人数和发病率居世界第一，近视的总数达到4.5亿左右，而其中一个因素是很多家长在尚没有区分清楚孩子是否真的近视前，就盲目的给孩子带上了眼镜，直接导致了孩子原本可恢复正常的假性近视变成了真性近视。如何区分真假性近视呢？让我们一起来看看。
&nbsp; &nbsp; &nbsp; 眼睛能够看近或看远的事物都要多亏了我们眼睛里的一个部位——睫状肌。当我们看远的事物的时候，睫状肌就会放松，眼睛里面的晶状体就会变得扁平，远处的光线就能聚焦到视网膜上了；而当我们看近的东西的时候，睫状肌收缩，晶状体变得凸起，相当于一个凸透镜，把近处的物体成像聚焦到视网膜上。
真性近视是眼球变长了，每延长1mm，近视增加300度，是不可逆的。
&nbsp; &nbsp; &nbsp;&nbsp;假性近视是指睫状肌长时间收缩、痉挛，一时间无法放松。通过药物或物理治疗或者经过一段时间的休息，睫状肌放松下来后，假性近视就会消失。
&nbsp; &nbsp; &nbsp; 爱大爱建议，若发现孩子视力下降，需要及时到医院检查，一般18岁以下的青少年以及儿童需要散瞳验光，使用滴眼液将睫状肌麻痹，之后再验光测定眼睛的度数。如果散瞳后仍有近视，很可能就是真性近视了。如果散瞳后近视度数消失，则为假性近视。
&nbsp; &nbsp; &nbsp; 假性近视通过积极的治疗，还是有可能让孩子恢复至正常视力的。在了解治疗前，我们先来看一下假性近视会有哪3方面的表现。
当孩子看远处时会显得模糊，检查时发现远视力一般会＞0.6而＜1.0，近视力则显示为正常。
视力无法稳定，休息一段时间可能会好转，一旦过度用眼时可能又会出现视力下降的情况。
一般假性近视用药后视力可以得到恢复，但停止治疗或是过度用眼后又会出现近视的状况。
&nbsp; &nbsp; &nbsp;&nbsp;若带孩子去正规医疗机构检查后确诊为假性近视，是不需要佩戴眼镜的。（8岁以下的小朋友可能需要使用1%阿托品眼用凝胶散瞳）。请遵医嘱给孩子用药，并带孩子定期就医复诊。

&nbsp; &nbsp; &nbsp; 注意用眼卫生及用眼习惯，无论是写作业还是看书请坐姿端正，做到“3个1”：手指持笔距离笔尖1寸，眼睛距离书本1尺，胸部距离桌子1拳。
&nbsp; &nbsp; &nbsp;&nbsp;建议孩子一次连续近距离用眼时间在45分钟左右后休息10分钟左右并眺望远方，让眼睛得到放松，这也是课间休息时间为10分钟的原因之一，为了让学生眼睛得到适度的休息。
&nbsp; &nbsp; &nbsp; 科技日益发展为我们的生活带来了便利，但随之而来的弊端就是更多的孩子患上了近视，因此使用电子产品30分钟后就需稍作休息。
&nbsp; &nbsp; &nbsp; 有效的户外活动是预防近视的最好方法，每天带孩子坚持户外活动2小时以上即可防治近视。

&nbsp; &nbsp; &nbsp; 因为，老花眼是眼睛的老化现象，只要年龄增长，谁都逃不掉这个问题。即便是近视的人，眼睛的对焦功能也确实不彰。
&nbsp; &nbsp; &nbsp; 不知道您是否看过戴着近视眼镜的人，在近看印刷小字的时候，把眼镜摘下来读的模样？他们会不自觉地摘下眼镜，是因为在利用眼镜矫正视力的状态下看不清楚近物的关系。毫无疑问的，这就是他们患老花眼的证据。

&nbsp; &nbsp; &nbsp; 老花眼是由于眼球结构和功能的衰老退化造成的，但上了年纪之后视力不好不一定都是老花眼。配老花镜前要到医院给眼睛做个全面检查，包括远视、近视、眼压和眼底的检查，在排除白内障、青光眼以及一些眼底疾病之后，才能验光确定度数。尽量在医院眼科或正规眼镜店验光和选配镜片，选定后先试戴半小时左右，没有不适后再将其加工成型。有的老人远的近的都看不清，这就需要配两副眼镜，也可以选用从上到下屈光度渐增的多焦渐变镜片，上方适用于看远，下方适用于看近，使用起来更加方便。
