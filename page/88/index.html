高度近视危险吗？

四月 8, 2019

我国是近视眼发病率最高的国家，每4个人当中就有1个人是近视眼。近视超过600度就属于高度近视，由于眼轴增长，视网膜被拉伸变薄，出现变性、裂孔的可能增大，所以高度近视很容易发生视网膜脱离。
特别是高度近视的人参加剧烈运动，如打篮球、玩跳水、或者眼球快速转动等，都会增加视网膜脱落的风险。
高度近视注意点：
1、注意用眼卫生，避免近距离用眼过度，包括看电脑、手机等。
2、避免重体力劳动及剧烈运动，如跳水、短跑、足球、打篮球等。
3、如果你有闪光感或眼前有固定黑影遮挡、视物变形等症状，很可能是视网膜脱离的征兆，应立即到院就诊，以免延误最佳治疗时机。
4、定期到医院检查，包括眼压、眼底，如发现视网膜裂孔可以预防性地施行激光光凝术治疗，以降低视网膜脱离发生的几率。
5、高度近视患者的小孩更容易得近视眼，要尽早检查和干预。

养护双眼绝非青少年之事

四月 4, 2019

&nbsp; &nbsp; &nbsp;&nbsp;作为一个眼保健事业的人员，从业十数载以来我突然发现一个有趣的现象：去医院说要治疗眼睛近视的都是孩子，很少有青年人或者中年人来治疗近视。
&nbsp; &nbsp; &nbsp; 这是一个很有趣的现象，大家都明白，近些年来人们不论做什么都大抵是离不开电子产品的。小孩子会打游戏，会从网上找先进的云课堂来补习；大人坐在办公室工作，就会用到电脑，偶尔放松一下就去看手机；就包括医院里，每个大夫桌子前都放着一台电脑.也就是说，现在不光是正在学习的青少年，就是成年人，甚至是老年人，都会产生视力问题。
&nbsp; &nbsp; &nbsp;&nbsp;我身边的有的朋友也会说，自己长时间看电脑屏幕，会感觉眼睛干涩酸胀，严重的还会看东西模糊不清，疼痛流泪，问我说敢怎么办，我对他们说，去医院看一看，去保养一下吧，他们都苦笑着摇摇头：没时间，不值当。

&nbsp; &nbsp; &nbsp; 这种想法可谓是大错特错，从小时候我们就知道，眼睛是心灵的窗户，小时候每天都要做眼保健操，听一句“为革命，保护视力”难不成现在就不用保护视力了？工作了，有孩子了，眼睛就不是心灵的窗户了？眼睛是跟随一个人一辈子的宝贝，现在觉得只是眼睛不舒服，你还觉得没什么大不了，等小病堆成疾，那就追悔莫及矣！对于现在的人们来说，用眼用眼，要用眼的地方太多了，尤其是电脑手机这些有电子屏幕的电器，本来就是最伤害眼镜的，大家还都离不开这些，是没有办法。
&nbsp; &nbsp; &nbsp;&nbsp;最后在强调一下，现在科技发达了，屏幕越来越多了，视力问题也越来越严重了，我们国家近视的人差不多有三个亿，眼睛不舒服，也就是视力亚健康的人超过7成，很可怕的事实，所以我觉得大家不管是不是近视，也不管大家是什么年龄都要爱惜自己的双眼，养护好自己的眼睛。

学生近视的预防

四月 1, 2019

&nbsp; &nbsp; &nbsp; 青少年如何进行眼的预防呢？根据实验报告得知，即使是不识字、不会读书的猴子，如果持续脖子向前弯曲的姿势几个月，也一样会造成近视。那么接下来从几个方面介绍近视眼的预防方法。

一、阅读或念书都要在桌前进行
&nbsp; &nbsp; &nbsp;&nbsp;不论是阅读或用功念书时，都一定要坐在椅子上，背骨挺直，眼睛距离书籍或薄本30公分以上。在这十项中，此乃是重要的基本事项。
二、绝对不可以躺着看书
&nbsp; &nbsp; &nbsp;&nbsp;两眼球具有经常保持水平的功能，因此，一旦躺着看书，两眼球就会拼命想保持水平。结果，焦距就会从正确的位置产生了差距，因而造成视力衰退。
三、有时必须下意识地伸展背肌
&nbsp; &nbsp; &nbsp;&nbsp;即使最初保持正确的姿势，长时间下来，也会变成人向前倾、脖子向前弯曲的姿势。因此，每看完一页，就要伸展背肌、改正姿势，一定要养成这种习惯才好。所以，在本人自觉之前，父母得时时提醒他注意。
四、要使用颜色较浓的铅笔
&nbsp; &nbsp; &nbsp;&nbsp;铅笔要使用b以上的，最少也得是hb。笔心颜色较浅的铅笔，在书写时必须用力，人很容易向前倾，是造成姿势不良的一个原因。因此，仅可能使用笔芯颜色较浓的铅笔。
五、与桌面保持倾斜度
&nbsp; &nbsp; &nbsp;&nbsp;在桌面上摆入一块可以调节高度的板子，以调整眼睛与书本的角度。板子的高度，以10-15公分为宜，其上则放着三夹板。可以减少眼睛的疲劳。此外，市售的直立式书架也不错。
六、选购适合的桌椅
&nbsp; &nbsp; &nbsp;&nbsp;保持正确的姿势坐在椅子上，调整桌子的高度到膝盖的高度。如果双脚无法碰到地面，可以利用旧书等来做为踏台，切忌不要使脚在半空中摇晃，否则，无法保持正确的姿势。
七、定期进行桌椅的调整
&nbsp; &nbsp; &nbsp;&nbsp;无法配合身体的桌椅，虽然不是直接造成近视的成因，但会影响正确的姿势，压迫胸部，导致背骨弯曲。因而形成姿势不良，致使近视，因此，要配合小孩的成长速度，定期调整桌椅，这点是非常重要的。最好选择能够任意调整高度的桌椅。
八、不在车上看书
&nbsp; &nbsp; &nbsp;&nbsp;车内的照明度不够，外来的光线容易使书面忽明忽暗。文字随着车子前进而晃动，很容易造成脖子弯曲、眼睛疲劳。在车上看书的人，姿势绝对不会正确。
九、适度的休息能保持正确的姿势、提升效率
&nbsp; &nbsp; &nbsp;&nbsp;在学校，可以利用休息时间，伸展背肌、两手向上高举、拉直背骨，或看看远景。在家里用功，每隔30分钟要休息五分钟，让身心充分休息，使情绪焕然一新。最好在纸上写明这十个注意事项，然后贴在桌前或压在桌子上，以便时时提醒自己努力保持正确的姿势。

隐形眼镜和框架镜的选择

三月 21, 2019

表示，隐形眼镜和框架镜的选择是按照自身情况来决定主次为好。
&nbsp; &nbsp; &nbsp; 当然，同时我们要根据自己的近视程度来选择眼镜以便得到一个清晰的视力。而框架眼镜和隐形眼镜都能够很好的辅助我们提升视力，但很多人也在这两种选择上犯了难。
那具体怎么选呢？
&nbsp; &nbsp; &nbsp; 首先，相比较而言，框架眼镜可以让人在审美上有一个更好的感官，而且使用的寿命也比较长，基本不需要护理，保持日常的清洁防止磨损就可以了，尤其是摘戴眼镜，不但节省时间，关键是安全可靠。
&nbsp; &nbsp; &nbsp;&nbsp;虽然隐形眼镜视野更宽阔，视物时有更好的立体视觉，影像大小真实无球面像差和色散。但是，隐形眼镜视力矫正功能局限性更大，例如：不能矫正晶体散光，矫正角膜散光较差，同时采用散光隐形眼镜矫正散光时，定位性也不一定能够达到一定的要求。

&nbsp; &nbsp; &nbsp; 尤其配戴操作卫生条件要求严格，配戴和护理起来会很麻烦，如果长期的佩戴隐形眼镜还会有眼干眼涩眼疲劳的情况发生。所以综合来了，还是佩戴框架眼镜效果会更好！不过小伙伴们可能要说了，有的时候框架眼镜不方便，我们不得不佩戴隐形眼镜啊。
&nbsp; &nbsp; &nbsp; 如果觉得传统的框架眼镜戴起来有负担，鼻梁有压迫感，那就选一副纯钛超轻材质的眼镜，不仅仅是轻便，更能保证时尚又舒适，当然还能起到装饰作用！

&nbsp; &nbsp; &nbsp; 如果觉得运动时框架眼镜易脱落，那就选一副专业的运动眼镜，走心的设计可以让你畅快淋漓的享受运动的乐趣。要是觉得外出的阳光太刺眼，那为什么不选一副近视太阳镜呢，在获得清晰视野的同时还能够有效的阻挡紫外线哦。
&nbsp; &nbsp; &nbsp; 如果你还在为冬季眼镜起雾而烦恼，那防雾镜片就可以为你解忧了；另外，框架眼镜也有很多不同的风格可以选择，复古的，小清新的，沉稳的，怎么搭配都可以。其实无论佩戴哪种眼镜都要注意用眼卫生，注意眼睛的休息，选择一副好的舒适的眼镜，就是给自己双眼最好的呵护。

近视能拖延吗

三月 20, 2019

&nbsp; &nbsp; &nbsp;&nbsp;很多家长面对学习成绩和视力健康两个问题的选择时，会毫不犹豫地选择学习成绩，而忽视孩子的视力健康。因此一旦发现孩子的成绩下降了，则赶紧为孩子报个学习班周末为孩子补课；发现孩子营养不好，赶紧为孩子购买进口食品补充营养。而对待孩子近视这个问题，却显得犹豫不决，展现出能拖就拖，实在看不清黑板影响成绩了，才去医院或眼镜店配一副近视眼镜解决。

近视能拖延吗？
&nbsp; &nbsp; &nbsp; 近视会拖累孩子的学习。近视后，孩子看不清黑板和投影上的字，孩子的成绩很容易下降。因为视物模糊、眼睛干涩酸疼，精神难以集中，以至情绪烦躁甚至头晕等现象，会让孩子多付出30%的时间、精力、脑力，在学习上花费更长的时间。
&nbsp; &nbsp; &nbsp;&nbsp;30多个专业报考受限，每年会有大批的考生，因为近视被限报专业拒之门外，涉及电子信息、生物工程、医学制药、飞行、航海技术等专业，甚至连参军的愿望也无法实现，严重影响孩子的人生规划。
&nbsp; &nbsp; &nbsp; 近视危害身心健康被起绰号伤自尊，戴上眼镜体育锻炼不积极，身心健康俱受损。
近视造成生活诸多不方便
&nbsp; &nbsp; &nbsp;&nbsp;视力不好，影响开车、影响吃饭、影响脸部美观、影响交际。
高度近视并发症多
&nbsp; &nbsp; &nbsp; 孩子近视不及时控制，每年会平均以50-100度迅速增加，一旦变成600度以上的高度近视，眼压高、黄斑出血病变、白内障等并发症让人防不胜防。
&nbsp; &nbsp; &nbsp;&nbsp;高度近视改变基因，遗传后代。高度近视将改变人体基因，变成隐形遗传因子遗传给后代。父母一方为高度近视，孩子近视的机率近50%，如果父母双方都高度近视，孩子近视机率为80%以上！即使激光手术也无法改变这样的现实，从而使整个家族世代成为眼镜的奴隶。

拯救孩子近视，刻不容缓
&nbsp; &nbsp; &nbsp; 孩子近视可预防。排除遗传因素，后天近视的形成，主要是环境因素，即长时间看近导致眼球过度发育，眼轴超过正常值，因此预防近视的最好办法是让孩子多望远少看近。远离手机、IPAD和电脑游戏，每天户外运动1小时。定期检查视力，不近视的孩子3个月检查一次，发现每只眼视力低于1.0，立即采取措施，尽早治疗。
&nbsp; &nbsp; &nbsp;&nbsp;孩子一旦出现近视，3个月以内的，往往是假性近视（眼肌痉挛导致），裸眼视力一般在0.6-0.8(4.8-4.9),如果及时处理，就可以完全恢复。
&nbsp; &nbsp; &nbsp; 孩子一旦近视时间过长，不及时处理，假性近视就会变成真性近视，然后又会产生部分假性近视，如此反复叠加，导致近视度数不断加深，裸眼视力持续下降。不戴眼镜，成绩受影响，还容易眯眼，产生斜视，甚至弱视，度数仍会加深；配近视眼镜，度数更是年年加深，年年更换眼镜。由此可见，近视不可怕，可怕的是拖延和不重视！

经常熬夜的男性小心患眼病

三月 19, 2019

&nbsp; &nbsp; &nbsp; 当你结束一整夜的工作后，是否也曾有过遭到一种眼病“奇袭”的经历：突然感到眼前一片模糊，视野中心仿佛飘着一团黑雾，看到的东西都是变形、缩小的。爱大爱指出，这种专挑通宵工作的中青年男性“下手”的眼病有个冗长的名字，叫做“特发性中心性浆液性脉络膜视网膜病变”。经常熬夜、过度疲劳、工作紧张、长期用皮质类固醇药物都是诱发疾病的因素。
&nbsp; &nbsp; &nbsp; 这种病变主要发生在眼睛地黄斑区，黄斑区是视网膜的一个重要区域，主要与精细视觉及色觉等视功能有关。和其他黄斑病变不同，这种病最常见于25至50岁的男性，仅有少部分是50岁以上的男性患者。最奇怪的是，女性则极少患这种病，但原因还没有查明。

&nbsp; &nbsp; &nbsp;&nbsp;熬夜工作的人往往平时的工作压力很大，生活也不规律，常常过度疲劳。如果还总是熬通宵，人处于长时间的焦虑紧张状态，毛细血管随之发生痉挛，引起供氧不足。由于眼睛的黄斑区是全身耗氧量最大的地方，因此最先受到影响。
&nbsp; &nbsp; &nbsp; 预防这种眼病，平时作息要有规律，避免长时间陷入焦虑紧张的状态。尽量避免频繁、连续熬夜，如果因工作原因难以避免，最好定期到眼科接受造影检查。此外，喝酒、抽烟都可造成毛细血管痉挛，最好能戒掉。
&nbsp; &nbsp; &nbsp;&nbsp;吸烟对眼睛的危害也很大。烟草中含有的尼古丁能使视网膜血管痉挛、诱发视网膜动脉或静脉阻塞，导致视网膜和玻璃体出血。有研究表明，吸烟的人患视网膜和玻璃体出血的几率明显高于非吸烟者，尤其对同时合并有糖尿病或高血压的病人危害更大。

预防近视日常的需求

三月 15, 2019

&nbsp; &nbsp; &nbsp; 近视——根据近视的程度不同决定所能清晰视物的距离，近视程度越深的人，清晰视物的距离越近，也就是由于种种原因造成眼睛的形状变化，晶状体变形，眼睛成像于视网膜前方造成视物不清。近年来越来越多的人得了近视眼，并且近视眼逐渐低龄化。而其形成的原因多是因为电子产品的普及，并且很多的青少年在过度使用眼睛情况下，很多的用眼习惯也很恶劣，因而对眼睛造成了不可磨灭的伤害，那么针对影响视力的不利因素，提出如何预防近视的方法，以进行有效的日常的近视调护。

&nbsp; &nbsp; &nbsp; 预防近视眼的方法
&nbsp; &nbsp; &nbsp; 1.针对用眼习惯不良提出的建议。阅读时候的姿势对视力的影响很大，通常来说阅读的时候要保持姿势的端正，并且保持眼睛与书本之间的距离，约为三十厘米；走路的时候阅读不利于视力的保持，乘车的时候看书对于视力也有不利的影响。
&nbsp; &nbsp; &nbsp;&nbsp;2.针对阅读环境对视力的影响提出的建议。光线对于视力的影响也非常大，照明的强度过强或者过弱对于视力来说都是由伤害的，其次闪烁光、黑板的反光、阳光直射的条件都是会对视力造成一定的伤害。
&nbsp; &nbsp; &nbsp; 3.根据视力变化所做的调控。对于已经近视的人来说应该及时佩戴合适的眼睛，不要自认为戴上眼镜视力就会更加恶化就盲目拒绝佩戴眼镜。并且要定时检查视力，以保证视力的保持以及做出适当的调节方案。
&nbsp; &nbsp; &nbsp; 4.营养和体质也是影响视力的两大因素，加强体育锻炼、注意营养的补充对于预防近视也是有帮助的。
&nbsp; &nbsp; &nbsp; 预防近视眼，避免过度使用眼睛，在阅读的时候注意光线和姿势，日常注意营养和锻炼，相信这能够帮助你有效预防近视眼。

爱大爱讲解老花眼的形成原因

三月 11, 2019

&nbsp; &nbsp; &nbsp; 年纪大了，多数人眼神都不好。近日，发表在《视觉科学》杂志上的一项最新研究显示，只有16%的老人视力正常。视力不好是引起老人摔倒、不敢出门等一系列生活难题的重要原因。然而，多数老人对待眼花问题却是不以为意，一些用眼及配镜误区使得老人眼睛越来越花。
&nbsp; &nbsp; &nbsp; 老花眼是由于眼球结构和功能的衰老退化造成的，但上了年纪之后视力不好不一定都是老花眼。配老花镜前要到医院给眼睛做个全面检查，包括远视、近视、眼压和眼底的检查，在排除白内障、青光眼以及一些眼底疾病之后，才能验光确定度数。尽量在医院眼科或正规眼镜店验光和选配镜片，选定后先试戴半小时左右，没有不适后再将其加工成型。有的老人远的近的都看不清，这就需要配两副眼镜，也可以选用从上到下屈光度渐增的多焦渐变镜片，上方适用于看远，下方适用于看近，使用起来更加方便。

&nbsp; &nbsp; &nbsp;&nbsp;三年换一次老花镜。随着年龄增加，老花程度也会加深，一旦老花镜不合适就得及时更换，否则会加速眼睛老花。建议每隔三年验光一次，若老花镜镜片出现划痕、老化等现象，会造成通光量下降，影响镜片的成像质量，也要及时更换。
&nbsp; &nbsp; &nbsp; 不能用放大镜代替老花镜。一些老人看书读报时，喜欢手里举一支放大镜代替老花镜，这样虽然看得清，但其实是在“纵容”自己的眼睛。放大镜如果折合成老花镜，相当于1000—2000度，长期使用放大镜，会使视力状况越来越差，习惯以后很难再配到合适的花镜。
&nbsp; &nbsp; &nbsp; 老人日常生活中，应多吃绿色蔬菜和水果，有助于视网膜上的重要蛋白质——视紫红质的再生。要经常用干净的手轻揉眼部，按摩时，两手食指弯曲，从内眼横揉至外眼角，再从外眼角横揉至内眼角;用食指尖按太阳穴数次，早、晚各做一遍，不仅可推迟老花眼，还能防治慢性眼病的发生。

长时间的蓝光照射会有什么后果呢

二月 25, 2019

&nbsp; &nbsp; &nbsp; 自然界有很多不同颜色的光谱，蓝光只是其中之一，是指波长在400—500纳米之间的光。波长越短，能量越高，穿透力越强，所以蓝光对于眼睛的影响为许多人所关注。各种关于蓝光危害的说法也流传开。对此专家表示：其实蓝光在我们生活中无处不在，我们要有正确的认识和评估，不要过大夸张蓝光的危害性。
&nbsp; &nbsp; &nbsp; 蓝光并不都是有害蓝光，真正有害的是400到440或450纳米以内的蓝光，而450到500纳米之间的蓝光有一种调整生物节律的作用，睡眠、情绪、记忆力等都与之相关，对人体反而是有益的。而且短波蓝光是不是一定会对人眼产生伤害，专家表示要客观对待，这与照射强度与照射时间有关，当照度达到一定程度后，持续两个小时以上，才有可能对视网膜产生损害。

怎么才能知道自己的眼镜防蓝光呢？长时间的蓝光照射会有什么后果呢？
&nbsp; &nbsp; &nbsp; 一个简单的方法（但不准确）说，如果你的眼镜阻挡蓝光是在彩色图形期待通过他们这样为这一个。如果您的PC眼镜擅长过滤蓝光，那么当您浏览它们时，两个图形之间应该没有明显区别。
&nbsp; &nbsp; &nbsp;&nbsp;蓝光是可见光谱的自然部分，约占所有阳光的25％。它还由电子屏幕和低色温的节能灯泡生产。
因为它具有所有可见光的最短波长，所以蓝光携带大量能量。长期接触它，例如每天在工作中使用计算机，可能会对视网膜造成严重损害。
&nbsp; &nbsp; &nbsp; 如果你是夜猫子，你应该考虑蓝光的另一个影响是它对你的自然睡眠和唤醒周期的影响，也就是你的昼夜节律。在一个典型的日子里，你的大脑使用来自太阳的蓝光来调节睡眠，提高警觉性和情绪，并为你的整体健康做出贡献。
&nbsp; &nbsp; &nbsp; 然而，黎明后太多的蓝光会导致头痛，眼睛疲劳，并破坏你的自然睡眠模式。眼睛疲劳和入睡困难等症状可能表明在使用手机，电脑或其他屏幕时缺乏适当的蓝光保护。
&nbsp; &nbsp; &nbsp; 有害蓝光也称为HEV蓝光。当它与眼睛相遇时，闪烁会产生眩光并降低对比度，从而影响视力的清晰度。从长远来看，这可能会导致视网膜损伤和“年龄相关的黄斑变性，这可能导致视力丧失”。
