近视为什么要戴眼镜

九月 22, 2020

表示：虽然儿童的睫状肌的调节能力是比较强的，但是一直处于痉挛状态的话，眼睛疲劳只会加重近视度数的加深，如果一直这样的话，眼睛视力下降会比较快。那近视戴眼镜有什么好处呢？
戴眼镜的好处
1.戴眼镜可以矫正视力
近视眼因为远方的光线不能在视网膜上聚焦，造成视远物不清楚，而配戴了近视镜后，就可以获得清晰的物像，从而使视力得以矫正。
2.戴眼镜可以减轻视疲劳
近视而不戴镜，必然导致眼镜极易疲劳，其结果只能是促使度数日益加深。正常戴镜后，视疲劳现象就会大大减少。
3.戴眼镜可以预防治疗眼外斜
近视眼视近时，眼的调节作用减弱，时间一长眼外直肌作用超过内直肌，就会引起眼位外斜。当然，近视伴外斜，仍可通过近视镜矫正。
4.戴眼镜可以防止眼球突出
青少年由于眼球仍处于发育期，调节性近视极易发展为轴性近视。特别是高度近视眼，眼球前后径显著延长，外表即表现为眼球突出，如果近视一开始就正常戴镜矫正，这种情况就会有所减轻，甚至可以不发生。
5.戴眼镜可以防治弱视
近视而没有及时戴镜，往往造成屈光不正性弱视，只要配戴合适眼镜，经过较长时间治疗，视力大多会逐渐提高。
温馨提示：养成良好的配戴眼镜习惯，保护好眼镜，可以有效的防治度数加深，甚至改善度数。祝大家都有个健康的眼睛来欣赏这个缤纷的世界。

近视的人长时间不矫正不戴眼镜会有什么后果

九月 17, 2020

&nbsp; &nbsp; &nbsp; 不少人开始出现看不清模糊的情况时，往往不以为意，不采取任何措施，听之任之。殊不知，近视并不是一项自限性病症，不会像一些感冒，只要时间够久总会见好。近视的人长时间不矫正不戴眼镜会有什么后果呢？在此，爱大爱罗列以下3点常见情况：
1·生活不便
&nbsp; &nbsp;&nbsp; 近视后最直观的感受就是视物模糊、看不清楚。5米外六亲不认、10米外雌雄难辨、30米外人畜不分，裸眼自带磨皮、点单全靠手机……“睁眼瞎”大概说的就是这种情况吧。
2·度数加深
近视眼本来是省流模式，为了执行大脑“看清楚”的指令，眼睛往往被迫加重调节，久而久之产生视疲劳，导致近视度数加深。长此以往，发展成高度近视指日可待。
3·影响发育
&nbsp; &nbsp; &nbsp; 青少年儿童处于生长发育期，如果长期模糊视物，眼底视网膜接收不到清晰物像刺激，容易影响双眼视功能发育，造成斜弱视。那这么说的话，就是一看不清就要赶紧去配眼镜咯？当然！也不是！
疑似近视该怎么做
&nbsp; &nbsp; &nbsp; 青少年儿童：医学验光，辨别真假近视，青少年儿童首次出现视力下降，一定要到正规的眼科医疗机构进行医学验光，确诊是否为假性近视。
假性近视多发于青少年，由于用眼过度，睫状肌持续收缩导致视物模糊，可通过药物等方式缓解调整，视力有望恢复正常。但如果诊断出近视，则无法通过调节恢复，需要采取戴眼镜这类光学方法矫正，控制近视度数加深。
成年人：排查眼部疾病，科学配镜
&nbsp; &nbsp; &nbsp; 对成年人来说，出现视力下降、视物模糊更不可掉以轻心。造成成年人视力下降的原因，除了近视，还有可能是白内障、青光眼、视网膜脱落等眼底疾病。需尽快到院检查，找出病因对症下药。若诊断为近视，也需及时配镜，避免向高度近视发展。

九个预防近视眼的方法

九月 14, 2020

小编表示：近视眼这种典型的需要进行全面的预防，一旦出现很大的危险就会造成人们的正常生活受到影响，那么具体的近视眼的预防应该怎么做？
养他们正确的读书、写字姿势，不要趴在桌子上或扭着身体。书本和眼睛应保持一市尺。学校课桌椅应适合学生身材。
2、看书写字时间不宜过久，持续1~1.5小时后要有一个短时间的休息眼睛向远眺，做眼保健操。
3、写字读书要有适当的光线，光线最好从左边照射过来。不要在太暗或者太亮的光线下看书、写字，减轻学生负担，保证课间10分钟休息，减轻视力疲劳。
4、积极开展体育锻炼保证学生每天有一小时体育活动。
5、教导学生写字，不要过小过密，更不要写斜、草字。写字时间不要过长。
6、认真做好眼保健操。
7、看电视时要注意高度应与视线相平;眼与荧光屏的距离不应小于荧光屏对角线长度的5倍;看电视时室内应开一盏支光小的电灯，有利于保护视力;在持续看电视1～1.5小时后要有一个短时间的休息眼睛向远眺，做眼保健操。
8、应多吃些含甲种维生素较丰富的食物各种蔬菜及动物的肝脏、蛋黄等。胡萝卜含维生素B,对眼睛有好处;多吃动物的肝脏可以治疗夜盲。近视患者普遍缺乏铬和锌，近视患者应多吃一些含锌较多的食物。食物中如黄豆、杏仁、紫菜、海带、羊肉、黄鱼、奶粉、茶叶、肉类、牛肉、肝类等含锌和铬较多，可适量增加。对近视要分档防治，抓早抓小。积极矫治和防止深度发展。如果已发生近视，要到医院去验光，配戴适宜的眼镜。可采用远雾视法、推拿操或晶体操以及物理疗法、苭物等进行矫治。
9.打羽毛球、乒乓球可防近视在打球过程中眼睛须快速追随羽毛球和乒乓球这类灵活性很强的“小球运动”轨迹变化，这对5～9岁的孩子的眼球功能完善有意想不到的好处。
爱大爱小编以上介绍了九大近视眼的预防方法，要想很好的预防近视眼就要注意经常做眼保健操，补充各种营养等等，才能让人们的视力得到很好的恢复。

弱视矫正愈早愈好

九月 10, 2020

&nbsp; &nbsp; &nbsp; 弱视多发生儿童发育的早期，孩子若不能及时接受矫正，视觉功能将长期受损。弱视主要在1～3岁形成，因此原则上弱视矫正愈早愈好，但由于6个月以内好眼的视力还很差，发育还不完善，即使弱视也不宜采取遮盖矫正，否则易引起好眼的弱视。一般来说遮盖的适当年龄为3～6岁，超过6岁效果很差，超过12岁者很难取得疗效。但由于现代科学的进步，弱视治疗方法的进展，治疗的有效年龄可能会延长。
(1)要在幼年时期就对儿童进行视力检查，一旦发现近视、远视、散光等屈光不正，应在医生指导下及时配戴合适度数的眼镜，以免发展成弱视。
(2)要耐心劝导孩子坚持戴眼镜。除洗澡、睡觉之外，一定要坚持戴眼镜，尤其是遮盖健眼时，这点很重要，镜腿要用链条连起来，防止眼镜打碎。
(3)孩子得了弱视后，要对弱视这一眼病有一个正确的认识，要充分了解弱视治疗过程是缓慢的，视力是逐渐提高的，不能操之过急，要有耐心、恒心。
(4)引导孩子在看近物时，一定要戴镜，尤其是绘图、写字时，这样常可达到事办功倍的效果。
(5)坚持及时复诊。2～3岁的儿童每两周一次，更大一些的儿童每一个月一次，请医生检查眼底，视力等，每年要散瞳验光一次。
一般儿童视力发育在6-8岁趋于稳定。如果不了解这一点就容易把正常视力的儿童误认为弱视。

两只眼睛度数有差别，怎么配镜

九月 7, 2020

&nbsp; &nbsp; &nbsp; 两只眼睛度数有差别，说的专业一点就是屈光参差，是指两眼的屈光状态不相一致。一般来说，人的两眼屈光状态普遍存在轻度的差异。
&nbsp; &nbsp; &nbsp; 屈光参差有多种类型，在眼睛发育过程中，远视的度数在不断减轻，而近视的度数在不断发展。如果两眼在远视的消减程度或近视的发展进度不同，就可引起屈光参差。先天因素者，出生时就有明显的两眼眼轴发育不平衡，两眼的屈光状态不相对称；另外，眼外伤、眼的手术也可造成屈光参差。
12岁以下的儿童
&nbsp;&nbsp;&nbsp; 不管是先天性的或者后天的双眼屈光参差，首先是要通过散瞳检影或常规验光，准确地检查出两眼屈光参差度。
12岁以下儿童由于其调节力、适应力强，一般对较高度数的足度矫正都是可以接受的。如果屈光参差较大，首次用框架眼镜完全矫正，感到不适应或两眼视疲劳，则可以先矫正一部分，然后等适应了再逐渐适度加深度数，进行完全矫正。
&nbsp; &nbsp; &nbsp; 通过实践证明，有些个别儿童的调节和适应力非常强，两眼屈光参差能够适应的完全矫正度达可600度。对儿童屈光参差尤其是远视性屈光参差，应及早全部矫正，以避免屈光参差性弱视。儿童双眼视力仍在发育，所以即使低度屈光参差也能导致弱视。屈光参差发生的年龄越小，弱视程度就越深。因此，儿童的屈光参差宜尽早发现、尽早全部矫正。
青少年
&nbsp; &nbsp; &nbsp; 由于学习任务非常繁重及每个人自身眼睛发育情况不同，所以青少年常常引发后天因素的双眼屈光参差。在适应的情况下，应最大限度地矫正其屈光参差，以便获得最好的双眼视觉。对于不能马上适应的高度屈光参差的矫正，首先查出主视眼，然后再完全矫正主视眼的屈光不正，并以此为基准，以次眼配合主眼融像，在患者能够适应的基础上，尽量提高完全矫正度，以增加矫正眼的视力。同时，两眼视野也得到扩大，视物清楚、舒适。
但是必须掌握次眼矫正不能过度，以免在与主视眼融像时，产生干扰而造成配戴不适。如屈光参差伴有弱视时，如果从未配戴过眼镜，可先令其两眼配戴正确矫正镜片以不使屈光度较高的眼完全废用。当然，如果患者不能忍受，应酌减弱视眼的屈光度数，保存该眼的有效视力，防止弱视加重。
中老年人
对于这个部分的人群，主要用眼是近距离伏案工作和中距离工作为主，所以矫正原则是：如果两眼屈光参差不大，视力接近，没有融像功能障碍，应该完全矫正。由于中老年人的调节、集合能力衰退，用框架眼镜无法矫正和适应时，如果屈光参差度较高，可以适当降低主视眼的屈光度，以其配合适应次眼的融像。这样，矫正远视力虽然受了一点影响，但近用工作的近视力不会有大的影响。对于年龄偏大，两眼屈光参差较大。
根据适应情况，可以部分矫正屈光参差，使其一只眼看远，另一只眼看近，矫正度数掌握原则是两眼看远、看近时，互不受干扰，配戴时无明显的不适感。

成年人视力下降，这是为什么？

九月 4, 2020

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 对于很多近视眼的人来说，心里总有这么一个概念：随着年龄的增加，视力会趋于稳定，眼镜度数也将不再增长。但实际上，不少成年人在检查视力时会发现，自己的视力越来越差了，有的人甚至担心近视度数无法控制。那么，为什么会出现这种情况呢？
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 爱大爱眼睛养护专家告诉你为什么：
一、近视增加有以下两种原因
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 一种是病理性近视，这是眼睛内部发生的病变，具有遗传性，表现为近视持续加深，且增加急速。对于成年人如果连续出现近视度数增长超过-1.00D/年，甚至-2.00D以上，或是累计达到-20.00D的超高度近视，就很可能是病理性近视。对病理性近视，目前没有特别有效的措施，后巩膜加固术可能对于某些患者有一定作用。
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 另一种是自身原因，不注意用眼卫生，用眼习惯不正确。比如，照明过亮或过暗，用眼时间太长，出现结膜炎、角膜炎等眼疾不及时诊治等，都会加重近视，这种原因较为常见。
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 除此以外，医学界还有一些说法，近视眼与缺乏钙、铬等微量元素，或甜食、碳水化合物摄入过多等有关。
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 近视度数一般要到20岁左右才会趋于稳定，而青少年的眼球正处于生长发育期，因此他们发生近视或是近视加深的几率很大。相对来说，成年人的视力状况应该比较稳定，但由于现在成年人看电脑、阅读时间过长，眼睛长期疲劳，造成近视不断加深也是不足为奇的。
二、减缓近视发展的五大方法
（一）养成良好的用眼卫生习惯，写字姿势要端正，距离书本一尺，不要趴着、躺着看书，不要在强光或昏暗的地方看书，且时间不要过长，每看书约30分钟就要让眼睛休息一会儿，向远处望一望。
（二）不应长时间看电视、电脑，玩电子游戏，许多人玩上瘾后就忘记了时间的存在，这是非常有害的。
（三）做眼保健操对于消除视疲劳、保护视力还是有作用的。但由于学生们年纪小，普遍认识不到眼保健操的重要性，做眼保健操只是应付差事，因此老师及家长要督促孩子做操到位。做眼保健操也不要局限于每天一、二次，每当看书时间久了或感到视力疲劳时都可以做做眼保健操。
（四）多参加些户外活动，比如打乒乓球、羽毛球等，多看看花草树木等绿色的植物，在视野开阔处极目远眺(但不要看蓝天白云等耀眼的物体)。
（五）定期检查视力，发现视力下降要及时养护，否则，有可能会加快近视增长速度。

秋夏交替时节，哪些疾病需要注意

九月 2, 2020

季节交替期间，由于天气多变，人体容易出现各种不适。尤其夏秋交替期间，不仅要预防肠胃炎、各种流行性传染病、感冒，还要时刻做好保暖工作，谨防慢性疾病复发。
哪些疾病需要注意
一、腹泻
夏季气温高，湿度大，食物容易变质造成感染性腹泻，而且夏季大家喜欢吃冷饮等冷食物，所以对肠胃的刺激比较大。
二、伤风
在夏秋季节，特别容易患上伤风感冒，这是因为白天和夜间的温差比较大，机体能量消耗大，致使人体免疫力和抵抗力下降。
三、慢性支气管炎
病毒或者细菌感染、物理化学刺激及过敏反应都会使支管收到伤害发生炎症，常发生于寒冷季节或者气温变化时。对于呼吸道比较敏感的人，一换季就“中招”！
四、胃肠型感冒
出现的症状有发烧、呕吐、腹痛、腹泻等。该病发病体征和胃肠炎很相似，容易引起混淆。预防胃肠型感冒，关键在于不要过度贪凉，胃肠型感冒患者需要多休息，要多吃清淡、易消化、营养丰富的食物，以降低对肠道的刺激。
五、心脑血管疾病
天气忽冷忽热易诱发心脑血管疾病。预防心脑血管疾病，要养成良好的生活习惯，戒烟戒酒、合理饮食、适当运动、控制体重等都是预防此类疾病的重要方法。
换季要注意哪些
1、进入秋季养生，首先调整的就是睡眠时间，改变夏季晚睡的习惯，争取晚上10时前入睡，比夏天增加1小时睡眠，早睡早起，通过睡眠调整身体状态，以适应秋季特点。
2、多进行体育运动可以选择散步、骑车、体操、慢跑、跳绳等，运动程度以全身稍微出汗即可。运动的同时可适度晒太阳，提高人体对氧气的利用率，以促进人体的血液循环，保持体内旺盛的新陈代谢。

开学季 | 我们呢共同守护孩子眼健康

九月 1, 2020

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 今天开学啦！家长在感叹“神兽们终于回笼”的同时，是否注意到孩子的视力变化呢？暑假期间，电子产品的频繁使用，不仅没让孩子的眼睛在暑假期间得到放松，开学后，还重新投入繁忙的学习生活中，这些高强度用眼，损害眼健康！
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 爱大爱表示，开学季，保护孩子的视力状况刻不容缓！首先我们得了解，近视不可怕，问题是很多家长对于佩戴眼镜保护视力存在误区！
误区一：戴镜后度数增长越厉害
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 在眼镜度数合适的情况下，近视度数是否加深，与眼镜本身并没有多大关系而是眼睛本身屈光状态改变导致的。只要导致近视的因素还在，比如缺少户外运动，近距离用眼时间过长，近视度数就会加深。
误区二：戴眼镜会导致眼球变形
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 单纯性近视，尤其是600度以下的中低度近视，一般不会有眼球突出的表现。600度以上的高度近视，近视度数越大，眼轴越长，当眼轴增长到一定程度就会出现眼球突出变形的表现。
误区三：出现视力下降，直接带孩子去配眼镜
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 家长应带孩子去正规医院做散瞳验光的检查，通过散瞳检查，分辨真、假性近视，在根据医生建议进行矫正治疗。
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 爱大爱表示：“孩子不管近不近视，都要注意保护眼睛视力！”在这里就有家长朋友可能有疑问，孩子没近视还有保护的必要吗？很多案例证明因为家长起初没在意等孩子视力出现问题才去医院检查的时候，孩子已经近视了所以说，预防才是最好的治疗。

新学期，近视预防小课堂

八月 31, 2020

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 暑假即将过去，又到了新一轮的开学季，许多孩子在暑期电子产品的疯狂轰炸后，患上了近视。爱大爱了解到，大部分学生在暑假期间视力出现下降。为什么会出现这样的情况？
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 假期常成为孩子们视力发展的“分水岭”，有些孩子利用假期，劳逸结合，注意用眼保健，下降的视力得到了恢复；但更多的孩子没有了功课的压力，开始疯狂地玩游戏、玩手机、看电视，导致视力明显减退，等开学时就要看不清前方黑板上的字了。
爱大爱小编表示，新学期快要开始了，现在该是为孩子调整视力的时候了。因此，提出了以下几点建议：
改变不良用眼习惯
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 很多孩子喜欢躺着看书、看电视，这对眼睛造成了损害。正确的方法应当是：调整坐姿，保证照明。每次持续阅读不要超过1小时，间歇时做眼睛保健操或者极目远眺，看电视距离不要太近，最好为电视屏幕对角线的4～5倍。控制好看电视、上网的时间。
加强户外活动
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 加强户外活动。应当鼓励孩子多进行室外活动，亲近大自然，在更多视野开阔、空气清新的环境得到彻底放松。
保持规律生活
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 保持良好的作息，饮食应当全面，可多补充维生素A、B、D，多吃胡萝卜、水果、海产品等。这些食品对视力保健会有帮助。切忌挑食，过于挑食的儿童如果缺乏某些微量元素也会导致近视。
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 如果发现孩子视力异常，比如眯眼、歪头等不良注视习惯，应当及时带孩子到医院就诊。如早期假性近视可以治疗恢复视力;而真性近视者要及时医学验光、医学配镜。切莫怀有“眼镜越戴越深”的错误观念，这会贻误最佳治疗时机。
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 暑假接近尾声，对儿童进行一次常规的眼部检查非常有必要，可以及早发现及早治疗。开学后，尤其是低年级的小学生，老师、家长应教育引导孩子养成好的用眼习惯，读书写字遵照“三个一”的规范(即：书本距眼睛一尺，胸部离桌子一拳，握笔时手指距笔尖一寸)，学习时注意劳逸结合，避免长时间近距离用眼。如果出现不良的用眼习惯要及时去做眼部检查，确定是否要矫正视力，是否配戴眼镜，避免快速加深度数。
