【爱大爱】如何有效预防近视眼度数加深

五月 18, 2022

&nbsp; &nbsp; &nbsp; 近视是一个视力概念，指目光所及的范围内看不清远物、却看清近物的症状。青少年学生近视已经成为了影响青少年健康的一个重要因素，近视不断加深的原因在于他们的学习负担过重，需长时间近距离用眼。那么，如何控制眼睛度数加深呢？

&nbsp; &nbsp; &nbsp; 1.在室内连续用眼如看书、看电视、上网等，在校学生每堂课结束后，走出阳台或学校走廊，望着远处光线强烈的地方，你会感到刺眼，眼睁不开，此时可闭上眼睛，稍作休息。当刺眼的感觉消失后，再重新睁开眼睛。望向远处（必须眼睛不觉得累，又能望清楚的地方）。以此类推，逐步把目标的距离拉远，以不能再看清楚目标时结束。
&nbsp; &nbsp; &nbsp;&nbsp;有条件者可固定同一目标，以比较视力提升的情况，当感觉望远处不刺眼，今天能看清楚昨天看不清的目标时，你的视力已经在提高。该作用是通过强光刺激眼球。
&nbsp; &nbsp; &nbsp; 2.每天起床一件事就是走出阳台，近距离看绿色植物，两眼睁大，眼睛与绿色植物相距小于10厘米，稍作停顿，然后俯视植物，转动眼睛，四处张望，彷如在植物丛中寻找蚂蚁、小虫等，看看叶子是否被虫子咬过。以绿色近距离刺激眼球，以假意寻找蚂蚁来活动眼球。
&nbsp; &nbsp; &nbsp; 3.头固定，两眼自然闭合(或张开)，眼睛尽可能向上望。当感到有麻痹感时，停顿5秒，缓慢平复眼睛(稍作休息)。当麻痹感消失后再依次依上法向下向左向右望。每个方向做完后必须休息片刻。当四个动作都完成后，两眼球顺时针转三圈，逆时针转三圈。
&nbsp; &nbsp; &nbsp; 4.配合明目养眼的食物。适用于25岁甚至30岁以前的青少年近视患者，只要坚持，视力将在不知不觉中得到提高。建议平时尽量不要戴眼镜，只在必要时才戴，譬如上课。让眼睛有更多的自我调节的机会，以锻炼它的功能，逐渐摆脱眼镜。

门诊各项眼检查的意义

五月 18, 2022

眼睛与身体各个器官密切相关，许多疾病的发生常会通过眼睛反映出来，可能是眼底有问题，也可能只是年龄大了眼睛发生白内障，但这些眼睛问题都需要通过眼检查才能做出诊断。但是为什么医生会先让我们做一系列的检查呢？
下面一起来看看各项眼检查的意义吧~
门诊为什么先要查视力?因为在日常生活中，大部分人都使用两只眼睛视物，所以很多患者无法感知到其中一只眼睛视力的严重下降。当发现问题时，很可能为时已晚。此外，眼病的主要症状之一是视力下降，视力检查是发现眼部疾病并进行诊断的重要依据。
为什么要测眼压?
眼压过高会引起视神经的损伤萎缩，进而造成各种视觉的障碍和视野的缺损，而且这种损伤是无法逆转的。眼压高很有可能与青光眼相关。眼压检查是发现青光眼的三大重要检查之一，也是最简单的检查。因此就诊患者应作眼压检查，以排除青光眼的可能性，以及早发现眼部问题及治疗。
为什么要做裂隙灯检查?
裂隙灯检查可以清楚地观察眼睑、结膜、巩膜、角膜、前房、虹膜、房角、瞳孔、晶状体及玻璃体前1/3、眼底、视盘、黄斑、视网膜等眼底关键性结构，通过检查可以明确病变位置、病变性质、区域及其严重程度。因而裂隙灯检查是眼科检查中必不可少的重要环节。
为什么要查眼底?
由于眼底可以反映出身体其他各器官的疾病。医生通过检眼镜可看到玻璃体、视网膜、视神经乳头和视网膜动、静脉。从视网膜的组织结构、形态、血管变化，即可发现有无异常，从而推断患者患了什么病。眼底检查不仅对眼科疾病的诊断有重要意义，还能发现不同的眼底表现，对某些全身性的疾病，特别是内科和神经科疾病并作出诊断或提供线索。

不同干眼症状，治疗方式不同

五月 18, 2022

&nbsp; &nbsp; &nbsp; &nbsp;迎风流泪、畏光流泪、专注看电脑或手机时，两眼不觉得掉下眼泪……这是为什么呢?结膜炎、泪道堵塞、长时间使用眼药水等都有可能让患者出现迎风流泪的现象。但有一些患者因为常常流泪，被医生诊断为干眼症时疑惑不解，这是不是南辕北辙?因为各种原因导致泪膜不稳定和眼表受损及炎症反应的这类疾病统称为干眼症，常见表现有：眼睛干涩、发痒、灼痛感、畏光等等。

&nbsp; &nbsp; &nbsp;&nbsp;长时间盯着电脑、手机，眼睛眨眼次数减少，泪液分泌就会减少，角结膜干燥而引起眼睛的干涩、烧灼和异物感等。由于缺少泪液保护，眼睛表面更加敏感、更容易受到刺激，刺激泪腺在短时间反射性地分泌大量泪液，超出了泪道排出系统的负荷，而引起阵发性流泪，部分患者反而表现为见风或烟等袭击后反射性流泪。常处空调环境的上班族、常需要佩戴隐形眼镜人群、熬夜玩手机人群都是干眼症的易患人群。
&nbsp; &nbsp; &nbsp; 干眼症状引起的流泪一般见轻、中度干眼症：眼睛干涩难忍，在空调环境或者吹风，会刺激泪腺导致大量流泪。临床上，干眼症分为这些类型：
&nbsp; &nbsp; &nbsp; 水液缺乏型干眼：水液性泪液生成不足或异常而引起的干眼;
&nbsp; &nbsp; &nbsp;&nbsp;蒸发过强型干眼：因为睑板腺功能障碍、睑缘炎、视屏终端综合征、眼睑缺损或异常引起蒸发增加导致干眼的;
&nbsp; &nbsp; &nbsp; 黏蛋白缺乏型干眼：如药物毒性、化学伤、热烧伤等外伤对眼表及角膜的伤害导致的;
&nbsp; &nbsp; &nbsp;&nbsp;泪液动力学异常型干眼：因为泪液排出延缓、结膜松弛等引起的;
&nbsp; &nbsp; &nbsp; 混合型干眼：是临床上最常见的干眼类型，为以上两种或两种以上原因所引起的干眼。
&nbsp; &nbsp; &nbsp; 确诊干眼之后，医生会了解患者的生活工作环境和方式，如隐形眼镜佩戴情况、药物使用、眼部手术史、吸烟和饮酒情况、确定患者干眼的类型，选择合适的治疗方法进行个性化治疗。

表示：越来越多的人近视了就是因为我们不懂保护眼睛，长期对着电脑，对着电视,对着手机。造成了眼疲劳。
&nbsp; &nbsp; &nbsp;&nbsp;舒适的光线，可以得到良好的视觉信息，光线过强或过暗都会给眼睛带来不良的影响，因此青少年平常看书的书桌应有边灯装置，其目的在于减少反光，以降低对眼睛的伤害。
&nbsp; &nbsp; &nbsp; 青少年看书写字、看电视、用电脑，几个小时不休息，有的学生甚至到深夜才睡觉，这样不仅影响身体健康，使眼睛负担过重，容易引起调节性(或称功能性)近视，即假性近视。而且还会使眼外肌对眼球壁的巩膜组织产生压力，眼内压增高，眼内组织充血。因此，学生看书学习每隔50分钟休息片刻为宜。

&nbsp; &nbsp; &nbsp; 不要弯腰驼背，或趴在桌上看书，更不能躺在床上，侧着身看书。眼与书本的距离应保持30—35厘米，身体与课桌保持一掌——大约10厘米的距离，书本与课桌的角度要保持在30—45度。如书本水平放在桌面上，看书时就要向前稍低头，这样就容易把书本移近眼睛，加重眼睛负担2—3倍，从而引起颈部肌肉和颈背的疲劳，而不自觉的向前倾斜，长期下去就会导致视力下降。
&nbsp; &nbsp; &nbsp;&nbsp;尽量减少与对人眼产生辐射的电视、电脑、游戏机等电器设备的接触，因为，显象管辐射出的X射线可大量消耗视网膜中的视紫质，可以使视力明显减退。电脑最好选用液晶显示器，以减少电磁波对眼睛的伤害。经常玩游戏机的同学更易损坏视力，而且自幼即玩游戏机的低视力同学，配镜连矫正视力都上不去，原因就在于视网膜和黄斑部的功能受到了损害。
&nbsp; &nbsp; &nbsp; 作息时间要有规律，睡眠要充足。睡眠不足会导致眼睛结膜充血、分泌物增多、畏光流泪，眼酸痛等结膜、角膜炎症。应尽量避免风沙、烟尘、紫外线、红外线、化学物品、医药用品等对眼睛的伤害。个人卫生要保持清洁，毛巾、脸盆、手帕等个人物品，要专人专用，尽量不用他人物品，以避免造成交叉感染，引起眼部疾病，导致视力下降。

&nbsp; &nbsp; &nbsp;&nbsp;营养摄取要均衡，偏食或过多摄入糖和蛋白质，从而缺乏如锌、钙、铬等微量元素，都不利于视力健康。预防方法是多吃一些蔬菜、水果、肝脏、鱼等食品。
&nbsp; &nbsp; &nbsp; 有的青少年喜欢边走路边看书，或在行走的车厢里看书，这样对眼睛很不利，因为车厢在晃动，身体在摇晃，眼睛与书本距离无法固定，加上照明条件不好，加重了眼睛的负担，经常如此就有可能引起近视。

&nbsp; &nbsp; &nbsp; 日抛说通俗点就是一次性的隐形眼镜，当天用完就可以抛弃了，不存在清洗浸泡的麻烦，既干净又卫生，可以说是很方便了；月抛就是佩戴一个月的隐形眼镜，在一个月期间你每天都可以戴，但是每天都得清洗浸泡，半年抛、年抛以此类推。
&nbsp; &nbsp; &nbsp;&nbsp;其实，就严格意义上来说，只有更换周期在三个月以下的隐形眼镜才配叫做“抛弃型”，所以半年抛和年抛只是传统的隐形眼镜，不管是传统型还是抛弃型，隐形眼镜都是周期越短越好的，一款好的隐形眼镜除了看他的材质还有最关键的一点就是看他的含水量，含水量的多少直接决定佩戴的舒适性，所以周期越短锁水功能越好。

&nbsp; &nbsp; &nbsp;&nbsp;最后，尽量建议大家不要长时间的佩戴隐形眼镜，毕竟对眼睛还是有损伤的，如果不出门尽量戴框架眼镜，要多多注意眼部卫生，做个电眼美女！

保护视力的训练秘笈都有什么

五月 18, 2022

近视、散光是很多人都有的，这样的视力问题引发原因有很多，常见就是后天因素，后天对自身眼睛没有保护好，就会引发这些视力问题，现象近视的人群比较多，而且严重视力的人，在数目量每年都不断上涨，因此对保护视力是很关键的，那保护视力的训练秘笈都有什么呢?

保护视力的训练秘笈：
1、远方凝视：
找一处10米以外的草地或绿树：绿色由于波长较短，成像在视网膜之前，促使眼部调节放松、眼睫状肌松弛，减轻眼疲劳。不要眯眼，也不要总眨眼，排除杂念、集中精力、全神贯注的凝视25秒，辨认草叶或树叶的轮廓。接着把左手掌略高于眼睛前方30厘米处，逐一从头到尾看清掌纹，大约5秒。看完掌纹后再凝视远方的草地或树叶25秒，然后再看掌纹。10分钟时间反复20次，一天做三回，视力下降厉害的要增加训练次数。
2、晶体操
●转眼：双手托腮，让眼球按上、下、左、右的顺序转动10次，接着再逆时针、顺时针各转动10次。
●找一幅3米外的景物(如：墙上的字画等)，同时举起自己的左手距眼睛略高处伸直(约30厘米)，看清手掌手纹后，再看清远物，尽量快速的在二者间移动目光，往返20次。
3、推拿操
采取坐式或仰卧式均可，将两眼自然闭合，然后依次按摩眼睛周围的穴位。要求取穴准确、手法轻缓，以局部有酸胀感为度。
1.揉天应穴：用双手大拇指轻轻揉按天应穴(眉头下面、眼眶外上角处)。
2.挤按睛明穴：用一只手的大拇指轻轻揉按睛明穴(鼻根部紧挨两眼内眦处)先向下按，然后又向上挤
3.揉四白穴：用食指揉按面颊中央部的四白穴(眼眶下缘正中直下一横指)。
4.按太阳穴、轮刮眼眶：用拇指按压太阳穴(眉梢和外眼角的中间向后一横指处)，然后用弯屈的食指第二节内侧面轻刮眼眶一圈，由内上->外上->外下->内下，使眼眶周围的攒竹鱼腰、丝竹空、瞳子寥、球后、承泣等穴位受到按摩。对于假性近视、或预防近视眼度数的加深有好处
这些就是保护视力的训练秘笈，选择这些保护视力方法，在改善视力上都是有很好帮助，不过在对这些方法使用上，都是要长期的进行，这样对自身视力保护有很好帮助，而且保护视力过程中，饮食上也要多吃一些利于视力的。
