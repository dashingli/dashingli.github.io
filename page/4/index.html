床头灯的使用事项

五月 18, 2022


&nbsp; &nbsp; &nbsp; 床头灯不能常用，一般情况下照明强度不足，照明分布不均及光源闪烁不稳定和有色光源(如红色或橘红色)照明等，都易引起相当程度的视觉干扰，而导致视疲劳症状。此外，光源的闪烁不定，以及光照的角度并非正常的垂直照射角度，不但对眼睛刺激最大，更容易引起心理烦躁不适，甚至于影响安然入睡。小孩在这样的灯光下看书，也影响视力，引起近视、散光等情况的发生。
&nbsp; &nbsp; &nbsp;&nbsp;人们在床上主要以半卧位状态阅读或看电视，这种体位也易出现颈部疲劳和视觉疲劳。本来是想创造舒适的睡眠环境，反而带来了困扰。最好在床边设立一个立式的落地灯，让灯光能够在你的身前亮起。荧光灯管最好，频闪在80~150HZ左右为佳。
&nbsp; &nbsp; &nbsp; 光源闪烁对视觉系统有刺激作用，容易引起头痛等。严重程度与光源闪烁的强度、频率、持续作用时间有关。目前多主张采用高频化的电子镇流器(>10000Hz)以消除频闪，大多数体验者认为无频闪直流荧光台灯光线较柔和，对眼睛刺激小，在灯下阅读60分钟后，眼睛的疲劳感觉较轻。
&nbsp; &nbsp; &nbsp; 客厅也并非光线越亮越好。照明光线过强，耀眼引起眩光现象，也易发生视觉障碍，导致视疲劳。一般认为家庭里灯光合适强度是，在1000l勒克斯以下，照度越高越好。照度在说明书中就有体现，也可用照度计测量。

见风流泪也是干眼症？

五月 18, 2022

&nbsp; &nbsp; &nbsp; 很多朋友平时会有见风流泪的情况出现，到医院一查，医生却告诉他是干眼症，这便令人费解，为什么明明是流泪，却诊断成干眼症呢？
&nbsp; &nbsp; &nbsp; 其实，人的泪腺分泌有两种形式，一种是基础分泌，另一种是应激分泌。基础分泌的眼泪虽然量少，却无时无刻不在进行，也是滋润眼球的主要途径，因此基础分泌是极其重要的，也是医生诊断干眼症的主要检查对象，应激分泌实在受到外部刺激时泪腺的一种反应性分泌，比如大哭，或者迷眼时分泌的眼泪，还有本文主要学要解释的见风流泪现象都属于此种分泌方式。这种分泌方式虽然分泌的眼泪量很大，常常要溢出眼眶，形成明显的流泪，但是它并不是每时每刻都存在的，因此它并不能替代基础分泌而持久的润滑眼球。

&nbsp; &nbsp; &nbsp; 患有干眼症的人，往往眼泪基础分泌量较少，眼泪蒸发速度又较快，遇风时，风加速了眼泪的挥发，使角膜暴露在外，引起刺激反应，从而启动了泪腺的应激分泌，于是眼泪开始大量分泌，形成了明显的流泪。这就是所谓的“见风流泪”的形成原因。
&nbsp; &nbsp; &nbsp; 其实这种现象不一定出现在“见风”之后，任何可以加速泪液挥发的因素，都可以引起所谓的“流泪”，因此希望此类朋友在发现自己“流泪”时千万要仔细思考一下。切勿乱用眼药或接受错误的治疗方法。

预防青少年近视要加强体育锻炼

五月 18, 2022

&nbsp; &nbsp; &nbsp; 青少年近视的原因有很多，除了环境和遗传因素等原因之外，与青少年自身的素质也有一定的影响。爱大爱眼科专家介绍，青少年正处于身体生长发育时期，身体各部位器官变化较大。在此期间，如果没有足够的睡眠、合理的营养、新鲜的空气、适当的体育锻炼及广阔的视野，预防近视是十分困难的。因此，多参加体育锻炼对预防近视的发生和发展非常有益。
&nbsp; &nbsp; &nbsp;&nbsp;具体来说，作为中小学生要锻炼身体最好不过就是坚持做课间操。此外，学校也应组织中小学生进行一些多样化的文体活动，如爬山、野营、打球、跑步和游泳等，以增强身体素质。爱大爱眼科专家表示，较多的室外体育活动，由于注视远方的机会较多，就等于眼睛进行远眺训练。尤其是室外的球类运动，如篮球、足球、排球和棒球等，由于眼睛随时都在注意着远处、近处的球，这就像给眼部睫状肌做训练体操，是预防近视眼简便而有效的方法。放风筝能使眼睛注视远处目标，也是预防近视眼的有益活动。
&nbsp; &nbsp; &nbsp; 此外，弹跳运动也能改善视力，因为人在弹跳时，全身器官都进入运动状态，两眼的数条眼肌进行协调配合，使物像清晰地呈现在视网膜上，从而确保身体下落时的平衡姿态，也能调整视力。若长期坚持，视力有可能得到改善。（爱大爱,眼睛养护中心）

眼睛与内在健康息息相关

五月 18, 2022

从眼睛可以看出一个人的健康状况，机体的状况反过来也会影响眼睛的健康状况。任何一个脏腑的生理功能出现问题，都会影响眼睛的状况，因此要想让一双眼睛灵动生辉，内在机体的健康就必不可少。要保持内在的健康，首先要做到以下几个方面，下面爱大爱小编就跟您一起了解下有哪些方面。
&nbsp;　　1.充足的睡眠
&nbsp;　　睡眠是身体消除疲劳的最主要方式，眼睛也在睡眠的过程中得到充分的休息和滋养。绝大多数健康长寿的老年人都拥有一个良好而正常的睡眠。对于女性来讲，充足的睡眠还有利于皮肤美容，这是因为在睡眠过程中皮肤毛细血管循环增多，其分泌和清除过程加强，加快了皮肤的再生，所以睡眠有益于皮肤美容。
&nbsp;　　2.健康均衡的饮食
&nbsp;　　健康均衡的饮食对于人体的重要性不言而喻，健康均衡的饮食原则遵循七大营养素均衡的原则。亲近自然，均衡营养，合理取用。
&nbsp;　　3.坚持运动
&nbsp;　　坚持规律的运动，最好是每天保持一定的适度运动，可以提高机体的新陈代谢水平，增强心肺功能，提高机体的免疫状况，同时在运动过程中可以释放压力，调节精神状况。&nbsp;
4.保持良好的精神状态
&nbsp;　　特别是女性，如果抑郁悲观，可能直接影响到激素的分泌，引起激素分泌失调，而这些化学物质的分泌失常对于免疫系统也会产生影响。长期的不良情绪或者过于剧烈的精神刺激可能成为肿瘤等恶性疾病的诱发因素。乐观的生活，是健康的开始。
&nbsp;　　5.定期身体检查&nbsp;
定期做身体检查可以做到对疾病的早发现、早诊断、早治疗。尤其是进入中年的女性，定期的身体检查尤为重要。定期的眼睛检查也有利于眼部疾病的早诊断和早治疗，患有糖尿病、高血压等疾病的女性更是要定期进行眼部的检查。
&nbsp;　　总结
&nbsp;　　此外，保持良好的用眼卫生也很重要。无论是看书还是用电脑、看手机、玩ipad都不宜时间过长，每隔一个小时左右就应该停下来休息10—15分钟来极目远眺，这样可以让眼睛得到充分的休息。此外要注意不要在光线下看书、看电子产品。还有人喜欢躺在床上看书、玩电子产品，姿势不良容易导致斜视或者加重眼睛的负担。

IT大神们，你们知道怎么保护你们的双眼吗？

五月 18, 2022

&nbsp; &nbsp; &nbsp; IT一族，也就是每天或长时间面对电脑工作/学习的一类人。IT一族，除了每日久坐以外，眼睛受到的压力也是非常大的，尤其是本来就患有近视的人群。患了近视，并不代表你的眼睛就完了，只要护理得当，视力是可以改善或者保持的，只要不继续降下去，就是好的开始。
&nbsp; &nbsp; &nbsp; 很多朋友都想改善视力情况，而手术又不是一个随便可以做的选择，其实可以通过日常饮食来改善视力。下面，爱大爱小编为你推荐几种有助改善视力的食物。
1、西红柿改善视力
&nbsp; &nbsp; &nbsp;&nbsp;西红柿有助改善视力。维生素A和C不仅有助于提高视力而且也可以帮助预防夜盲症，而西红柿则是维生素A和C的极好来源。最近的一项研究表明，番茄的高维生素A含量可以帮助预防黄斑变性和严重眼病。此外，食用西红柿可以降低患上白内障的风险。西红柿丰富的植物化学物质、抗氧化剂番茄红素、叶黄素和玉米黄质，能保护眼睛不受光线损害。
2、海带改善视力
&nbsp; &nbsp; &nbsp; 海带除含碘外还含有1/3的甘露醇，晒干的海带表面有一层厚厚的“白霜”，它就是海带中的甘露醇，甘露醇有利尿作用，可减轻眼内压力，用来治疗急性青光眼有良好的功效。其他海藻类如裙带菜等食物也含有甘露醇，也可用来作为治疗急性青光眼的辅助食物。
3、珍珠改善视力
&nbsp; &nbsp; &nbsp;&nbsp;珍珠含95％以上的碳酸钙及少量氧化镁、氧化铝等矿物质，并含有多种氨基酸、如亮氨酸、蛋氨酸、丙氨酸、甘氨酸、谷氨酸、天门冬氨酸等，珍珠性味甘咸寒，用珍珠粉配龙脑、琥珀等配成的“真珠散”可抑制白内障的形成的功效。
4、黑豆改善视力
&nbsp; &nbsp; &nbsp; 黑豆能缓解眼睛疲劳，也有助于防止视力下降。因为黑豆中富含抗氧化成分——花色素和对眼睛有益的维生素A。醋泡黑豆是一种不错的饮食方法，其中陈醋能促进黑豆中的营养元素溶出，有助于高效摄取营养成分。
&nbsp; &nbsp; &nbsp; 最后，不管是体力劳动还是脑力劳动，身体过度劳累后都易使眼压波动，所以要注意生活规律，劳逸结合，避免过劳，也要少在光线暗、光线弱的环境中工作或者娱乐。在暗室工作的人，每1～2小时要走出暗室或适当开灯照明。情绪易激动的人，要少看电影，看电视时也要养成在电视机旁开小灯照明的习惯，形成健康双眼的生活新方式。

近视和散光两者的区别，你知道多少？

五月 18, 2022

在现代生活中很多孩子的用眼习惯都不是很好，最后去医院检查后发现孩子近视、散光了。得了近视或散光，这可不是什么好事，近视散光度数持续加深会对孩子的日常生活产生影响。
得了近视与散光，似乎眼前的世界就被蒙上了雾气。有人或许会有所疑惑，同样是影响视力，近视和散光哪个更严重？在此爱大爱小编就来给大家讲解一下近视和散光的相关内容。
近视和散光两者的区别，你知道多少？
首先有一点需要跟大家明确的就是近视和散光谈不上哪个更严重一些，这两种眼疾都有轻重之分，不管是被确诊为得了哪种眼科疾病都必须尽早接受矫正治疗。
临床上近视眼也被称为短视眼，近视眼只能在近处看到了一些远的距离就看不清楚了。
近视也分为真性近视、假性近视和混合性近视三种类型。近视眼的人除了眼镜会有度数之外，还会出现晚上视力变差、眼前有漂浮物、闪光感等症状，并且绝大多数近视眼的人眼底都会出现病变。
散光的人眼球在不同的子午线上屈光力也是不太一样的，光线进入眼球最终形成两条焦线和相当小的弥散斑点，这种现象被称为散光，进入眼球里面的光不能在视网膜上面形成焦点。
临床上散光又分为规则散光和不规则散光两种类型，其中最大屈光力和最小屈光力不相互垂直的成为不规则散光。
规则散光又分为逆规散光、斜向散光和顺规散光三种情况。顺规散光指的是最大屈光力主子午线在90°±30°位置，逆规散光位置是在180°±30°，剩下来的就是斜向散光了。
综上所述我们晓得了无论散光还是近视都有轻重之分，不能单纯地比较这两者谁严重一些。爱大爱希望家长不论发现自己孩子近视还是散光都要尽快去医院接受治疗，并且要教孩子合理用眼，多吃对眼镜有好处的食物，如胡萝卜等。

高度近视患者孩子会近视

五月 18, 2022

“全面实施一对夫妇可生育两个孩子政策，积极开展应对人口老龄化行动”，自从国家全面放开二胎政策以来，很多人都在讨论到底要不要生？什么时候生？然而，从眼科学角度来看，令人关注的还是近视遗传。近视妈妈怀孕是否会遗传给孩子？遗传几率有多大？专家表示，如果父母均为高度近视，孩子高度近视发生率很可能达到85%。
近视的遗传性这么大，我们应该如何保护二胎的视力？要弄清楚这个问题，首先要知道近视的几个主要原因：
1.遗传因素：
父母高度近视的孩子比普通孩子近视风险要高。如果夫妻一方是高度近视，孩子近视的几率约50%；夫妻双方都是高度近视，孩子近视的几率高达85%以上。
2.环境因素：
缺乏自然光照射，在光线太亮、太暗、眩光、反光的地方看书，在颠簸的环境下阅读等，在不合适的环境下用眼的行为。
3.行为因素：
过少的户外运动和过多的近距离用眼、长时间看书玩手机，不正确的写字姿势，睡眠时间少等行为都会加大近视的风险。

除遗传因素不可控外，其他近视的因素都是家长们可以防范，因此想要二胎不近视，父母们在孩子出生后，要注意观察孩子的眼睛，从以下几个方面来预防近视：
Tip1：增加户外运动时间
研究表明，增加户外活动能有效降低近视眼的发病率。因此，对于想预防近视的孩子，户外活动时间至少为2小时，3小时更佳。
Tip2：建立眼屈光发育档案
屈光发育档案就是给孩子眼睛做一个屈光发育评估，看看孩子有没有近视、远视和散光等。六七岁之前的儿童都是低度远视眼，其屈光状态会随着眼球的发育而变化，所以我们常见的近视眼是完全可以提前预测的。为详细了解孩子的屈光变化情况，及时发现屈光发育异常，定期带孩子进行屈光检查，并建立完善的屈光发育档案，是非常有必要的。
爱大爱建议，36岁开始每年做一次全面视光检查并建立屈光发育档案。在孩子3岁以后，家长应每年带孩子到专业的眼科医院做12次全面视光检查，包括屈光度数、眼轴的长度、角膜的弧度，看看孩子是否属于高危近视眼人群，通过历年的检测合理干预近视的发生和发展。

防近视先防“电脑虫”

五月 18, 2022

&nbsp; &nbsp; &nbsp;&nbsp;全国低视力者600多万，儿童斜弱视者1000万。其中青少年近视率平均达40%。大学生近视率远远超过70%。在“爱眼”活动上，眼科专家们公布了这样的数据。

&nbsp; &nbsp; &nbsp; 爱大爱提醒，防治青少年近视还需先防“电脑虫”。随着科技的发展，青少年应用电脑网络查阅资料、网上学习的时间越来越长，很容易造成眼睛疲劳。眼睛疲劳程度与学习时间长短有关，与眼睛疲劳密切相关的近视也随之高发。因此，使用网络的家庭，一定要监督孩子避免用眼疲劳。
&nbsp; &nbsp; &nbsp;&nbsp;比较有效的预防近视的办法就是，在学习的过程中，养成学习中间休息的习惯，每1小时站起来，看看窗外、喝喝水、扭扭腰什么的，就可以达到不让眼睛连续工作的效果。另外，电脑屏幕的清晰度是否适当，环境的光线是否柔和，桌椅的高度及舒适度是否能配合屏幕的高度，这些因素如果控制不好都可能会使眼睛容易疲劳，从而引发或加深近视。
&nbsp; &nbsp; &nbsp; 最后还要提醒，如果调节了以上因素后，孩子还是学习一会儿就感到眼睛十分疲劳，则必须请教眼科医生，做进一步检查。
&nbsp; &nbsp; &nbsp;&nbsp;除此之外，专家还提醒广大青少年朋友，要减缓、预防近视眼，还要经常参加户外活动，养成健康、规律的饮食习惯，提高身体的各项免疫力，从而把近视眼发病的机会降到最低。

&nbsp; &nbsp; &nbsp; 就好比一部机器，刚出厂的时候是崭新的，长时间的用就会老化沾满油渍灰尘，但是，如果你适度使用而且经常上油保养擦拭，机器就会一直很新。

&nbsp; &nbsp; &nbsp;&nbsp;倒一杯菊花茶水，记得要用滚滚的水，然后冲茶，趁冒热气的时候，把眼睛保持一定的距离放在茶花水下，然后用热气去熏自己的眼睛处，这样也能起到很好的缓解疲劳的作用。但，记得不要伤着眼睛了

青少年真性近视有哪些恢复方法

五月 18, 2022

&nbsp; &nbsp; &nbsp;&nbsp;真性近视是目前青少年比较常见的视力问题，对日常生活和学习都会造成影响，因此需要及时进行恢复保健。随着青少年近视技术的发展，近视恢复的方法非常多。那现在社会上，青少年真性近视有哪些恢复方法呢？爱大爱为了让大家更好的选择近视恢复的方法，下面为大家介绍各种方法。

&nbsp; &nbsp; &nbsp; 1.佩戴眼镜：在配镜之前，首先要通过检影难光弄清近视的真实度数。对于青少年配镜验光要在睫状肌麻痹下进行，以控制调节作用，排除假性近视。配镜的原则应采取同样可使近视眼的视力矫正到最佳视力的最低度镜片。一般低于6.00D的近视眼，要充分矫正并经常配戴;高度近视者，既要完全矫正以获得较好视力，但又往往不能耐受，因此，只好降低镜片度数(一般在1.00D～3.00D之间)，争取能够保持舒适和双眼视觉功能。
&nbsp; &nbsp; &nbsp;&nbsp;2.角膜接触眼镜：配戴接触镜可以增加视野，有较佳的美容效果，又可使两眼屈光参差明显减少，使之维持双眼视觉功能。青少年近视者，不但可用接触镜增加视力，还可以压迫角膜防止近视继续发展。但一定要注意清洁卫生，按要求消毒保养和经常更换。
&nbsp; &nbsp; &nbsp; 3.手术恢复：有些人一旦近视了需要恢复视力，就想着去做手术。激光手术是目前比较常见的一种恢复方法，爱大爱眼科专家介绍，以上这些近视恢复方法虽然可以暂时缓和青少年度数增加，但是都存在一些弊端。譬如说，佩戴近视眼镜如果度数不合适，或者佩戴的时间过长，也会造成近视度数加深。而激光手术并不适用与青少年，青少年眼睛还处于非常脆弱的时期，发育并不完全，做手术恢复，很容易伤害眼睛，造成更严重的后果。
