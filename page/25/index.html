爱大爱提醒家长们：注意孩子视力健康问题

五月 18, 2022

，但是许多家长们在孩子检查出现低度近视、低度远视或低度散光时，是否需要配镜时，家长们才会在心里打一个大大的问号，这是为什么？
儿童低度远视戴镜问题
&nbsp; &nbsp; &nbsp; 远视者看近时需要调节，而且距离越近需要的调节越多，调节幅度就是眼睛能使用的最大的变焦能力，相当于调节的储备。当调节幅度足够大时，远视不需要配镜；当调节幅度不足时，远视需要配镜，否则很容易视疲劳。儿童调节幅度比较强，低度远视可轻易代偿，可以不用配镜。而且儿童低度远视还是对近视的一种有效保护，即：低度远视的儿童以后不容易近视，或近视发生的年龄可推迟。

儿童低度散光配镜问题
&nbsp; &nbsp; &nbsp; 一般来说，低度逆规散光也要矫正，单纯的低度顺规散光（≤0.75D）可不配镜。如果除了散光外还有近视，那就要按近视的配镜原则考虑给处方。
儿童低度近视配镜问题
&nbsp; &nbsp; &nbsp; 近视和远视不一样，近视眼成像在视网膜前。越使用调节，像点越向前移动，距离视网膜越远，越模糊，所以近视眼是没有调节机制代偿的。
&nbsp; &nbsp; &nbsp; 第一，如果没有眼位、双眼视异常等问题。如果低度近视能轻松看清黑板，也不一定需要戴镜。如果上课看不清黑板，或者看黑板时很吃力，则上课时戴镜，其他时间可不戴。
&nbsp; &nbsp; &nbsp; 第二，如果单眼、双眼的裸眼视力都差（单眼裸视≤0.6；双眼视力≤0.8），则应该戴镜。
&nbsp; &nbsp; &nbsp; 第三，如果同时伴有斜视、隐斜视、AC/A异常等问题，则根据具体的检查结果决定如何配镜。
&nbsp; &nbsp; &nbsp; 总之，是否戴镜，要结合孩子的实际用眼需求和视光检查结果来决定。视光检查不仅包括屈光检查，还包括调节、双眼视功能等。所以，儿童配镜还是要找专业的验配机构。
