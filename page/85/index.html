爱大爱提示：司机开车用眼安全条例

六月 14, 2019

资料显示，司机中有40%或轻或重地患有夜盲、近视、惧光等眼科疾病，可见，光污染日渐成了伤害司机视力的罪魁祸首。因此，驾驶员在开车时要注意眼睛保健，眼睛出现疲劳，要马上要缓解。
&nbsp; &nbsp; &nbsp; 汽车驾驶员预防眼疲劳保护视力是很重要的，对安全行车顺利完成驾驶任务有直接关系。汽车驾驶员要“眼观六路、耳听八方”，必须保持有良好的视力才能确保行车安全。眼疲劳也就是视疲劳，它的症状有双眼干涩、眼皮下垂、眼球酸痛、眼眶疼痛、继而视物模糊、出现双影，较严重的还出现头晕、头痛、恶心、视力下降等症状。

&nbsp; &nbsp; &nbsp;&nbsp;避免视疲劳就要减少连续开车时间，长途行车时，要做眼保健操或远眺绿色树木等。驾车每间隔1～2小时就应休息片刻，用食、中两指指腹向鼻侧轮刮眼眶3～5圈，用两手中指尖点按眉毛外侧上的凹陷处(新明穴)按摩30～50次，在车上或下车后闭目养神片刻。
&nbsp; &nbsp; &nbsp; 看完电视最好能休息一两个小时再出车。因为根据测定，看几小时电视后，人体血液中的维生素A会减少一半，保持在20毫克左右，这时只有27%的人对黑暗的适应能力正常，视觉变得迟钝，辨色能力减弱，视力甚至会降低30%。看1小时电视，眼睛视力大约要经过30分钟才能恢复正常。
&nbsp; &nbsp; &nbsp;&nbsp;强光反射会对驾驶员的眼睛造成强烈刺激。因此，驾驶员应佩戴驾驶镜。一般来说，驾驶员选购眼镜作为驾驶镜的主要有墨镜、偏光镜和夜视镜。

突然发现眼睛近视怎么办

六月 14, 2019

近视怎么办?

首先，爱大爱提醒广大人群，预防需要从生活的注意事项做起
&nbsp; &nbsp; &nbsp; 1.眼药水是眼科疾病最常用的药物剂型之一，对于许多眼病，眼药水都有直接、快捷的治疗作用，用眼的时间越来越多，比如长时间看电脑、看电视、玩电子游戏等等，很容易患视觉疲劳症和干眼症。偶尔用一下眼药水还是可以的，但是不能经常用。
&nbsp; &nbsp; &nbsp;2.一定要注意养成良好的用眼习惯，姿势端正，不在乘车、走路或卧床情况下看书，用眼1小时后应休息10分钟左右并远眺，多看绿色，使调节得以松弛。平时注意营养，加强锻炼，增强体质，使眼和全身均能正常发育。
&nbsp; &nbsp; &nbsp; 3.一定要多补充维生素A、维生素B以及钙、铬、锌等等的微量元素，平时要注意少吃一些糖果以及高糖食品。而且要少吃海鲜类的食品，多吃水果蔬菜，有助于缓解眼疲劳。
&nbsp; &nbsp; &nbsp; 想要彻底的摆脱近视眼我们就需要预防近视眼的产生，我们一定要戒掉上时间对着电脑或者手机的习惯，同时饮食上面也许要多吃一些对于视力有好处的食物，例如蛋白质非常丰富的食物等，适当的做眼保健操也非常的有好处。

&nbsp; &nbsp; &nbsp; 由于学生的眼球发育还未成熟，这种持续性用眼活动，对眼球肌肉的伤害特别大，造成假期结束，孩子到学校上课时，忽然发现看不清黑板上的字，才知道已经得了近视。
为家长们简单阐述一下，为何暑假会加深近视的原因：
&nbsp; &nbsp; &nbsp;&nbsp;近视的主要原因一方面是遗传，另外一方面就是后天环境的影响，比如用眼习惯、光线、姿势等。暑假期间，有的孩子整天看电视，有的长时间看书，而手机、电脑、IPad几乎是孩子们假期时亲密的小伙伴，有的孩子暑假要“充电”，上各种辅导班，做各种作业，户外活动少，用眼过度及用眼不当现象十分普遍。

提醒家长们，青少年自身要培养视力养护的习惯，积极参加体育锻炼，多进行户外活动，增强体质，平时尽量不长时间接触手机和电脑，少看电视，用眼后可闭眼休息或看看远处的景物，做做眼保健操，养成良好的生活习惯，合理安排自己的学习时间。

什么是屈光不正

六月 13, 2019

视力养护中心下面将为我们做详细的介绍。
&nbsp; &nbsp; &nbsp; 屈光不正是指当眼球调节松弛状态下，来自5m以外的平行光线，经过眼的屈光系统折射后，不能在视网膜上清晰成像者。最常见的屈光不正主要有近视、远视及散光。

&nbsp; &nbsp; &nbsp;&nbsp;近视眼症状表现为看不清远处物体；远视眼症状表现为轻度远视患者远近视力均好，中高度远视患者远近视力都不好；散光是眼睛屈光不正的一种状态，造成散光的主要原因是角膜或晶状体表面弯曲度不一致，散光症状表现为看远看近都不清楚。
&nbsp; &nbsp; &nbsp; 屈光不正一般需要用镜片矫正，包括框架镜和角膜接触镜，部分病人也可用激光手术治疗。儿童配镜首先需要散瞳验光，这是因为儿童存在比较强的调节因素，不散瞳会因调节的存在而影响验光的准确性。散瞳一般不会对儿童眼球造成伤害，而且还可以降低假性近视对验光造成的影响，但选用哪种散瞳药和如何使用应遵从医生的指导。
&nbsp; &nbsp; &nbsp;&nbsp;引起屈光不正的原因有遗传因素、发育因素、环境因素以及各类眼病因素等，主要表现为视力下降、视力疲劳、视野变化、光觉色觉障碍、眼球改变、并发症等。另外镜架的选择也很重要，不能只注意美观，更要注意镜架的大小，形状是否适合自己的脸型，能否使双眼从镜片中心注视，否则会影响矫正的效果。

高度近视人群需要做哪些事情呢

&nbsp; &nbsp; &nbsp;1.需要去医院查眼底，这样可以预防视网膜变性，裂孔，那么，怎么区分眼底病变了？如果成年后你的视力还在以100度加深，很有可能是眼底病变。百度上面说什么高度近视会一直近视这个不可信。远离百度，不要给自己加重负担。
&nbsp; &nbsp; &nbsp;&nbsp;2.普通高度近视，成年后可以趋于稳定，所以大家不要害怕。保护好自己的眼睛，走路的时候尽量看远处，不要老是低着头。电子产品不要玩太久每40分钟休息20分钟，眼睛疲劳的时候可以用毛巾加温水热敷眼睛。补充一定的维生素，多吃对眼睛好的食物。

&nbsp; &nbsp; &nbsp; 3.不要勤换眼镜。为什么这么说了，一般的眼镜店配镜都是给你配到1.2的，1.2的清晰度对高度近视来说十分容易导致视疲劳，所以高度近视眼镜能够看到1.00.8都是比较合适的，这样佩戴舒服。
&nbsp; &nbsp; &nbsp; 4.不要相信其他学说。近视是没有治疗方式的！！！只能矫正，不管你做近视手术还是你配镜这些只能叫做矫正。高度近视都是真性近视，只能好好保护不要再加深。
&nbsp; &nbsp; &nbsp;&nbsp;5.再说说大家都想了解的近视手术，近视手术大致分为激光和晶体植入，激光又分为全飞秒等激光手术属于角膜手术，一般度数太高的话都不会让你做激光，都是让做晶体。但是本人不建议高度近视做手术。高度近视的话比较容易反弹风险也稍微大一些。

近视能带来哪些眼科问题呢

六月 11, 2019

&nbsp; &nbsp; &nbsp;&nbsp;青少年近视如果处理不及时，当发展到高度近视的时候就引起各种并发症，对青少年的身心健康造成更加严重的伤害。爱大爱表示，近视度数越高，引起的一些其他眼科疾病的可能性越大。那么，近视能带来哪些眼科问题呢？爱大爱对此作了以下总结：

&nbsp; &nbsp; &nbsp; （1）近视眼可并发症白内障：近视眼眼内营养代谢不正常，使晶状体的囊膜通透性改变，晶状体营养障碍和代谢失常而逐渐发生混浊，视力逐渐减退产生并发性白内障。这种白内障发展缓慢，以核心混浊和后囊膜混浊为主。
&nbsp; &nbsp; &nbsp; （2）近视眼可并发症玻璃体液化变性：玻璃体是无色透明胶冻状。当近视眼眼球增大，玻璃体却不会再增加，所以玻璃体不能充填眼内全部空间，出现液化，活动度增加，混浊，引起眼前黑影，诱发视网膜脱离。
&nbsp; &nbsp; &nbsp;&nbsp;（3）近视眼可并发症斜视、弱视：近视眼可引起外斜或外隐斜，如双眼近视度数相差大于300度者，易引起度数深的眼外斜和弱视。
&nbsp; &nbsp; &nbsp; （4）近视眼可并发症视网膜脱离：是近视眼最常见的并发症。由于近视眼眼轴伸长及眼内营养障碍，视网膜周边部常发生囊样变性、格子样变性等，变性区视网膜非常薄，极易发生穿孔，有的已经穿孔形成于性裂孔，再加上玻璃体液化、活动度增加，牵拉视网膜发生脱离。在视网膜脱离中，70%是近视眼。
&nbsp; &nbsp; &nbsp; （5）近视眼可并发症黄斑出血和黄斑变性：近视眼眼部血液供应差，视网膜缺血，视网膜产生一种新生血管生长因子，这些因子使视网膜下新生血管生长，这些新生血管管壁极薄，极易破裂出血，出血后形成黄斑出血。出血吸收后，新生血管可再破裂、再出血，多次出血后局部形成短痕，致黄斑变性，黄斑变性永久性损害视力。
&nbsp; &nbsp; &nbsp; （6）近视眼可并发症青光眼：近视眼眼房角处滤帘结构不正常，所以眼内的房水流出阻力较大，容易引起眼压升高。据统计高度近视眼30％有青光眼，这种青光眼会造成视力渐渐丧失。
&nbsp; &nbsp; &nbsp;&nbsp;由此可见，近视对青少年的危害是不容忽视的，应该引起警觉。爱大爱眼科专家表示，近视危害大，如果发现孩子患有近视的时候应该做好相应的预防和保护措施，防止眼睛度数加深，同时减轻近视对生活的影响。

高度近视需要注意哪些问题？

六月 4, 2019

表示，近视加深的最主要恶果是——高度近视。
按近视程度分，近视度数大于600度（儿童>400度）的屈光不正即称为“高度近视”。

高度近视需要注意哪些问题？
&nbsp; &nbsp; &nbsp; 1、高度近视需要配戴眼镜，如果不戴眼镜，近视的度数会越来越深。眼睛如果是真性近视，那是不可逆转的；还有一方面，近视患者不佩戴眼镜还会导致眼部增加或加深皱纹，因为近视患者看不清东西就会不自觉地眯着眼睛观察事物，以至于眼睛部位的面部肌肉长期处于紧皱的状态，皱纹也就随之而来了。
&nbsp; &nbsp; &nbsp;&nbsp;2、高度近视者别做剧烈运动，当高度近视患者进行足球、篮球等对抗性比赛，或者参加短跑等过于剧烈的运动时，外力的突然冲击，可通过玻璃体的传导，促使视网膜破裂，造成视网膜脱离。
&nbsp; &nbsp; &nbsp; 3、高度近视患者需注意用眼卫生，因为眼睛里面的细菌会导致眼疾的发生。因此，应该保证眼睛的卫生，不要用手揉眼睛。
&nbsp; &nbsp; &nbsp;&nbsp;4、有些人认为近视无所谓，高度近视也没什么。殊不知，高度近视患者不仅有近视问题，而且往住伴随着多种并发症；而度数越高，诱发并发症的可能性就越大。
高度近视患者需防范三大并发症
&nbsp; &nbsp; &nbsp; 1、飞蚊症：随着眼球的转动，常在眼前出现黑影飘动，就像蚊蝇在飞舞，这是玻璃体正常的胶体状结构发生液化的缘故。
&nbsp; &nbsp; &nbsp; 2、黄斑出血：由于眼球变大造成对眼球壁的不断牵拉，引起眼底视网膜和脉络组织的正常关系出现代谢失调，特别是黄斑区的小血管因牵拉等因素而出现破裂时，就会造成黄斑出血，使中心视力受到很大损害。此时病人自觉眼前有一块固定的黑影挡住视线。尽管出血经治疗可以吸收，但总会或多或少留下斑痕，影响视力。还有一些患者黄斑出现异常新生血管，反复出血，对视力损害更严重。
&nbsp; &nbsp; &nbsp;&nbsp;3、视网膜脱离：因眼轴变长，眼球后半部变薄，视网膜、脉络膜萎缩、变性，出现裂孔，导致出血和范围不等的视网膜脱离，从而丧失视力。同时黄斑区变性也是高度近视眼的最大危险，表现为视物变形，视力下降，最终导致失明。
&nbsp; &nbsp; &nbsp; 最后需要提醒的是，近视人群，特别是近视青少年养成良好的用眼习惯，多点户外运动，高度近视患者每年定期做一次专业的眼底视网膜检查，以便及时发现视网膜隐患并加以治疗，从而免除后顾之忧。

青少年真性近视有哪些恢复方法

六月 3, 2019

&nbsp; &nbsp; &nbsp;&nbsp;真性近视是目前青少年比较常见的视力问题，对日常生活和学习都会造成影响，因此需要及时进行恢复保健。随着青少年近视技术的发展，近视恢复的方法非常多。那现在社会上，青少年真性近视有哪些恢复方法呢？爱大爱为了让大家更好的选择近视恢复的方法，下面为大家介绍各种方法。

&nbsp; &nbsp; &nbsp; 1.佩戴眼镜：在配镜之前，首先要通过检影难光弄清近视的真实度数。对于青少年配镜验光要在睫状肌麻痹下进行，以控制调节作用，排除假性近视。配镜的原则应采取同样可使近视眼的视力矫正到最佳视力的最低度镜片。一般低于6.00D的近视眼，要充分矫正并经常配戴;高度近视者，既要完全矫正以获得较好视力，但又往往不能耐受，因此，只好降低镜片度数(一般在1.00D～3.00D之间)，争取能够保持舒适和双眼视觉功能。
&nbsp; &nbsp; &nbsp;&nbsp;2.角膜接触眼镜：配戴接触镜可以增加视野，有较佳的美容效果，又可使两眼屈光参差明显减少，使之维持双眼视觉功能。青少年近视者，不但可用接触镜增加视力，还可以压迫角膜防止近视继续发展。但一定要注意清洁卫生，按要求消毒保养和经常更换。
&nbsp; &nbsp; &nbsp; 3.手术恢复：有些人一旦近视了需要恢复视力，就想着去做手术。激光手术是目前比较常见的一种恢复方法，爱大爱眼科专家介绍，以上这些近视恢复方法虽然可以暂时缓和青少年度数增加，但是都存在一些弊端。譬如说，佩戴近视眼镜如果度数不合适，或者佩戴的时间过长，也会造成近视度数加深。而激光手术并不适用与青少年，青少年眼睛还处于非常脆弱的时期，发育并不完全，做手术恢复，很容易伤害眼睛，造成更严重的后果。

爱大爱眼睛养护：“六一”护眼篇

五月 31, 2019

其实一般儿童的眼睛是最亮的，越长大用眼越多眼睛就越暗淡无光，但是很多人生活习惯很好，注意休息和调整，年纪大也仍然眼睛很亮。
&nbsp; &nbsp; &nbsp; 就好比一部机器，刚出厂的时候是崭新的，长时间的用就会老化沾满油渍灰尘，但是，如果你适度使用而且经常上油保养擦拭，机器就会一直很新。
&nbsp; &nbsp; &nbsp;&nbsp;但是更多小孩是因为日常学习和生活的不良习惯导致眼睛疲劳让眼睛失去神采，再加上紫外线照射、电脑辐射导致白眼珠色素沉着的问题。常表现为巩膜处有泛黄斑点或是眼球比较暗淡。

&nbsp; &nbsp; &nbsp; 其实这些都是一些常见的问题，在日常生活中注意眼部健康，适量补充维生素，时间长了，眼睛一定会有意想不到的变化哦。
缓解眼睛疲劳的办法：
&nbsp; &nbsp; &nbsp; 眼睛处在疲劳状态下，有一种方法能帮助你缓解这种症状，那就是泡一杯浓浓的茶，然后加适量的盐，搅拌均匀后，用棉棒沾浓茶水，涂抹自己的眼睛四周，能及速地缓解眼疲劳的症状的。
&nbsp; &nbsp; &nbsp;&nbsp;眼睛感觉到不舒服或者是疲劳的时候，可以用热毛巾来敷。把毛巾沾热后拧水后，然后大面积把自己的眼部盖住，这样，能让眼睛四周的血液活跃，并能带动其新陈代谢，从而达到缓解疲劳的作用。
&nbsp; &nbsp; &nbsp;&nbsp;倒一杯菊花茶水，记得要用滚滚的水，然后冲茶，趁冒热气的时候，把眼睛保持一定的距离放在茶花水下，然后用热气去熏自己的眼睛处，这样也能起到很好的缓解疲劳的作用。但，记得不要伤着眼睛了
&nbsp; &nbsp; &nbsp; 经常有眼疲劳的人，可以在大量使用眼睛的时间内，隔半个小时或者是更短的时间内，就把眼睛从专注的地方里挪开，望向远处或者是看一些绿色的植物，这样，能让自己的眼睛平和，而不一直处在紧张状态中。
&nbsp; &nbsp; &nbsp; 爱大爱温馨提示：眼睛是心灵的窗户，拥有一双明亮有神的眼睛能给人留下深刻的印象同时也能提高自己的气质。所以每天休息的时候练练眼睛，让自己精神一整天!

防近视先防“电脑虫”

五月 30, 2019

&nbsp; &nbsp; &nbsp;&nbsp;全国低视力者600多万，儿童斜弱视者1000万。其中青少年近视率平均达40%。大学生近视率远远超过70%。在“爱眼”活动上，眼科专家们公布了这样的数据。

&nbsp; &nbsp; &nbsp; 爱大爱提醒，防治青少年近视还需先防“电脑虫”。随着科技的发展，青少年应用电脑网络查阅资料、网上学习的时间越来越长，很容易造成眼睛疲劳。眼睛疲劳程度与学习时间长短有关，与眼睛疲劳密切相关的近视也随之高发。因此，使用网络的家庭，一定要监督孩子避免用眼疲劳。
&nbsp; &nbsp; &nbsp;&nbsp;比较有效的预防近视的办法就是，在学习的过程中，养成学习中间休息的习惯，每1小时站起来，看看窗外、喝喝水、扭扭腰什么的，就可以达到不让眼睛连续工作的效果。另外，电脑屏幕的清晰度是否适当，环境的光线是否柔和，桌椅的高度及舒适度是否能配合屏幕的高度，这些因素如果控制不好都可能会使眼睛容易疲劳，从而引发或加深近视。
&nbsp; &nbsp; &nbsp; 最后还要提醒，如果调节了以上因素后，孩子还是学习一会儿就感到眼睛十分疲劳，则必须请教眼科医生，做进一步检查。
&nbsp; &nbsp; &nbsp;&nbsp;除此之外，专家还提醒广大青少年朋友，要减缓、预防近视眼，还要经常参加户外活动，养成健康、规律的饮食习惯，提高身体的各项免疫力，从而把近视眼发病的机会降到最低。
