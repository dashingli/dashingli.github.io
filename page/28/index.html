眼镜度数会越戴越深吗？

五月 18, 2022

&nbsp; &nbsp; 因为开学的原因，这几天经常有妈妈跑来问小编：“我们家孩子有点近视，度数不高，大约100度，要不要戴眼镜？会不会把眼球戴变形呢？会不会越戴越深呢？”
&nbsp; &nbsp; 这个话题验光师们在前线阵地中，相信也是几乎天天遇到，从上面的问题可以看出，其实大家对于眼镜是害怕的，并不是把它当成了一个可以帮助自己的工具,而是当成了洪水猛兽，我们应该怎样帮助消费者正确面对这个问题，今天我们有必要来聊一聊关于“眼镜”这个话题。

&nbsp; &nbsp; 6-8岁，孩子的眼球发育基本成熟，但视觉系统的功能并不成熟，如果孩子在低龄段视力下降形成近视，视网膜上的影像是模糊的，模糊的影像则无法给与视网膜视杆细胞和视锥细胞充分的刺激，时间久了视网膜细胞的敏感性会慢慢下降。
&nbsp; &nbsp; 因此！！！这个时候我们去测量孩子的矫正视力，往往不是很理想，如果长期近视不戴眼镜，严重的可能形成废用性弱视。
&nbsp; &nbsp; 所以如果孩子的眼睛近视了，首先我们应该选择配镜，同时验光师在测量时应该给与充分的矫正，以保证孩子戴上眼镜后，有足够清晰锐利的影像落在视网膜上，越锐利的影像给与视网膜细胞的刺激越充分，细胞敏感度越高，同时也保证了看近是调节与集合的协调性，能帮助孩子的视功能建立的更好。
其次戴眼镜会不会导致眼球变形呢?
顾客所说的变形通常指眼球向外突出突出原因有两个：
&nbsp; &nbsp; 一是遗传基因导致
&nbsp; &nbsp; 二是高度近视的原因遗传的因素我们无法控制，但可以用科学的手段干预孩子的视力变化，保证增长速度不要过快，从而不发展为高度近视。眼镜由于是非接触性的矫正方式，所以眼镜本身是不会导致眼球变形的。
关于度数增长
&nbsp; &nbsp; 眼镜是用来进行光学矫正，以帮助我们看清楚。孩子长度数的原因和两个因素有关：一个是遗传因素，一个是用眼环境，比如中近距离用眼量过大，用眼环境不合理，过度疲劳等，科学合理的配镜是不会导致度数过快增长的。
总结
&nbsp; &nbsp; 所以一旦确认孩子眼睛近视，应该选择科学合理的配镜，配镜时不要人为要求验光师降低度数，同时建议验光师给与孩子充分的矫正，以保证视网膜成像足够清晰锐利，,让视网膜细胞获得足够充分的刺激，保持高度敏感性，这样对于视觉系统的建立更有帮助。
