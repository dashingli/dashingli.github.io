预防青少年近视要加强体育锻炼

五月 6, 2019

&nbsp; &nbsp; &nbsp; 青少年近视的原因有很多，除了环境和遗传因素等原因之外，与青少年自身的素质也有一定的影响。爱大爱眼科专家介绍，青少年正处于身体生长发育时期，身体各部位器官变化较大。在此期间，如果没有足够的睡眠、合理的营养、新鲜的空气、适当的体育锻炼及广阔的视野，预防近视是十分困难的。因此，多参加体育锻炼对预防近视的发生和发展非常有益。
&nbsp; &nbsp; &nbsp;&nbsp;具体来说，作为中小学生要锻炼身体最好不过就是坚持做课间操。此外，学校也应组织中小学生进行一些多样化的文体活动，如爬山、野营、打球、跑步和游泳等，以增强身体素质。爱大爱眼科专家表示，较多的室外体育活动，由于注视远方的机会较多，就等于眼睛进行远眺训练。尤其是室外的球类运动，如篮球、足球、排球和棒球等，由于眼睛随时都在注意着远处、近处的球，这就像给眼部睫状肌做训练体操，是预防近视眼简便而有效的方法。放风筝能使眼睛注视远处目标，也是预防近视眼的有益活动。
&nbsp; &nbsp; &nbsp; 此外，弹跳运动也能改善视力，因为人在弹跳时，全身器官都进入运动状态，两眼的数条眼肌进行协调配合，使物像清晰地呈现在视网膜上，从而确保身体下落时的平衡姿态，也能调整视力。若长期坚持，视力有可能得到改善。（爱大爱,眼睛养护中心）

检测视力时，为什么应该将屏幕亮度调到最大？

五月 5, 2019


超级视力表设置功能中，有一项检测亮度调节，在检测视力时，我们究竟应该怎样设置这一参数呢？

解答这个问题，我们首先要从超级视力表APP开发的理论基础——标准对数视力表（GB-11533-2011）说起。在此项国标中，对检查视力时的照明调件进行了明确的要求：视力表应采用人工照明，如果采用直接照明法，照度应不低于300lx，如果采用后照法（视力表灯箱或屏幕显示），亮度应不低于200cd/m2。
就目前智能手机而言，最高的屏幕亮度是可以确保满足国标对亮度的要求，如果调节至较低的档位，可能会因为亮度不足而影响测量结果的准确性。所以建议您在检测时，最好将检测亮度调节至最大。
另外值得注意的一点是，不同手机的最大亮度可能或多或少都存在一定的区别，因此最好能长期用同一部手机进行测量，确保每一次检测都是在同样的条件下进行的，避免亮度因素导致视力测量值的波动，影响准确性。

OK镜对于青少年视力矫正真的OK吗？

五月 5, 2019

视博士护眼知识小讲堂第三期开讲啦，本期关注的话题是，市面上流行的OK镜对于青少年矫正视力，是否真的有用？


首先我们先来说说OK镜到底是什么。OK镜（Othokeratology lens）是采用了一种特殊逆几何形态设计的角膜塑形镜片，其内表面由多个弧段组成。镜片与泪液层分布不均，由此产生的流体力学效应改变角膜几何形态，逐步使角膜弯曲度变平、眼轴缩短，从而有效地阻止了近视的发展。

OK镜最早起源于美国，在发展过程中曾广受争议，近年来随着OK镜材质的不断优化，逐渐被医学界广泛认可，被誉为“睡觉就能控制和矫治近视的技术”。
OK镜的原理通俗来讲，实际上非常的简单粗暴，就是通过佩戴镜片时产生的外力作用，迫使角膜暂时性变平，减小角膜曲率，暂时性达到矫正近视的目的。

值得注意的一点是，相比于近视手术，佩戴OK镜只是一种对近视的暂时性矫正，如果停戴，视力可能会回复到佩戴前的水平。然而对于青少年而言，它最大的作用在于能够延缓近视发展。青春期时期的孩子，近视度数增长速率往往特别快，而OK镜能够有效抑制度数的增长，等发育生长期过去，就算停戴，也不会出现报复性近视度数加深。同时OK镜有一个最大的好处，就是只需夜间睡眠时佩戴，等到早晨起来摘掉镜片，便可拥有一整天的清晰视力。
说了这么多的好处，再来说说OK会有哪些潜在的危害。
首先，OK镜最大的危险因素在于目前市场鱼龙混杂，不少不法商家以次充好OK镜是与眼角膜直接接触的医疗器械，在购买时一定要保证其材质符合安全标准。2008年2月13日我国国家食品药品监督管理局批准国内首家生产塑形镜的企业，并进一步规范其使用的屈光范围及各类参数。配OK镜一定要去靠谱的机构，
其次，OK镜和眼镜一样，不是标准化的产品，需要经过专业的验配。很多无良商家，甚至完全不经过验配，就把OK镜卖给了客户，不仅无法达到矫正视力的目的，还可能对角膜产生损伤。
最后，OK镜的佩戴及护理也是使用过程中非常重要的一环。由于OK镜的佩戴时间是晚上睡觉前，不少家长因为嫌累，可能会让孩子三天打鱼两天晒网式地佩戴，极大地影响矫正效果。同时，OK镜的摘戴过程中，以及清洗都需要保持高度的洁净，不然很可能引起感染，这非常考验家长的耐心和细心程度。

见风流泪也是干眼症？

四月 29, 2019

&nbsp; &nbsp; &nbsp; 很多朋友平时会有见风流泪的情况出现，到医院一查，医生却告诉他是干眼症，这便令人费解，为什么明明是流泪，却诊断成干眼症呢？
&nbsp; &nbsp; &nbsp; 其实，人的泪腺分泌有两种形式，一种是基础分泌，另一种是应激分泌。基础分泌的眼泪虽然量少，却无时无刻不在进行，也是滋润眼球的主要途径，因此基础分泌是极其重要的，也是医生诊断干眼症的主要检查对象，应激分泌实在受到外部刺激时泪腺的一种反应性分泌，比如大哭，或者迷眼时分泌的眼泪，还有本文主要学要解释的见风流泪现象都属于此种分泌方式。这种分泌方式虽然分泌的眼泪量很大，常常要溢出眼眶，形成明显的流泪，但是它并不是每时每刻都存在的，因此它并不能替代基础分泌而持久的润滑眼球。

&nbsp; &nbsp; &nbsp; 患有干眼症的人，往往眼泪基础分泌量较少，眼泪蒸发速度又较快，遇风时，风加速了眼泪的挥发，使角膜暴露在外，引起刺激反应，从而启动了泪腺的应激分泌，于是眼泪开始大量分泌，形成了明显的流泪。这就是所谓的“见风流泪”的形成原因。
&nbsp; &nbsp; &nbsp; 其实这种现象不一定出现在“见风”之后，任何可以加速泪液挥发的因素，都可以引起所谓的“流泪”，因此希望此类朋友在发现自己“流泪”时千万要仔细思考一下。切勿乱用眼药或接受错误的治疗方法。

床头灯的使用事项

四月 26, 2019


&nbsp; &nbsp; &nbsp; 床头灯不能常用，一般情况下照明强度不足，照明分布不均及光源闪烁不稳定和有色光源(如红色或橘红色)照明等，都易引起相当程度的视觉干扰，而导致视疲劳症状。此外，光源的闪烁不定，以及光照的角度并非正常的垂直照射角度，不但对眼睛刺激最大，更容易引起心理烦躁不适，甚至于影响安然入睡。小孩在这样的灯光下看书，也影响视力，引起近视、散光等情况的发生。
&nbsp; &nbsp; &nbsp;&nbsp;人们在床上主要以半卧位状态阅读或看电视，这种体位也易出现颈部疲劳和视觉疲劳。本来是想创造舒适的睡眠环境，反而带来了困扰。最好在床边设立一个立式的落地灯，让灯光能够在你的身前亮起。荧光灯管最好，频闪在80~150HZ左右为佳。
&nbsp; &nbsp; &nbsp; 光源闪烁对视觉系统有刺激作用，容易引起头痛等。严重程度与光源闪烁的强度、频率、持续作用时间有关。目前多主张采用高频化的电子镇流器(>10000Hz)以消除频闪，大多数体验者认为无频闪直流荧光台灯光线较柔和，对眼睛刺激小，在灯下阅读60分钟后，眼睛的疲劳感觉较轻。
&nbsp; &nbsp; &nbsp; 客厅也并非光线越亮越好。照明光线过强，耀眼引起眩光现象，也易发生视觉障碍，导致视疲劳。一般认为家庭里灯光合适强度是，在1000l勒克斯以下，照度越高越好。照度在说明书中就有体现，也可用照度计测量。

建议您最好使用2.5米的视力检测距离

四月 25, 2019


超级视力表APP在检测视力的方式上，给予了用户很大的自由空间，提供了2.5m，2m和50cm三种选项。50cm的检测距离意味着，用户无需他人帮助，也能单手完成全部的视力检测操作。

然而在实际检测的过程中，我们建议您最好选择2.5m作为检测距离，并在他们的帮助下完成整个测量过程。这是因为，50cm只是一个预估的成年人手臂长度，不同的测量者可能有很大的差距。当选择50cm作为检测距离时，哪怕只是距离的估计上差了1cm，也会对检测结果带来2%的最终误差。
然而当检测距离设置为2.5m时，1cm的误差对检测结果的影响只有0.4%，极大地提升了测量结果的准确性。
另外需要请您注意的是，超级视力表APP是基于对数视力表国标进行研发的，在使用过程中测量的准确性与标准的纸质视力表完全一致。然而在实际使用过程中，用户对于检测距离的估计总是或多或少有些偏差，从而导致测量误差。
对于这一点，超级视力表APP最关注的其实并不是测量结果的绝对准确性，而且每一次测量结果之间的相对准确性，确切地来说，就是整体的视力变化趋势的准确性，这也正是我们最希望用户关注的点。所以，只要您可以确保每一次检测，都是在大致相同的距离间完成的，那么便可以保证视力变化趋势是准确的。

这些近视误区，作为家长你了解吗？

四月 25, 2019


家长的正确指导对于青少年的用眼健康至关重要，然而不少家长却有着许多常识性的误区。作为家长的你，对于这些口口相传的误区真的全都了解吗？快来跟着视博士一起学习，给予孩子们清晰光明的未来！

1.孩子年龄还小，即使有点近视长大了就好了

假性近视确实在青少年时期普遍存在，然而等到家长察觉孩子视力出现问题才到医院进行检查时，大部分已成不可逆地变成真性近视。所以对于儿童近视，家长一定要做到尽早发现，尽早去医院进行治疗矫正，避免近视情况进一步加剧。

2.千万别给小孩戴眼镜，度数会越戴越深

研究表明：青少年近视度数发展与戴眼镜并无直接关系，而是与眼睛在发育期间眼轴增长有关。有了真性近视如果不戴眼镜会产生两种后果：一是看不清远处的目标，习惯性眯眼视物，容易造成视力疲劳；二是视物时两眼眯成一条缝，长此以往，上下眼睑压迫眼球，有可能会出现散光。视疲劳、模糊像刺激与眼肌对眼球的压迫，可能会使近视度数加深。所以戴上一副合适的眼镜，既可解决视力疲劳，又可防止度数增加过快。

3.越亮的护眼对孩子视力越好


护眼灯这一概念已经在市场普及了很多年，采用LED光源的护眼灯没有频闪，比较接近自然光的，对于视力很有益处。然而不少家长在使用过程中却有着不少误区，例如在选择台灯的亮度时，认为越亮越好。建议家长们可以选用无极调光的台灯，在使用时，如果孩子用右手写字，台灯则应该放在左边，这样不会出现局部暗区。台灯的高度应该高于孩子的头顶，这样光线的覆盖范围广，周边的明暗对比不会太强烈，孩子的使用感也会比较舒适。

4.散瞳对孩子眼睛有伤害

有家长认为散瞳验光对孩子的眼睛有伤害，还有家长认为散瞳验光后会有一段时间影响孩子视力，耽误学习，其实这些都是错误的。散瞳验光可使睫状肌调节放松，让孩子的眼睛得到充分的休息，帮助孩子排除或消除假性近视。因此，只有做了散瞳验光，眼镜度数才能配得准确。仅靠电脑验光给孩子配眼镜是不可取的，这样对孩子的眼睛有害。

5.认为配眼镜，能看清就可以了

这一条误区其实和上一条是紧密相关的，有些家长认为给孩子配了眼镜就万事大吉了，平时不注意孩子的用眼卫生，过度用眼也不干预；让孩子佩戴没有经过验光或者验光不准的眼镜；或者是一副眼镜戴很常时间都不换，这些行为都会加重孩子眼睛的负担，最终引起视力减退、视疲劳等症状。孩子近视了一定要定期进行视力检查，因为科学的医学验光配镜不仅仅是验出一个度数，而是综合考虑眼部健康、双眼平衡、眼调节功能和戴镜习惯等给出的一个合理的配镜处方和进行视功能保健的科学指导。

6.忽视户外活动的重要性

如今很多孩子整天待在室内，其实对视力很不好。在户外，自然光线的照射会使我们的身体分泌一种多巴胺，可以抑制近视的发展。此外，孩子总是待在室内，观看的东西都在眼前，会长时间处于一种“近视”的状态，对眼睛很不好

7.认为眼保健操毫无用处


做眼保健操肯定有用，它可以放松眼睛，达到缓解视疲劳的作用，但是关键在于孩子有没有按照正确的做法认真去做。许多孩子在做眼保健操的时候偷工减料，很不认真，自然也就没有什么效果。

预防近视眼的措施

四月 22, 2019

&nbsp; &nbsp; &nbsp;&nbsp;近视是眼病当中多见的一种，很多的人会遇到此病，特别是一些儿童是近视的多发人群，近视一旦出现会导致视力下降，带来了极强的伤害，所以说近视预防方面要及时，来认识一下有效预防近视的措施会有哪些吧！
近视的预防措施：
&nbsp; &nbsp; &nbsp; 许多家长为了预防和治疗孩子的近视眼想尽了办法，却仍不能阻止孩子近视眼的发生、发展。其实与其被动地保护婴幼儿视力，不如帮助孩子建立健康的生活内容和方式，从根本上解决孩子用眼过度的问题。父母可以从以下几个方面入手：

&nbsp; &nbsp; &nbsp;&nbsp;一是帮助孩子选择电视节目，减少看电视的时间。爸爸妈妈千万不要用电视做保姆。婴幼儿虽然能从电视中学到许多东西，但专家经研究发现，被动学习对孩子大脑发育益处远低于主动学习。父母应学会关掉电视，用其他方式娱乐。
&nbsp; &nbsp; &nbsp; 二是用外出活动代替电子游戏。大家都知道电子游戏损害视力，但苦于无法将孩子从游戏机旁拉开。日本的母亲每天都会安排固定的时间带孩子外出游戏，一些妈妈会结成互助小组，由一个母亲带领几个孩子玩耍，孩子不仅锻炼了身体，减少孩子玩游戏机的时间，保护了视力，也增强了社交能力。这些中国的父母应该好好学习。
&nbsp; &nbsp; &nbsp;&nbsp;三是挑选精华读物让孩子阅读。有些父母望子成龙，十分重视幼儿教育，从小就让孩子阅读许多书籍，孩子眼睛过度疲劳，视力就会下降。幼儿教育应少而精，尤其应注意每次时间不应超过15—20分钟，这样学习才符合幼儿学习心理，效果会更好。

那些你看不见的光对眼睛的伤害

四月 17, 2019

&nbsp; &nbsp; &nbsp; 走在户外被太阳“晃眼睛”，你一定会立刻戴上遮阳帽或太阳镜，以防阳光伤眼。在生活中还藏着许多“隐形光”，也在悄悄地伤害我们的眼睛，比如常见的灭蚊灯、验钞机等。
&nbsp; &nbsp; &nbsp; 眼角膜和晶体对紫外线非常敏感，呆在紫外线强的房间里超过20分钟，即使不直视也会导致角膜电光性眼炎，一般会在照射后两三个小时内发病。电光性眼炎症状虽然严重，但是角膜上皮细胞的修复能力极强，通过消炎止痛治疗，一般两三天就能痊愈，不会对日后生活产生影响。

&nbsp; &nbsp; &nbsp;&nbsp;消毒用的紫外线灯、夏天的灭蚊灯和银行常用的验钞机都是利用紫外线来工作的。紫外线虽然能杀菌、消毒、治疗皮肤病和软骨病等，但其波长较短，携带的能量较大，在没有防护措施的情况下，会对人体造成直接伤害。人的眼睛与皮肤若暴露在紫外线灯光线下3分钟以上，就有可能超过人体安全界限；如果直接照射15分钟后，就会使眼角膜损伤导致电光性眼炎，双眼突然剧烈疼痛，有异物感，畏光、流泪或眼睑痉挛等；长期照射会损害眼睛及皮肤，导致眼部创伤及皮肤严重灼伤，并可能导致皮肤癌。
&nbsp; &nbsp; &nbsp; 因此，一定要对这些悄无声息的“隐形光”提高警惕。使用灭蚊灯时，最好提前打开，有人靠近时要关掉，不能在灯前长时间逗留，尤其是小孩子，更不要开一整晚。在10多平方米的房间内，照射时间过长，一样会像烈日一样，造成皮肤晒黑、灼伤、眼睛干涩疼痛。对于经常要接触验钞机的人，验钞的时候，最好在上面遮盖一张报纸，阻挡绝大部分的紫外线。

五大不良用眼习惯你有哪些

四月 15, 2019

&nbsp; &nbsp; &nbsp; 在日常生活中，不良的用眼习惯是非常常见的，这些用眼习惯不仅会对视力造成损害，甚至还可能引发眼部感染，诱发眼部疾病，更严重的甚至会导致失明！每个人或多或少都会有这些习惯，只是通常都不会被人重视，本文为大家总结常见的不良用眼习惯，看看你占了多少？
1.用手揉眼
&nbsp; &nbsp; &nbsp; 当眼睛发痒时，很多人会习惯性地用手去揉眼睛，但其实手上会带有很多细菌，如果不洗手就去揉眼睛很容易会将细菌带进眼睛里，引发眼部感染发炎。
2.长时间面对电子产品
&nbsp; &nbsp; &nbsp;&nbsp;长时间面对电脑、手机这些电子产品很容易引发视疲劳，如果视疲劳得不到有效缓解，时间长了就会形成近视或引起近视加深。无论是青少年还是成年人，连续使用电子产品1小时后要休息5~10分钟，减少眼部疲劳。
3.关灯看电视、玩手机
&nbsp; &nbsp; &nbsp;许多年轻人都有关灯玩手机的习惯，关灯看电视、玩电脑手机都会加剧眼睛的负担，久而久之很容易出现干眼症、慢性结膜炎等疾病，还容易增加青光眼的发病风险。因此，如果夜间需要使用电子产品时，一定要开照明灯，对眼睛起到保护作用。
4.戴隐形眼镜睡觉
&nbsp; &nbsp; &nbsp;&nbsp;隐形眼镜透氧性低，长时间配戴很容易引起角膜缺氧、角膜水肿甚至感染的后果。普通隐形眼镜每天的配戴时间要控制在8小时内，最长不要超长12小时。无论是白天睡觉还是晚上睡觉都应该摘下隐形眼镜。
5.过度依赖眼药水
&nbsp; &nbsp; &nbsp; 当出现眼红、眼痒、眼干、眼涩的情况时，很多人习惯使用眼药水来缓解。但目前药房出售的大部分眼药水都含有防腐剂，防腐剂会对泪液的成分有影响，还会改变眼球表面的正常环境，继而引起角膜上皮缺损、脱落、糜烂的情况，严重者甚至会发生角膜溃疡、角膜穿孔。
