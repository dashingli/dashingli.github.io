假日眼睛保健小锦囊

五月 18, 2022

&nbsp; &nbsp; &nbsp; 眼睛保健小锦囊，学校放假了，广大青少年都沉浸在假日的欢乐中。然而，爱大爱眼科专家提醒大家，玩乐的时候不要忘了眼睛保健。每个人都希望拥有一双健康美丽的大眼睛，那么，除了日常护理，我们还应该怎样保护眼睛呢？下面爱大爱眼科专家为广大青少年提供了几点意见。

1、食物搭配
&nbsp; &nbsp; &nbsp;&nbsp;要多喝水，保持皮肤的水份充足，小细纹、鱼尾纹、眼袋、黑眼圈、眼睛浮肿是眼部肌肤最容易出现的问题，这些问题的产生有时是因缺水，或循环不良造成水分囤积；经常食用一些胶质性物质，如猪蹄、鸡爪等，以保持皮肤的弹性。
2、生活工作习惯
&nbsp; &nbsp; &nbsp; 常在计算机前打字、戴隐形眼镜增加了眼部疲劳，使眼部老化。这种情况除了靠保养品改善外，生活习惯的配合也是十分重要的。
3、克服不良习惯
&nbsp; &nbsp; &nbsp;&nbsp;很多人属于先天愁眉苦脸型，整天的皱眉，眯眼，使眼部周围皮肤紧张，从来变得松弛，另外过多的面部表情也会使面部包括眼睛周围皮肤松弛老化；长期熬夜，无规律的睡眠，自然也是一个主要的敌人。
4、生物保养法
&nbsp; &nbsp; &nbsp; 用胡萝卜汁加一些橄榄油涂敷眼周和眼角皱纹处，或于睡前在上述部位敷以维生素E油剂，以增强皮肤的抗衰力，减少或减轻皱纹的形成与加深。
&nbsp; &nbsp; &nbsp; 眼睛是心灵的窗户，是我们身体的重要组成部分，因此我们在任何时候都不能忽视眼睛保健。如果眼睛出现不适的症状，应该及时到专业的眼睛保健机构进行检查，及时做眼睛保健训练，让眼睛时刻保持最佳的状态！

如何拥有一双漂亮的眼睛

五月 18, 2022

&nbsp; &nbsp; &nbsp;&nbsp;爱大爱表示，长期使用电脑的人普遍患有干眼症，即容易眼干、眼红和疲倦。这是因为电脑荧光屏由小荧光点组成，眼睛必须不断地调整焦距，以保证视物清晰，时间过长，眼肌会过于疲劳。眼睛出现酸痛、发胀、干涩、视力模糊等问题，很容易导致头痛、注意力不集中、情绪烦燥、反应迟钝、疲乏无力和失眠、恶心、欲吐等一系列疲劳症状。而全身不适等症状，多在年老体弱者和亚健康人群中出现。
&nbsp; &nbsp; &nbsp; 同时，眼睛长时间聚焦时，肌肉的紧张度会降低，影响大脑皮层调节肌肉运动的区域，从而使神经的兴奋性下降，使其对颈部、肩部肌肉运动的控制失调，引起肌肉张力增高。当这种紧张长期不能缓解时，相应部位就会出现慢性疼痛。
&nbsp; &nbsp; &nbsp;&nbsp;拥有一双健康、美丽的眼睛，对一个人来说是多么地重要。但是白内障、沙眼、青光眼、夜盲症、视网膜病变等各种眼病正在威胁、困扰着每一个人。提高爱眼、护眼意识，预防眼部疾病的发生已经迫在眉睫。导致眼病的因素多种多样，其中营养摄入不均衡，使得视神经、眼底得不到足够的养分是引发眼病的一个重要原因。
&nbsp; &nbsp; &nbsp; 蓝莓果实中除了常规的糖、酸和VC外，富含VE、VA等多种维生素、熊果苷、蛋白质、花青苷、食用纤维以及丰富的矿物质元素。除了丰富的营养元素外，爱大爱眼科专家家还发现，蓝莓中富含的花青素，可直接加速视网膜、视紫质的合成与再生，让眼睛绽放活力光彩；蓝莓能帮助营养预防近视、干眼症等各类眼部问题。

&nbsp; &nbsp; &nbsp; 另外，鳕鱼肝油中富含的天然维生素A，对眼睛视网膜感光功能等也起着一定的作用。所以对于学生族、长期使用电脑或长期佩戴隐形眼镜的人群来说，若是同时服用些蓝莓和鳕鱼肝油，双管齐下保护眼睛，对解决各种眼部干燥、干涩、干眼症、视力下降等眼部问题来说都是较为不错的选择。

远视度数会降低吗

五月 18, 2022

&nbsp; &nbsp; &nbsp; 由于青少年的眼球仍在继续发育，远视度数会逐渐下降，所以对于远视的儿童应该定期复查，必要时更换镜片，由于复查时常需用阿托品行睫状肌麻痹，其副作用是瞳孔散大导致视物模糊，影响学习，故最好在寒暑假时各复查一次。

&nbsp; &nbsp; &nbsp; 人们常有一个错误的看法，认为远视患者的眼睛比正常人看得远，或者说远视患者看近不清楚，看远清楚。实际上，远视眼的定义是：处于休息状态的眼睛，平行光线在视网膜后面形成焦点，这种屈光状态称为远视眼。这种眼的光学焦点在视网膜后，因而在视网膜上形成的物像是模糊不清的，患者的视力受到影响。
&nbsp; &nbsp; &nbsp; 如果远视的度数比较小，患者能够通过自身的调节加以克服，使视网膜上的物像变清晰；而中度或重度远视，患者很难通过调节克服屈光能力的不足，故远视眼经常处于调节紧张状态，容易疲劳。中度以上或重度远视，即使通过自身的调节功能，也不能保持视网膜上的物像总处于清晰状态。
&nbsp; &nbsp; &nbsp; 在日常生活中，患者不能看到清晰的物像，视网膜上没有清晰物像的刺激，视觉系统不能得到正常的发育，便形成弱视。一般情况下，这些患者的远视都是与生俱来的。他们在胚胎期，眼球的形状没有发育到正常的水平，其前后径比同龄的正常孩子小-些，屈光能力不足。这种弱视称为屈光不正性弱视，他们两只眼屈光不正的度数相同或基本相同，两只眼的视力基本相等。

劳逸结合预防青少年近视

五月 18, 2022

&nbsp; &nbsp; &nbsp; 青少年近视的人数越来越多，预防近视成为当下一个热门话题。爱大爱眼科专家表示，眼球在调节静止的状态下，来自5米以外的平等光线经过眼的屈光后，焦点落在视网膜前，不能准确地在视网膜上形成清晰的像，称为轴性近视。劳逸结合可以缓解眼部疲劳，是预防近视做好的方法。

小编为我们做详细的介绍。
&nbsp; &nbsp; &nbsp; 1.避免长时间连续操作电脑和看书学习，千万注意不要等到感觉眼睛疲劳时才休息，而应该是每隔一小时左右休息10—15分钟，休息时或是看看窗外的绿树或远景，或是做做眼保健操，使眼睛充分放松。
&nbsp; &nbsp; &nbsp; 2.操作电脑和看书学习时要保持良好的工作姿势。不要躺着看书或者在光线差的地方看书以及在移动的物体上看书。看书的灯光以用日光灯为宜。
&nbsp; &nbsp; &nbsp; 3.如果出现眼睛干涩、发红，有灼热感或异物感，眼皮沉重，看东西模糊，甚至出现眼球胀痛或头痛，则要立即停止操作电脑和看书学习，休息一段时间。
&nbsp; &nbsp; &nbsp; 4.长时间操作电脑和看书学习出现视觉疲劳症状时，可以闭上双眼按摩眼眶周围的几个穴位.例如：揉睛明穴，揉四白穴，揉太阳穴。还可以用湿热毛巾热敷双眼，每次15分钟。如果通过上述处理，仍无明显好转，那就需要到医院看眼科医生了.
&nbsp; &nbsp; &nbsp; 5.眼局部进行热敷、药熏、理疗、针灸等。

疫情期间，孩子上网课如何保护双眼

五月 18, 2022

&nbsp; &nbsp; &nbsp; 疫情期间，许多学校老师们纷纷选择了网上授课，这对于孩子们来说无疑是防范病毒的最佳授课方式，但电脑的有害蓝光及屏幕的频闪容易对眼睛造成伤害，长时间使用电脑，眼睛会感到酸痛、干涩，甚至会影响孩子眼视力。那么该如何护眼？
首先我们先来了解一下护眼有哪些误区
护眼一定要吃维生素A？
&nbsp; &nbsp; &nbsp; 如果缺乏维生素A，容易出现干眼病、夜盲症等症状。如果你不缺维A，吃多了反而对身体有影响：维生素A是脂溶性的，摄入过量时不易排出，可能导致骨骼生长异常，对孕妇的影响更大。
看绿色就能护眼？
&nbsp; &nbsp; &nbsp;&nbsp;大家认为绿色可以护眼，无非是因为眺望窗外的绿色植物时会觉得眼睛很放松。但其实，让眼部放松的并不是窗外的“绿色”，而是“窗外的绿色”。远眺才能让眼睛更健康。
开了护眼模式放肆用？
&nbsp; &nbsp; &nbsp; 手机的护眼模式就是让屏幕变黄变暗，看上去变得不那么刺眼。但是蓝光对于眼睛的伤害目前并没有定论，所以护眼模式未必可以护眼。想要保护视力，关键还是在于用眼时间。
那么对于孩子们，离不开电子产品怎么护眼
黑夜开灯
&nbsp; &nbsp; &nbsp;&nbsp;关灯后，人眼的瞳孔会在昏暗中散开，注视手机屏幕的光亮时会受到更多刺激。所以，看手机、平板、电脑时，最好打开灯，改善用眼环境。
多眨眼
&nbsp; &nbsp; &nbsp; 人眼需要通过眨眼来把泪液均匀地涂抹到角膜上，长时间不眨眼，角膜表面泪液蒸发，容易引起眼睛干涩。因此，看看电子屏幕就眨眼休息一下。
护眼操不能忘
&nbsp; &nbsp; &nbsp;&nbsp;护眼操是帮助我们缓解视疲劳的，看累了电子产品后，一定要把手洗干净，做做护眼操。

1、用眼环境要湿润
家中有暖气空调的时候，室内环境会比较干燥，室内适当使用加湿器，提高房间里的空气湿度。
2、注意环境卫生
注意环境清洁，空气中的细菌、螨虫、灰尘都增加了过敏性结膜炎发生的可能性，定期更换床上用品及清洗空调
1、长时间使用电子产品，得温补。加上一些人习惯性熬夜，可以试试温补护目——泡上一壶温补茶。这对居家上班族来说也是最好的饮品。
2、使用电脑的时候准备一些护眼贴，一旦眼睛酸涩就必须开启护眼模式，不能长时间盯着屏幕。
3、长时间盯着电脑的人为了不影响视力下降，可以多吃维生素A以及维生素B对预防视力减弱有一定的效果。因此，多吃富含维生素A的食物以及多吃富含维生素B的食物。
三种护眼模式提供选择
1、分段观看电子产品

2、常敷眼睛
一旦我们感觉到眼睛有酸涩或其他异常时，停下手中工作，用毛巾或其他护眼产品敷一下眼睛，让眼睛得到充分休息。
3、多喝水喝茶
多喝水能使身体保持水分平衡，眼睛不至于干涩严重。可以适当喝一些明目茶，对眼睛也是有益的。
