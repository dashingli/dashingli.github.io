爱大爱老花小课堂：四十而立小心“老花”来袭

五月 18, 2022

&nbsp; &nbsp; &nbsp; 老视(俗称老花眼)不是病态，也不属于屈光不正，它是一种生理现象，是人生的必经阶段。爱大爱眼科专家提醒40岁以后如果看东西出现上述症状，就要警惕是不是老花眼了，强撑着不去配老花镜，反而会产生头晕眼胀等症加重。
&nbsp; &nbsp; &nbsp; 老花眼，顾名思义，是随着变老而出现的眼部症状。无论男人女人，从40—45岁开始出现。如果按照平常的用眼习惯，需要把书报拿远一点才能看清；喜欢光线强一点来阅读；看书读报以及用电脑时间稍微长一点，出现串行或看字重影，甚至出现眼睛酸胀，鼻根眼眶疼痛，那你很可能就花眼了。并且，随着年龄越大，这种感觉会越来越明显。

&nbsp; &nbsp; &nbsp;&nbsp;对付眼花最好的“武器”就是老花镜，一旦确诊为老花眼，很多人在配镜子时并不像年轻人要求那么严格，往往是在街上随便买一副老花镜就戴着了。其实，大街上或地摊上的老花镜往往双眼镜片度数相同，瞳距固定，但其实每个人都会存在近视、远视或散光等屈光不正的情况，并且每个人的瞳距不同，如果随便买副眼镜，往往是不合适的。
&nbsp; &nbsp; &nbsp; 老花镜就像我们的衣服一样需要量体裁衣。不经过医学验光而直接购买的老花镜可能一时半会可以看清，但是时间久了就会出现视力疲劳、视物模糊等症状，甚至加重老化症状。所以购买老花镜，一定要进行科学的验光。当然最好先到医院做个眼睛的全面检查，在排除了影响眼部健康的其他眼病之后配镜最好。
&nbsp; &nbsp; &nbsp; 很多人老花镜一戴几年都不换，能凑合看清楚就行了，觉得没有那么讲究。其实，老年人戴上老花镜后，至少应该每隔5年去眼科验一次光。这是因为老花眼是人眼生理调节机能的减退，随着年龄增长，老花的度数会逐渐加深。专家表示：“如果45岁时度数是150度，那么50岁度数会增加为200度，到了55岁则会变成250度。如果一直戴一个度数的老花镜，几年都不更换，不仅会造成视物不清，还会加速眼睛的老花症状。”
&nbsp; &nbsp; &nbsp;&nbsp;如果一副旧的老花镜在佩戴了几年之后，发现看书、报纸或者电脑用平常的惯用距离看不清了，离远一点才能看清，就应该尽快更换一副新的老花镜了。

&nbsp; &nbsp; &nbsp;&nbsp;同时，爱大爱将联动全国各地的代理商为全国100多家敬老院中有老花镜需求的贫困老人捐赠爱大爱老花镜，为全国各地敬老院的老人们护眼送爱心，将清晰的视界送给他们！

近视常见的分类有哪些

五月 18, 2022

&nbsp; &nbsp; &nbsp; 近视眼是指眼在不使用调节时，平行光线通过眼的屈光系统屈折后，焦点落在视网膜之前的一种屈光状态。所以近视眼不能看清远方的目标。若将目标逐渐向眼移近、发出的光线对眼呈一定程度的散开，形成焦点就向后移，当目标物移近至眼前的某一点。此点离眼的位置愈近，近视眼的程度愈深。

近视常见的分类方法有三类，分别依据近视度数，屈光成分和病程进展近视分类！
≤300度（≤﹣3.00D）。
300度600度（﹣3.00D﹣6.00D）。
＞600度（＞﹣6.00D）。
最常见，眼球前后径过长（即眼轴长度超出正常范围），而屈光力（即角膜和晶状体等眼其他屈光成分的屈光性能）基本在正常范围。
主要由于角膜或晶状体曲率过大，或各屈光成分之间组合异常，导致屈光力超出正常范围，而眼轴长度基本在正常范围。
近视度数一般在﹣6.00D以内，大部分患者的眼底无病理变化，进展缓慢，用适当的镜片即可将视力矫正至正常，其他视功能指标多属正常。
一般近视都市较高，且伴有不同程度的眼底改变。患者除了远视力差之外，常伴有夜间视力差，飞蚊症，漂浮物，闪光感等，发生视网膜脱离，撕裂，裂孔，黄斑出血，新生血管和开角型青光眼的危险性要大得多。
&nbsp; &nbsp; &nbsp; 以上就是近视眼的几种分类，大家可以根据自己的视力情况加以区分，有利于近视眼的养护以及矫正，爱大爱专家提醒您，近视一定要注意，须及时验光配镜，到专业可靠的视力矫正机构或者医院进行视力养护、矫正。

高度近视需要注意哪些问题？

五月 18, 2022

表示，近视加深的最主要恶果是——高度近视。
按近视程度分，近视度数大于600度（儿童>400度）的屈光不正即称为“高度近视”。

高度近视需要注意哪些问题？
&nbsp; &nbsp; &nbsp; 1、高度近视需要配戴眼镜，如果不戴眼镜，近视的度数会越来越深。眼睛如果是真性近视，那是不可逆转的；还有一方面，近视患者不佩戴眼镜还会导致眼部增加或加深皱纹，因为近视患者看不清东西就会不自觉地眯着眼睛观察事物，以至于眼睛部位的面部肌肉长期处于紧皱的状态，皱纹也就随之而来了。
&nbsp; &nbsp; &nbsp;&nbsp;2、高度近视者别做剧烈运动，当高度近视患者进行足球、篮球等对抗性比赛，或者参加短跑等过于剧烈的运动时，外力的突然冲击，可通过玻璃体的传导，促使视网膜破裂，造成视网膜脱离。
&nbsp; &nbsp; &nbsp; 3、高度近视患者需注意用眼卫生，因为眼睛里面的细菌会导致眼疾的发生。因此，应该保证眼睛的卫生，不要用手揉眼睛。
&nbsp; &nbsp; &nbsp;&nbsp;4、有些人认为近视无所谓，高度近视也没什么。殊不知，高度近视患者不仅有近视问题，而且往住伴随着多种并发症；而度数越高，诱发并发症的可能性就越大。
高度近视患者需防范三大并发症
&nbsp; &nbsp; &nbsp; 1、飞蚊症：随着眼球的转动，常在眼前出现黑影飘动，就像蚊蝇在飞舞，这是玻璃体正常的胶体状结构发生液化的缘故。
&nbsp; &nbsp; &nbsp; 2、黄斑出血：由于眼球变大造成对眼球壁的不断牵拉，引起眼底视网膜和脉络组织的正常关系出现代谢失调，特别是黄斑区的小血管因牵拉等因素而出现破裂时，就会造成黄斑出血，使中心视力受到很大损害。此时病人自觉眼前有一块固定的黑影挡住视线。尽管出血经治疗可以吸收，但总会或多或少留下斑痕，影响视力。还有一些患者黄斑出现异常新生血管，反复出血，对视力损害更严重。
&nbsp; &nbsp; &nbsp;&nbsp;3、视网膜脱离：因眼轴变长，眼球后半部变薄，视网膜、脉络膜萎缩、变性，出现裂孔，导致出血和范围不等的视网膜脱离，从而丧失视力。同时黄斑区变性也是高度近视眼的最大危险，表现为视物变形，视力下降，最终导致失明。
&nbsp; &nbsp; &nbsp; 最后需要提醒的是，近视人群，特别是近视青少年养成良好的用眼习惯，多点户外运动，高度近视患者每年定期做一次专业的眼底视网膜检查，以便及时发现视网膜隐患并加以治疗，从而免除后顾之忧。

小心“青光眼”

五月 18, 2022

据权威研究证实，全球900万人因青光眼致盲，而随着老龄化进程加剧，这个数字还在不断上升。

青光眼致盲无法复明
“青光眼是全世界第二大致盲眼病，也是排名首位的不可逆性致盲眼病。”据介绍，我国流行病学研究资料显示，40岁以上人群青光眼患病率为2.6%，致盲率约30%。
&nbsp; &nbsp; &nbsp; 由于不管如何治疗，青光眼致盲都不能复明，因此其对人类健康和社会发展的影响，较世界第一的致盲眼病白内障更为严重。预计到2020年，我国将有2100万例青光眼患者，可能产生近630万名盲人及超过1000万名视觉残障人士。
定期检查可预防青光眼
&nbsp; &nbsp; &nbsp; 医生指出，虽然青光眼致盲后不能复明，但如果早期发现、早期干预，患者可以在有生之年保持一定的视觉功能，甚至能够维持正常生活和工作。
同时，糖尿病、高度近视、有家族史者，都是青光眼的高危人群，更应提高警惕，定期检查。
青光眼的几种类型
&nbsp; &nbsp; &nbsp; 有的表现为急性发作，眼球内压力急剧升高，一般超过50mmHg，患者就会感觉到头痛、眼痛，此类患者如果没能迅速把眼压控制到正常水平，在短时间内就可致盲；
发生在婴幼儿的青光眼主要表现为眼球大于正常孩子，且怕光、流泪，这些患儿在确诊后需要尽快手术，才有可能保留一定的视觉功能。

如何防止蓝光对眼睛造成伤害

五月 18, 2022

&nbsp; &nbsp; &nbsp; “儿童近视”一直是父母很担心的一个问题。现在大家的日常生活，都离不开手机、电脑、iPad，长时间使用电子产品容易造成眼睛疲劳、视力下降。在电子产品中，潜伏这一种“隐形杀手”——高能短波蓝光。可怕的是，许多人并不了解“蓝光”。

爱大爱将和大家聊一聊“蓝光伤害”的问题。
&nbsp; &nbsp; &nbsp; 自然界的可见光一共有7种颜色，七彩世界这个词就由此而来，更学术地说，蓝光指HEV光线（High-energyvisiblelight），即高强度、高频（波长400-500nm）的蓝紫色可见光线，属于短波蓝光，该波长内的蓝光我们称之为有害蓝光。
&nbsp; &nbsp; &nbsp; 所有的光源，要想成为通用照明光源，都需要过人眼这一关。人眼最习惯的光照是太阳光，在比较其他照明光源时通常都以太阳光为比较对象，这也是测量光源显色指数的原理。
&nbsp; &nbsp; &nbsp; 从光谱图上可以看出不同时段太阳光在可见范围内基本是黑体辐射的连续谱，和此光谱越接近，人眼对颜色的感觉就越自然。
&nbsp; &nbsp; &nbsp; 作为自然光的三基色之一，蓝光是波长最短、频率最高的高能量光线，因而被广泛应用于LED技术中，来提升电视、手机等电子产品的品质。
&nbsp; &nbsp; &nbsp; LED有几种类型，目前常见的很多LED本身都是发出蓝光，然后通过荧光剂发出黄光，混合成白色光源，基于LED技术的数码产品显示屏会发出高能蓝光。
&nbsp; &nbsp; &nbsp;LED虽然具有节能省电的优点，但其产生的高能蓝光可以穿透角膜和晶状体，直达眼底黄斑区，对眼球末端的黄斑区造成伤害。在这样的光照下，视网膜色素上皮毒性大大增加。
&nbsp; &nbsp; &nbsp;&nbsp;高能蓝光的损害作用是一个连锁反应，先引起光敏感细胞死亡，再引起视网膜黄斑变性，继而导致视力逐渐下降甚至完全丧失，而这种伤害往往不可逆转，马德里康普顿斯大学2013年就研究则表明，LED光源照射会对视网膜造成永久性伤害。
&nbsp; &nbsp; &nbsp; 科学家发现，蓝光对眼睛的危害程度因年龄而异，6-19岁为80%90%，20-39岁为60%70%，40-60岁约50%。
&nbsp; &nbsp; &nbsp; 而高能蓝光最有可能造成的危害主要体现在：加重视网膜黄斑部疾病、导致白内障术后的眼底损伤、引发视觉模糊、导致视觉疲劳、引起CVS视疲劳综合征、引发眩光；除此之外，蓝光还会抑制褪黑素的分泌、打扰睡眠，提高自身重大疾病的发生率。
&nbsp; &nbsp; &nbsp;&nbsp;由于儿童及青少年正处于人类视觉系统发育的敏感期，相较成人更容易受到高能蓝光的伤害。
&nbsp; &nbsp; &nbsp; 家长需要严格把控孩子使用电子产品的时间，同时应该借助于防蓝光产品对其进行防护。0-2岁的儿童不宜使用电子数码产品，2-6岁以内的幼儿每次使用时间不应超过20分钟，一天累计以不超过1小时为宜。
&nbsp; &nbsp; &nbsp;但这也带来了另外一个问题，现在21世纪信息时代，孩子和青少年都无法避免接触屏幕来获取信息，如果无法从产品端解决蓝光危害的问题，“近视”将成为这一代孩子所付出的“时代价值”。
&nbsp; &nbsp; &nbsp; 目前市场上也出现号称可以保护视网膜的手机贴膜，防蓝光软件等，宣称可使眼睛轻松，看东西清楚，还“可过滤掉手机屏幕发出的散色短光波的干扰，使视网膜黄斑区的叶黄素免受蓝光的损害，因而有保护视网膜的作用”。请大家勿要盲目相信，这些大部分都是营销噱头。

&nbsp; &nbsp; &nbsp; 核桃仁(微炒去皮)300克，红枣(去核)250克，枸杞子150克，与鲜猪肝200克同切碎，放瓷盆中加少许水，隔水炖半小时后备用。每日取2～3汤匙，打入两个鸡蛋，加糖适量蒸为羹。有益肾补肝、养血明目的作用，可改善近视、视力减退及头昏健忘、腰膝酸软等症状。
&nbsp; &nbsp; &nbsp; 花生米100克，南瓜子50克，红枣肉60克，黄豆粉30克，粳米粉250克，与枣肉共捣为泥，再调入些面粉，加适量油与水，调匀做糕，蒸熟，一日吃完。可补脾益气、养血明目，改善近视、视物模糊，心悸气短、体虚便秘等症状。
&nbsp; &nbsp; &nbsp; 赤豆、扁豆、花生仁、薏苡仁、核桃肉、龙眼、莲子、红枣各30克，粳米500克，加水煮粥，拌糖温食。可健脾补气、益气明目。宜于近视、不耐久视、寐差纳少、消化不良者食用。
&nbsp; &nbsp; &nbsp; 人参10克，远志30克，共杵为末，每包8克，每次1包，沸水冲泡代茶饮，连服7～10天。可益气养心、益智明目。
枸杞肉丝
&nbsp; &nbsp; &nbsp; 枸杞100克，猪瘦肉300克，青笋(或玉兰片)10克，猪油100克，各种佐料适量。将猪瘦肉洗净，切成6厘米左右的细丝，青笋也同样制作，枸杞子洗净。待油七成热时，下入肉丝、笋丝煸炒，加入料酒、酱油、食盐、味精，放入枸杞，翻炒几下，淋人麻油即可。枸杞可滋补肝肾、润肺明目，猪肉富含蛋白质，两者共食，可使气血旺盛，营养眼内各组织。

食物与眼睛的相关性

五月 18, 2022

小编就给大家介绍都有哪些对眼睛好的食物。

1、胡萝卜
&nbsp; &nbsp; &nbsp;&nbsp;维生素A啊是保持眼睛健康不可或缺的微量元素。维生素A包含有维持眼睛健康的最重要的物质。那就是抗氧化剂β-胡萝卜素。胡萝卜就是摄入维生素A最好的选择了，是最好的对眼睛好的食物之一了。当然除了胡萝卜之外，卷心菜和生菜也是维生素A含量很高的食物。身体如果缺少了维生素A，人就可能会得夜盲症。因为维生素A与感光有直接的关系，所以维生素A是和视力直接相关的营养素。B-胡萝卜素也能够在体内转换成维生素A。
2、菠菜
&nbsp; &nbsp; &nbsp;&nbsp;菠菜可以预防一些眼类疾病。科学家们在菠菜中发现的一种叫做黄体素的类胡萝卜素可以预防白内障和黄斑变性。
3、水果
&nbsp; &nbsp; &nbsp; 水果富含各种维生素和营养物质，可以说不仅仅是对眼睛好的食物，更是能治百病的食物。像猕猴桃，橙子和枸杞子等这类就是富含维生素C的水果了。维生素C是可以消除能对眼睛造成伤害的自由基的。当然如芒果等黄色水果也富含维生素A也对眼睛好。
3、豆制品
&nbsp; &nbsp; &nbsp;&nbsp;豆制品中呢含有人体必需的对眼睛有益的脂肪酸，植物性雌激素，维生素E和天然抗发炎剂。豆奶，黄豆，大豆乳酪这类豆制品都是有益眼睛健康的。橄榄油和谷类食物等富含维生素E的食物同样有益于我们的眼睛。
4、大蒜
&nbsp; &nbsp; &nbsp;大蒜呢是用来提高视力的极好的食物。大蒜中的硫是能稳固晶状体的，并且让它变得更有韧性。此外富含硫的洋葱也对视力很有好处。
5、鸡蛋
&nbsp; &nbsp; &nbsp;&nbsp;如果有朋友要是个喜欢吃鸡蛋的人，那么就得恭喜你了。因为啊鸡蛋对视力非常有益。鸡蛋中呢含有半胱氨酸，硫，卵磷脂，氨基酸和黄体素，这些物质都是可以防止生成白内障的。
6、鲑鱼和沙丁鱼
&nbsp; &nbsp; &nbsp;&nbsp;它们富含的ω-3脂肪酸对视力是非常有好处的。ω-3脂肪酸的主要来源呢就是鱼类，而鲑鱼和沙丁鱼就富含DHA这种ω-3脂肪酸。
7、西兰花
&nbsp; &nbsp; &nbsp;&nbsp;西兰花呢不仅可以提高视力还可以预防白内障。西兰花中含有黄体素和玉米黄质(类胡萝卜素属中的一种植物营养素抗氧化剂)。这种物质对晶状体是非常有好处的。这些类胡萝卜素是可以使眼细胞免受自由基对其造成的压力的。
8、黑巧克力
&nbsp; &nbsp; &nbsp;&nbsp;中医认为黑色的食物入肾可促进补肾的作用，同时肾为先天之根本，可以对各个器官起到一个调节的作用，常吃黑色的食物具有改善眼部疲劳的作用。而黑巧克力中的黄酮类化合物就可以很好地保护眼部血管，从而可以强化眼角膜和晶状体。需要强调的一点是只有纯黑巧克力才能达到这种预期的效果哦。
9、肉类
&nbsp; &nbsp; &nbsp;&nbsp;肉类中以牛肉，羊肉，鸭肉，鸡肉，鱼肉动物内脏等都具有补肾益精血的作用。尤其是羊肉和牛肉还具有平肝明目的强大功效。同时肉类食物中又富含丰富的蛋白质，而蛋白质本身又是又具有修补眼部所需要的营养素的作用。

为何有人近视手术后视力回退，近视手术有没有“保质期”

五月 18, 2022

&nbsp; &nbsp; &nbsp; 近视激光手术诞生于二十世纪末，自1985年应用于临床治疗近视以来，为使人类拥有清晰完美的视觉不断革新进步，现今已经进入微小切口无创时代，视力恢复迅速，视觉质量优秀，其中以2011年起开展的“SMILE”全飞秒激光手术为代表，其表现更是卓越。

&nbsp; &nbsp; &nbsp; 然而，很多人仍然会有疑惑，临床工作中经常遇到这样的问题——“听说整容手术有保质期，你们这个激光手术能保几年啊？”“过了20年会不会又要带戴眼镜啦？”“我朋友做了手术5年后视力就不好了，为什么啊？”……那么，近视激光手术究竟有“保质期”吗？让我们先从原理上来认识一下这种手术。
&nbsp; &nbsp; &nbsp; 无论何种激光近视矫正手术，其原理都是通过精准切削角膜中央部光学区，重塑角膜表面曲率，使角膜从陡变平，从而改变全眼屈光度。在保证安全的前提下，远期矫正效果如何取决于几个关键因素：近视度数、角膜厚度、角膜曲率；近视度数越高，切削的角膜厚度越多、角膜曲率改变越多。
&nbsp; &nbsp; &nbsp; 由于角膜组织的生物力学特性和创伤修复反应，激光术后短时间（6个月）内会发生一定程度的反应，但可以用药物加以控制，一般不会对视力造成影响。而长期的效果一则与手术前的近视度数、角膜厚度、角膜曲率密切相关，这是在术前通过严格的检查评估后可以被预测的；二则与近视是否再次新增有关，这与用眼习惯和近视遗传因素密切相关。下面我们再通过举例分析来深度认识一下。
&nbsp; &nbsp; &nbsp; 一位25岁300度的稳定近视患者，角膜厚度520微米，角膜曲率43D。这位的角膜厚度和曲率都是正常人群的平均值，整体条件非常适合近视手术，可以说几乎没有反弹回退的风险；如果术后注意用眼习惯，保护双眼，那将长时间拥有清晰视觉。
&nbsp; &nbsp; &nbsp; 另一位25岁800度的近视患者，近年来每年还有极少量近视增加。角膜厚度550微米，角膜曲率44D。虽然度数高达800度，但角膜厚度足够，角膜曲率适宜，结合患者的病史，可以预测患者术后还有可能出现新增近视，有经验的医生会额外再附加矫正50～75度，术后注意不能长时间用眼，养成良好习惯，那么，也可以长时间拥有清晰视觉。
&nbsp; &nbsp; &nbsp;还有一位18岁600度近视患者，近年来每年增加50度，母亲有高度近视，因参军需求想做手术矫正。角膜厚度540微米，角膜曲率40D。虽然近视度数中等，角膜厚度足够，但角膜曲率较平坦，术后角膜曲率将变得更平坦，因角膜的生物力学特性，预测会有一定程度的回弹，约在100度左右，医生手术设计会适当附加矫正度数。然而，他的年龄才18岁，近视还不稳定，因此，术后新增近视机率高，一定要注意用眼，但从长期来看，还是极有可能出现低度数的近视。这些情况，术前医生都会预测到并告知其短期长期的效果。他在手术后顺利通过体检，实现了当军人的梦想，但长远的效果则要靠他的自我维护。
&nbsp; &nbsp; &nbsp; 当然，现今我们还有更多的近视矫正手术可以选择，对于一些术前检查后预测激光手术矫正效果不佳，术后远期视力易回退的患者，医生还会推荐其他手术方式，比如ICL晶体植入手术，但一定需要更加严格详细的评估，选择可靠的医院和可信的医生。

&nbsp; &nbsp; &nbsp; 看到这里，大家都应该明白近视激光手术的“保质期”了吧？如果一定要问医生，我们的回答是希望“保质期”是“永久”的。但这个“永久”需要医生和患者的共同努力。根据国家教育部、卫计委最新调查表明：我国小学生近视率超过25%，初中生近视率达到70%，高中生近视率达到85%，居世界第一位。长期从事过度的近距离阅读工作，眼睛负担过重是最主要的原因。近视激光术后也一样，实现“永久保质期”更需要爱眼护眼。

为什么眼干症这么难缠

五月 18, 2022

由于泪液缺乏或蒸发过强等，干眼往往导致眼表损害，其主要症状是各种眼部不适，影响包括阅读、看电视、电脑相关的工作和驾驶汽车等行为，严重干扰病人的生活质量，但是不会导致盲等不良后果。

&nbsp; &nbsp; &nbsp; 根据干眼的发作持续时间可以分为过性干眼和慢性干眼，前者表现为短暂的眼部症状和体征后很快恢复正常，而后者的眼部不适症状和体征一直持续存在，是临床最常见的干眼现象。
干眼症的治疗
&nbsp; &nbsp; &nbsp; 据文献报道干眼的发病率为5%～35%。干眼是一种眼表的慢性病理状态，一旦确诊为干眼后，首先需要了解患者的生活工作环境和方式，如角膜接触镜配戴情况、激素替代治疗史、抗组织胺药物使用、眼部手术特别是角膜屈光手术史、吸烟和饮酒情况、睡眠呼吸暂停、计算机相关工作和空调环境长期工作等均可能加重干眼症状。
次，根据中华医学会眼科学分会角膜病学组“干眼临床诊疗规范(讨论稿)”意见，干眼可以分为水液缺乏型(aqueousteardeficiency，ATD)、蒸发过强型、黏蛋白缺乏型、泪液动力学异常型和混合型干眼等五种类型，其中混合型干眼是临床上的主要类型，临床眼科医生的主要任务应该是确定患者干眼的类型，选择合适的治疗方法进行个性化治疗。
&nbsp; &nbsp; &nbsp; 另外，很多患者在治疗后，应该根据患者对人工泪液等治疗手段的反应不同，调整或增加治疗方法，以期提高患者的生活质量，减少眼表损害。
干眼治疗目的是减轻患者的症状，包括对生活工作环境的调整，如增加空气湿度、减少气流影响、适当休息避免过度用眼，还包括停止使用可能导致干眼的口服药等。
&nbsp; &nbsp; &nbsp; 如果这些措施无效，就需要采用润滑眼球的人工泪液、抗炎药物、口服基础脂肪酸、泪点栓塞和血清等手段，甚至更严重患者给予手术治疗。
因此，含有脂质的人工泪液就特定用于睑板腺功能障碍泪液蒸发过强型干眼，提高泪膜中脂质层的稳定性，而不含防腐剂的人工泪液对于需要长期多次(＞4次)使用人工泪液的患者更适用，主要是避免了防腐剂如苯扎氯铵等对眼表上皮细胞的毒性作用。
&nbsp; &nbsp; &nbsp;&nbsp;如果能够确定患者因为蒸发过强泪液表现为高渗状态，给予低渗性人工泪液就可以减低泪膜的渗透压，降低眼表炎症。
&nbsp; &nbsp; &nbsp; 人工泪液的选择应遵循原则是干眼症状越重、人工泪液的黏性越大、使用频率越高。当然，一般患者白天更倾向于用黏性小的人工泪液，而夜晚使用黏性大的人工泪液。至于人工泪液的使用频率，对于计算机相关工作人员需要更频繁。
此外，必要时采用泪点栓塞减少结膜囊中的泪液流失，甚至血清的使用均是治疗难治性干眼的重要手段。

干眼症的病因治疗
&nbsp; &nbsp; &nbsp; 干眼的病因治疗是更需要关心的问题，争论眼表炎症是干眼的病因还是结果不重要，但是通过抗炎症治疗采用如糖皮质激素、抗生素和环孢素等能有效减轻眼表炎症、减少人工泪液的使用量。
&nbsp; &nbsp; &nbsp; 这些药物包括糖皮质激素滴眼液，特别是不含防腐剂的糖皮质激素滴眼液和非甾体类消炎药滴眼液减轻眼表炎症，口服四环素恢复睑板腺分泌脂肪酸的正常成分，减少泪液蒸发，0．05%环孢素滴眼液有助于减轻眼表炎症，增加泪液分泌量，口服Ω-3脂肪酸添加剂减轻眼睑炎症，恢复睑板腺分泌液的正常构成，从而使泪膜更稳定。
&nbsp; &nbsp; &nbsp; 在病因治疗中也不能忽视一些传统治疗方法的作用，根据采用性激素水平的变化对更年期妇女给予补充可以缓解口干甚至眼干，中药滋阴生津也可能有积极的调理作用，还有报道采用穴位针灸和敷贴刺激也具有恢复泪液分泌的作用。
&nbsp; &nbsp; &nbsp; 对于极重度干眼患者，如Steven-Johnson综合征、瘢痕性类天疱疹和Sjgren’s综合征，由于患者的泪腺和副泪腺均完全破坏，因此，治疗的目的主要是保护角膜的完整性，恢复泪液分泌，减少眼表和睑缘的炎症。
由于自体颌下腺移植手术复杂，需要口腔科配合手术，因此自体唇腺移植也许是更有希望的干眼治疗手段。
&nbsp; &nbsp; &nbsp; 总之，干眼作为一种影响人们生活质量的多因素疾病，正确诊断和选择合适的治疗方法才能更有效地缓解患者的眼部不适，使患者逐渐适应眼干状态，提高生活质量。

眼睛出现红血丝，以为是没休息好？

五月 18, 2022

这就给大家说一说缓解的方法~
&nbsp; &nbsp; &nbsp; 年轻人眼睛出现红血丝若排除用眼过度的情况，休息几天便可缓解，一般这种情况最多两三个星期就可以自行吸收了。如果出现好几天都无法缓解的情况，一定要及时到医院检查，避免耽误治疗。
&nbsp; &nbsp; &nbsp; 老年人出现这样的情况，要提防是否是动脉硬化，更容易导致血管脆性增加，局部结膜小血管破裂、出血，最好及时到医院查眼底。

哪些原因造成此类情况
如长期熬夜、长时间观看电脑，揉眼睛等。
无论是在生活还是在工作当中，几乎每天都要对着电脑和自己的手机，如果视疲劳过度就会很容易让眼睛里面出现红血丝。如果没有长期的改正和良好的措施方法，那么很容易使我们的眼睛视力变得越来越差。
当肝火旺盛的时候或许就能够引起红血丝出现肝火旺中红血丝就会越多。所以说我们在生活当中应该吃一些降肝火的食物，比如说苦瓜等。
&nbsp; &nbsp; &nbsp;很多女性都习惯化眼妆，浓妆往往损伤眼部四周的肌肤，有的化妆品还会直接损伤眼睛。
&nbsp; &nbsp; &nbsp;1、减轻眼部干涩情况。及时休息，经常眨眼睛，让眼睛有泪液湿润眼睛。
&nbsp; &nbsp; &nbsp;2、用干净的棉签清洁眼睑，以免因残屑、油脂、细菌、化妆品等尘屑导致眼睛发炎，布满血丝。
&nbsp; &nbsp; &nbsp;3、保证充足的睡眠。保持充足的睡眠对眼睛非常重要，要保证自己每天都能达到7个小时的睡眠。充足的睡眠是有利于眼睛的滋润，减少眼睛的红血丝。
&nbsp; &nbsp; &nbsp;4、看电脑时注意距离。上班族记得眼睛与电脑屏幕的距离不可少于30公分，最好不要长时间面对电脑、电视、书籍，建议每一两个小时停5分-15分钟。
&nbsp; &nbsp; &nbsp;5、眼睛休息的时候可以用毛巾冷敷。以冷毛巾敷眼部，可收缩血管，滋润眼睛。
