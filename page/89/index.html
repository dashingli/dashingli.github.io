如何拥有一双漂亮的眼睛

二月 22, 2019

&nbsp; &nbsp; &nbsp;&nbsp;爱大爱表示，长期使用电脑的人普遍患有干眼症，即容易眼干、眼红和疲倦。这是因为电脑荧光屏由小荧光点组成，眼睛必须不断地调整焦距，以保证视物清晰，时间过长，眼肌会过于疲劳。眼睛出现酸痛、发胀、干涩、视力模糊等问题，很容易导致头痛、注意力不集中、情绪烦燥、反应迟钝、疲乏无力和失眠、恶心、欲吐等一系列疲劳症状。而全身不适等症状，多在年老体弱者和亚健康人群中出现。
&nbsp; &nbsp; &nbsp; 同时，眼睛长时间聚焦时，肌肉的紧张度会降低，影响大脑皮层调节肌肉运动的区域，从而使神经的兴奋性下降，使其对颈部、肩部肌肉运动的控制失调，引起肌肉张力增高。当这种紧张长期不能缓解时，相应部位就会出现慢性疼痛。
&nbsp; &nbsp; &nbsp;&nbsp;拥有一双健康、美丽的眼睛，对一个人来说是多么地重要。但是白内障、沙眼、青光眼、夜盲症、视网膜病变等各种眼病正在威胁、困扰着每一个人。提高爱眼、护眼意识，预防眼部疾病的发生已经迫在眉睫。导致眼病的因素多种多样，其中营养摄入不均衡，使得视神经、眼底得不到足够的养分是引发眼病的一个重要原因。
&nbsp; &nbsp; &nbsp; 蓝莓果实中除了常规的糖、酸和VC外，富含VE、VA等多种维生素、熊果苷、蛋白质、花青苷、食用纤维以及丰富的矿物质元素。除了丰富的营养元素外，爱大爱眼科专家家还发现，蓝莓中富含的花青素，可直接加速视网膜、视紫质的合成与再生，让眼睛绽放活力光彩；蓝莓能帮助营养预防近视、干眼症等各类眼部问题。

&nbsp; &nbsp; &nbsp; 另外，鳕鱼肝油中富含的天然维生素A，对眼睛视网膜感光功能等也起着一定的作用。所以对于学生族、长期使用电脑或长期佩戴隐形眼镜的人群来说，若是同时服用些蓝莓和鳕鱼肝油，双管齐下保护眼睛，对解决各种眼部干燥、干涩、干眼症、视力下降等眼部问题来说都是较为不错的选择。

烧烤吃多了也会得近视眼

一月 22, 2019

&nbsp; &nbsp; &nbsp;&nbsp;烧烤吃多了也会得近视眼？美国的一个研究结果表明：摄入烧煮、熏烤太过的蛋白质类食物，如烤羊肉串、炸鱼串等，将严重影响青少年的视力，促成近视眼。

&nbsp; &nbsp; &nbsp; 近视眼的形成是在眼球发育期，所以，眼球发育期就是近视眼形成的高峰期。人的眼球发育期是在1218岁，大约1820岁停止。这正是青少年求知欲强烈，读书学习考大学的时期。紧张的学业、繁重的功课，致使眼睛睫状肌长期持续地收缩，先造成调节痉挛，视力疲劳，此时若再食用过多烧煮、熏烤太过的蛋白质类食物，可使人体内钙的代谢发生异常，造成微量元素钙的缺乏，其作用好比催化剂，促进了近视眼的发生。
&nbsp; &nbsp; &nbsp;&nbsp;大量的临床资料及动物试验说明：近视眼的形成与机体缺乏钙、铬等微量元素有关。微量元素钙、铬对于眼内液压的调节、维持正常眼压、眼球壁硬度以及防止眼睛近视起着重要的作用。
&nbsp; &nbsp; &nbsp; 大自然为人类提供了富含各种人体所必需的营养和微量元素的食物，如果偏食很容易引起一些元素的短缺，机体会因摄入不足而逐渐导致近视眼。如果摄入过多烧煮、熏烤太过的蛋白质类食物，会造成体内缺钙。
&nbsp; &nbsp; &nbsp;&nbsp;另外，若进食过量的糖会导致体内微量元素铬的储存减少。同时，糖是酸性物质，在机体新陈代谢过程中会大量的消耗体内的碱性钙，结果同样也会造成体内缺钙引起近视。
&nbsp; &nbsp; &nbsp; 这么说来，青少年近视与饮食密切相关。过去人们只注重用眼卫生，忽视了饮食习惯与营养要求。所以为预防近视眼的发生，平时除了养成良好的用眼卫生习惯、坚持锻炼身体、注意眼睛保健外，还要养成良好的饮食习惯。

冬季注意预防几种眼病

一月 8, 2019

&nbsp; &nbsp; &nbsp; 不知从什么时候起，流传这么一种说法：“人全身上下，唯独眼睛不怕冷。”其实，在寒冷的冬季，眼睛也遭受各种困扰。随着气温的降低，眼疾患者也越来越多，几乎每天都有人咨询或应诊。为此，专家提醒广大市民，寒冬时节要慎防4种眼病，别让自己的眼睛“着凉”。

干眼症
&nbsp; &nbsp; &nbsp;&nbsp;冬天受寒风刺激，眼睛易干涩、刺痛，尤其是在寒冷干燥时更容易发生。应多补充人工泪液，注意保湿，出门要戴防风镜。
青光眼
&nbsp; &nbsp; &nbsp;&nbsp;青光眼是以眼内压力升高、视乳头血流灌注不良、合并视功能障碍的一种常见致盲眼病。此病分先天性、原发性和继发性3种。除先天性外，后两种均以冬季发病率最高，尤其是在强冷空气过境后24小时内最容易诱发。这是气温降幅过大，使植物神经调节功能发生紊乱而干扰了血压，造成眼压波动所致。特别是65岁以上的老人，一旦发现眼部不适伴头痛脑胀，反复出现视力模糊、虹视(看灯周围有彩圈)等症状时，就应及时去医院检查。
结膜下出血
&nbsp; &nbsp;&nbsp;天气变冷时血管会收缩，血压跟着升高。若血管弹性差，则容易破裂造成结膜下出血。通常结膜下出血不痛不痒，当眼睛突然充血、眼白部分出现血块，就要小心是否得了结膜下出血。
眼中风
&nbsp; &nbsp; &nbsp; 又称“视网膜血管阻塞”，是由于天冷血管收缩造成阻塞出血。高血压、糖尿病或胆固醇过高的人要格外小心。眼中风是突发且无痛的。视物模糊，甚至单眼视力突然丧失，必须尽快找眼科专业医师检查。平时要注意控制血压，包括体重、盐分摄取量，多运动也是维持正常血压的方法。

莫让屏幕坏了你的眼神

一月 3, 2019

我国青少年近视眼的流行调查表明，近视眼的发生与学生课业负担过重、近距离用眼或照明不足、阅读距离过近、阅读时间过久、字体不清或过小以及姿势不良等具有明显的关系。
而从20世纪90年代以来，人们对外界信息的获取更多地依赖了视觉信息，书刊及报章杂志、电视和网络电脑等，数字化教学及办公的实现将使人们的视觉环境更多地面对电脑终端。毋庸置疑，青少年近视眼的发生将越来越高。

爱大爱认为，青少年近视的发生除部分是由于父母亲有高度近视的遗传外，大部分还属于用眼不当所造成，如因长时间看电视、看电脑所造成的“视频终端综合征”，越来越成为青少年视力下降的主要原因。眼科医生们在临床治疗中发现，由于视觉环境的改变，“视频终端综合征”的发生将大量增加，所谓视频终端综合征是指由于长期注视电脑或者电视屏幕，眼瞬目动作减少，泪液不能很好地均匀分布在眼表，导致眼干、眼涩，疲劳，慢性结膜炎等眼部不适症状。这些综合征可能是干眼症的一种。据调查表明，在我国老年人中的干眼患病率可能在50%左右，尽管干眼症一般不导致视力的下降，但眼部极为不适，严重影响患者的生活质量，而且，干眼的存在，不利于一些眼表疾患的恢复。
另外，由于都市化的进程，市内缺少可以提供给青少年极目远眺的环境。而且，在城市中，由于马路、公用设施等建筑物，常彻夜通明，改变了光线的自然节律性的昼夜变化。城市中并没有真正的“黑夜”。已有大量的研究表明，光线的明亮程度可以明显地干扰动物的生物节律，干扰人体松果体内褪黑激素的分泌。已有实验证明，在美国夜间开灯睡眠的孩子中，近视眼的发病率很高。
还有，家长在对孩子的眼方法上存在着误区。随着人们健康意识的提高，在大多数居民家中，可能都备有一到两种眼药水，用于“消疲劳”或者“消炎”。殊不知，眼科，千变万化，仅凭外眼的观察，不能把握病的根本随意用药，常可祸害终生。如“红眼”可能为结膜炎，也可能为角膜炎，角膜溃疡，结膜下出血，急性虹膜睫状体炎，也可能为急性闭角型青光眼。
而一些眼痒或者慢性结膜炎的患者，因视力无碍，便自行到药店买药，药店工作人员并非眼科医生，也并不完全知道病人的用药时间长短便根据病人需要给药，就这样，一大批的“皮质类固醇性青光眼”就产生了，而且，这种类型的青光眼往往不易及时发现，至晚期才被确诊为青光眼，此时，损害的视力已无可挽回。
因此，爱大爱建议，在目前的医疗条件下，家庭中没必要备有眼科用药，有眼部问题出现，应及时找眼科医生或者专业的眼睛保健机构进行咨询，即使家庭中备有药物或需长期使用，也应不定期地进行咨询。

眼睛畏光是哪里出了问题

一月 2, 2019

太阳底下或是碰到强光照射时，眼睛会有怕光及不舒适感，正常反应会将眼睛眯小或使用其他物品帮忙隔离阳光的照射，以减轻眼睛不适的现象，但有些人在普通的光度下也会觉得不适，可称这些人的眼睛有畏光的情形，引起眼睛畏光的因素有：

1、戴眼镜时间越长，眼睛怕光，度数也越高;
2、散光、近视会有一定程度的眼睛怕光;
3、散瞳剂使用次数较多、时间较长时;
4、弱视造成感光系统退化、虹膜对光度的控制失调，也会引起眼睛怕光;
5、视网膜老化、退化、病变者，也容易出现眼睛怕光;
6、长期在暗室工作或在强光照射下工作者，也会有眼睛怕光的现象;
7、成年人接受近视激光开刀手术后容易出现眼睛怕光。

近视预防从日常生活做起

十二月 21, 2018

&nbsp; &nbsp; &nbsp; 近视往往在我们不经意间就发生了，严重影响生活和学习。爱大爱视力保健中心表示，预防近视成为当下青少年的护眼的首要任务。那么，青少年日常生活应该怎样预防近视的发生呢？下面爱大爱就为你提供几点建议：

一、定期检查裸眼视力
&nbsp; &nbsp; &nbsp;&nbsp;裸眼视力的大小是由晶体的屈光度、透明度及眼轴的长短共同决定的，裸眼视力能够很精准的反映出眼睛的屈光度数。
二、坚持做眼保健操
&nbsp; &nbsp; &nbsp; 眼保健操可以提高眼部及头部的血液循环，调节睫状肌、改善眼疲劳及视疲劳。眼保健操通过对周围眼部穴位的按摩，使眼内气血通畅、改善眼球的营养供应，从而有效的消除处于痉挛状态的睫状肌的目的。
三、多参加户外活动
&nbsp; &nbsp; &nbsp; 眼科学家发现，造成目前近视泛滥成灾的局面主要是由于儿童户外活动的机会较少的缘故，多参加户外活动、经常呼吸户外的新鲜空气、让眼睛常时间视远，都是有效预防近视的方法。
&nbsp; &nbsp; &nbsp;&nbsp;近视的保健是社会合作的结果，家长要耐心的对待孩子，近视以后，可能成绩会下降，也不要急于批评孩子，对于学校来说，一些有利的设施要健全，社会更是要加大宣传，为孩子远离近视做一份贡献。
