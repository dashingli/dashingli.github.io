简述电脑对于眼睛的压力

十二月 17, 2018

&nbsp; &nbsp; &nbsp; 当今的社会状况，看电脑现、上网则成了宅人必不可少的“娱乐活动”，特别是一些青少年，更爱坐在电脑机前消磨时间。结果弄得头晕脑胀，疲乏无力，医学界称为“电脑病”。可别小看“电脑病”，危害可大着，“看”错了不仅直接伤害眼睛，而且严重时可能要了你的命!下面爱大爱为您讲解。
&nbsp; &nbsp; &nbsp; 关灯看电脑：人们每天都离不开电脑，而电脑的使用都需要眼睛高度集中，时间一长，不少人就出现了不同程度的眼疲劳症状，尤其是整天与电脑打交道的白领人群中，90%以上都有不同程度的眼疲劳。时间长了，就会影响视力甚至损害眼睛。

据研究人员一项最新的调查显示，与长时间看电脑相比，频繁用眼对眼睛的伤害更大。看电脑对眼睛的伤害，是对眼睛刺激有持续性的效果，因为多数电脑在运作的时候，会出现光线的变化情况，此时光线骤变，会对眼睛造成一定的刺激，过于频繁，更容易导致眼睛疲劳。
&nbsp; &nbsp; &nbsp; 公交车上看电脑：现在，很多城市的公交车上都装有移动电脑设备，在枯燥的乘车路途中，丰富的电脑节目成了不少乘客惟一的消遣。但是，大多数人在乘车时，会长时间盯着电脑屏幕看，其实这样做是不科学的。很容易使人出现眼睛干涩、酸胀、疼痛等不适症状，甚至还会出现头昏、恶心、呕吐等不良症状，还会加重近视，甚至诱发青光眼急性发作。
&nbsp; &nbsp; &nbsp; 酒后看电脑：酒后看电脑对眼睛更加不利，因酒本来就能损伤眼睛，使眼睛充血，视神经萎缩，严重的可导致失明。酒后看电脑等于火上浇油，对视力大有损伤。因此，酒后不宜看电脑，一般人看电脑也不宜时间太长。

预防眼干的饮食调养方法

十二月 14, 2018

&nbsp; &nbsp; &nbsp; 用眼过度是眼睛疲劳的主因之一。眼睛疲劳的一般症状是看东西久了会模糊、眼睛干涩、头昏痛、恶心、呕吐等。保护眼睛减缓眼疲劳，除了保持正确操作姿势、保证休息之外，在饮食中加入一些营养眼睛的食物，不但可以缓解这些症状，还可以消除眼疲劳。

&nbsp; &nbsp; &nbsp; 补充维生素维生素A是预防眼干、视力衰退、夜盲症的良方，胡萝卜及红枣中含量最多。维生素B是视觉神经的营养来源之一，维他命B1不足，眼睛容易疲劳，维生素B2不足，容易引起角膜炎。
&nbsp; &nbsp; &nbsp;&nbsp;所以，眼疲劳者要注意饮食和营养的平衡，平时多吃些粗粮、杂粮、红绿蔬菜、薯类、豆类、水果等含有维生素、蛋白质和纤维素的食物。也可以多吃些芝麻、大豆、鲜奶、小麦胚芽等食物。
&nbsp; &nbsp; &nbsp; 现代医学研究表明，维生素与眼疾的发生、视力的好坏有着非常密切的关系。用眼过多者，需要更多的眼睛所需的维生素及矿物质。合理补充眼睛所需的营养素，对保护眼睛、防止视力伤害、防治眼疾、提高视力非常重要。
要预防干眼症，日常应保持情绪稳定，以免肝火上炎，用眼时间不宜太长，少吃燥热食物多吃鱼。另一方面要注意用眼卫生，不要长时间地看电脑屏幕，每隔一段时间休息一下，比如远眺或者闭眼休息等。如果眼睛干涩还可以去药店购买一些人工泪液滴眼，帮助消除视觉疲劳。另一方面，调整饮食结构，增加富含维生素A的食物的摄入，也有助于干眼病的防治。
维生素A在维持视觉中的作用已为人们所熟知。中国营养学会推荐每日维生素A的适宜摄入量为男性800国际单位、女性700国际单位，而全国营养调查数据表明，我国居民维生素A摄入不足，主要与饮食结构有关，也与偏食、挑食有关。

解析“近视王国”的近视根源

十二月 13, 2018

&nbsp; &nbsp; &nbsp; &nbsp;中国是一个“近视王国”，相信这个观点并不会难以让人们接受了。如今中小学生的近视率达到50%以上是一个多么恐怖的数据？那么除了不可避免的遗传基因导致的近视以外其他的近视又是如何发生的呢？有人说，是因为现在的孩子常常看电视、上网导致孩子视力开始出现问题。但是你们有没有想过电视、电脑是哪里最先开始使用，开始普及？是西方的发达国家！所以大家要对近视的产生要有一个正确的认识，而不是把近视的原因仅仅归于此。其实真正的原因就是我们国人的健康观念不正确，并且还以中国现今的教育体制有着直接的关系。

青少年视力为什么越来越差

十二月 10, 2018

&nbsp; &nbsp; &nbsp;&nbsp;眼睛是人类观察世界的重要器官，是人类沟通外部世界的渠道，眼睛是心灵的窗口。如今戴眼镜的人越来越多，尤其青少年眼疾屈光不正的人也越来越多，一到寒暑假，很多父母都要忙着带孩子去配眼镜。在学校，有些班级更是全班学生都戴眼镜。青少年的眼健康已经不容忽视。保护青少年视力刻不容缓。
&nbsp; &nbsp; &nbsp; 研究发现，长时间看电视或上网，可使人的视力减退，产生视疲劳，引发干眼症、慢性结膜炎等。随着生活空间的改变，电视、电脑成为青少年娱乐的主要对象，视力低下的发展具有渐进的特点，一旦形成就会失去防治机会。因此要提醒青少年注意看电视、玩游戏、用电脑的时间，画面的亮度、对比度要柔和、适宜。

&nbsp; &nbsp; &nbsp;&nbsp;生活中，一些人得了近视却不戴眼镜，爱大爱认为，从用眼卫生的角度，近视到一定程度，就应该戴眼镜，千万不要轻视。出现视力下降时，要及时到眼科就诊，鉴别是否为真性近视，可靠的方法是散瞳验光。
&nbsp; &nbsp; &nbsp; 如为假性近视，可试用睫状肌松弛剂类药物进行治疗，不要随便佩戴眼镜或手术。如为真性近视，应首选戴镜，但一定要注意验光的准确性和眼镜的质量。如果是中高度近视应注意避免剧烈冲击性头部运动，防止视网膜脱离，并定期进行眼科检查。

眼睛干涩的病因

十一月 30, 2018

引起干眼症直接原因除了如配戴隐形眼镜不注意用眼卫生等因素外，主要是人们专注于工作。学习时，眼睛眨动的次数减少，从而影响泪膜的功能，使角膜、结膜失去原来的湿润和光滑，导致眼睛干涩，这些都是干眼症的病因。
据有关资料显示，一般人眨眼次数为每分钟15次，看书时为10次，操作电脑时为4～5次，而驾车或玩游戏机时仅为2～3次。如果眨眼次数减少75%，分泌的泪液就会大大减少，从而不能形成完整的泪膜，使角膜受到损伤。

导致干眼症常见原因有下列这些：
1、由于长时间从事计算机操作，汽车驾驶，读书及其他精细作业，瞬目次数减少，引起泪液减少。
2、由于自身眼裂大，或所处房间干燥等，引起泪液的蒸发。
3、由于隐形眼镜，过敏性结膜炎，大气污染，紫外线等其他原因，而引起的泪液减少，质量下降。
4、服用部分降压药及部分精神安定剂对泪液的产生有影响。
5、由于高龄，睡眠不足，精神紧张等生理原因，引起泪液质量下降。
此外，干眼症还与空气干燥、粉尘污染、使用镇静类药物等因素有很大关系。上述症状如得不到及时治疗会引起角膜混浊。溃疡，甚至失明等严重后果。因此，经常用眼过度的人，平时应注意用眼卫生，利用闭目休息、远眺、做眼保健操等方式缓解眼部的疲劳，多吃一些富含维生素A的食物来给眼睛补充营养。

近视的预防措施

十一月 29, 2018

&nbsp; &nbsp; &nbsp;&nbsp;许多家长为了预防和治疗孩子的近视眼想尽了办法，却仍不能阻止孩子近视眼的发生、发展。其实与其被动地保护婴幼儿视力，不如帮助孩子建立健康的生活内容和方式，从根本上解决孩子用眼过度的问题。父母可以从以下几个方面入手：

&nbsp; &nbsp; &nbsp; 一是帮助孩子选择电视节目，减少看电视的时间。爸爸妈妈千万不要用电视做保姆。婴幼儿虽然能从电视中学到许多东西，但专家经研究发现，被动学习对孩子大脑发育益处远低于主动学习。父母应学会关掉电视，用其他方式娱乐。
&nbsp; &nbsp; &nbsp;&nbsp;二是用外出活动代替电子游戏。大家都知道电子游戏损害视力，但苦于无法将孩子从游戏机旁拉开。日本的母亲每天都会安排固定的时间带孩子外出游戏，一些妈妈会结成互助小组，由一个母亲带领几个孩子玩耍，孩子不仅锻炼了身体，减少孩子玩游戏机的时间，保护了视力，也增强了社交能力。这些中国的父母应该好好学习。
&nbsp; &nbsp; &nbsp; 三是挑选精华读物让孩子阅读。有些父母望子成龙，十分重视幼儿教育，从小就让孩子阅读许多书籍，孩子眼睛过度疲劳，视力就会下降。幼儿教育应少而精，尤其应注意每次时间不应超过15—20分钟，这样学习才符合幼儿学习心理，效果会更好。

是不是蓝光都会对眼睛造成伤害

十一月 19, 2018

&nbsp; &nbsp; &nbsp;&nbsp;据世界卫生组织爱眼协会介绍：电子产品所发出的亮度更高，甚至亮到发出蓝色光芒的蓝光，对于人类的双眼有着累积性、附加性的伤害，眼睛长期接受这种波长为400-500纳米的高能量的蓝光照射，会对视网膜产生强烈的光伤害，加速黄斑区细胞氧化，也就是说，我们几乎每天都要看的手机、工作中时时刻刻都要用到的电脑，都会对我们的眼睛造成伤害。
&nbsp; &nbsp; &nbsp; 眼科专家报告称，当人造蓝光进入我们的双眼，会引起为人们双眼的功能失调，尤其是电子产品屏幕上所发出的不规则频率的短波蓝光，其强大的穿透力能穿透晶状体，直接到达视网膜，对视网膜产生伤害，还可能导致黄斑区病变。

&nbsp; &nbsp; &nbsp; 那么是蓝光都会对眼睛造成严重的伤害吗？
&nbsp; &nbsp; &nbsp; 对此，专家表示：这也不全是有害蓝光，因为自然界有很多不同颜色的光谱，蓝光只是其中之一，是指波长在400—500纳米之间的光。波长越短，能量越高，穿透力越强，所以蓝光对于眼睛的影响为许多人所关注。各种关于蓝光危害的说法也流传开。其实蓝光在我们生活中无处不在，我们要有正确的认识和评估，不要过大夸张蓝光的危害性。
&nbsp; &nbsp; &nbsp; 蓝光并不都是有害蓝光，真正有害的是400到440或450纳米以内的蓝光，而450到500纳米之间的蓝光有一种调整生物节律的作用，睡眠、情绪、记忆力等都与之相关，对人体反而是有益的。而且短波蓝光是不是一定会对人眼产生伤害，这与照射强度与照射时间有关，当照度达到一定程度后，持续两个小时以上，才有可能对视网膜产生损害。
&nbsp; &nbsp; &nbsp; 所以虽说不要过大夸张蓝光的危害性，但对于部分长期处于有危害蓝光下的人，也可以适当进行一些防护，在医生指导下佩戴合格的防蓝光眼镜，避免不合格镜片导致色觉偏差及视疲劳等。

冬季护眼：南瓜最“养眼”

十一月 15, 2018

&nbsp; &nbsp; &nbsp; 进入冬季，气温的降低，使得本来就不爱运动的那些人，更喜欢窝在家里了。周末除了睡觉就是电脑，这样下去眼睛哪受得了，很容易近视。这该怎么办呢？真不想出去走动呢？凉飕飕的……别急，小编今天就带来一个好方法：冬季护眼南瓜最“养眼”，南瓜可以帮你保护视力哦。

&nbsp; &nbsp; &nbsp; 据报道，常吃南瓜人群中，80岁以上的高龄者，仍能保持良好视力，不戴眼镜也可看报纸。
&nbsp; &nbsp; &nbsp; 南瓜有抗氧化、防衰老、护视力的作用，因为其含有丰富的维生素E和β-胡萝卜素。β-胡萝卜素在肌体内能转变为对人体具有重要生理作用的维生素A，对保护视力、预防眼疾、维持人体上皮组织的健康、促进儿童的生长发育等均有重要作用，同时对维持正常视觉和明目、促进骨骼发育及肌体抗癌防癌方面具有重要生理作用。
&nbsp; &nbsp; &nbsp;&nbsp;作为食物中维生素A的来源，南瓜食品与动物性食品(如肝脏、蛋黄等)相比，经济实惠。
&nbsp; &nbsp; &nbsp; 其实南瓜不仅可以保护视力，还有其他功效。
&nbsp; &nbsp; &nbsp; 常吃南瓜有助于防治糖尿病，因为南瓜中含有丰富的微量元素钴和果胶。钴的含量较高，是其他任何蔬菜都不可相比的，它是胰岛细胞合成胰岛素所必需的微量元素。果胶则可延缓肠道对糖和脂质吸收。
