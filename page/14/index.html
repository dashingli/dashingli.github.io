爱护眼睛要从日常做起

五月 18, 2022

&nbsp; &nbsp; &nbsp; 镜片是眼镜重要的组成部分，也是保障视觉清晰的重要元件，因此要做好日常的镜片护理。最好常备擦镜纸每天擦拭镜片；

1.镜片上有灰尘，要用清水进行冲洗，把镜片灰尘冲洗干净后，可用镜布轻轻吸去或擦去水渍；
2.镜片上有油脂，可用少许洗洁精配合水进行眼镜冲洗，再擦拭干净（洗洁精需稀释，最好选择中性洗洁精）；
3.定期去眼镜店进行保养清洗，用超声波清洗器把镜框夹缝中鼻托上的油脂灰尘清洗掉；
&nbsp; &nbsp; &nbsp; 为避免公用清洗器带来的交叉感染等问题，家中可自备一台小型超声波清洗器，定期清洗，注意超声震动会使螺丝松动，需及时用专用螺丝刀拧紧。镜架的调整也很重要，如果镜架发生变形，需请专业人员使用专用工具进行调整，否则易导致镜架和镜片的损坏，影响眼镜的使用寿命。
&nbsp; &nbsp; &nbsp; 爱护眼睛要从日常做起，养成良好的用眼习惯。爱大爱建议您定期到正规、专业的医疗机构进行眼睛检查，及时更换和调整眼镜，给自己一个健康清晰的视觉。

别让这些东西伤害孩子的眼睛

五月 18, 2022

日常生活中，即便我们再小心翼翼，也免不了一不小心就让孩子的眼睛受到伤害，为此，爱大爱小编特地整理下日常生活中常见的但是我们容易忽略的东西，以便您更好地保护孩子的眼睛。

日常的洗刷工具
不要看宝宝小就随便找一个脸盆，或者为了表示你们的母子或者父子情深就与宝宝共用一套洗刷的工具。宝宝也是需要一套专用的。
共用的洗刷工具，最容易使父母与宝宝之间产生交叉感染。毛巾脸盆等也要定期更换清洗。也不可以用成人的手帕或直接用手去擦宝宝的眼睛。给宝宝清洗眼部的时候，先把几个棉球在湿水里沾湿，再挤干水分，每一只眼睛都要换一个新的棉球，从内眼角向外眼角擦就可以。
家中忽视的噪音
吵杂的噪音不仅会令宝宝心烦多梦，睡眠不好，而且还会影响宝宝的视力。有研究指出，噪音能使人眼对光亮度的敏感性降低，还能使视力清晰度的稳定性下降。例如噪音达70分贝时，视力清晰度恢复到稳定状态时需要20分钟，而噪音在85分贝时，至少需要1个多小时。
噪音还会使色觉、视野发生异常，使眼睛对运动物体的对称性平衡反应失灵。因此，小宝宝的家里不要摆放高噪音的家用电器，看电视或听歌曲时，为了宝宝，也为了自己，不要把声音放得太大，宝宝居室里要注意环境的安静。
外出游玩的强光
在阳光明媚的日子，让宝宝在外头走动走动，抱着宝宝外出是一个很不错的选择。但是，宝宝到户外活动要防止太阳直射眼睛。如果阳光太强，可以临时用纱巾遮盖一下，以避免强光刺激宝宝的眼睛。
还有一点需要注意的是，爸爸妈妈都想把宝宝的每时每刻都记录起来，所以，非常喜欢带着宝宝到各个地方，拍下各种可爱的照片。但是，婴幼儿照相时也不能用闪光灯照相，因为闪光灯的强光会损伤宝宝视网膜，记得要关闭闪光灯。
宝宝床上的吊挂物
挂在宝宝床头上的小玩具也可能会影响到宝宝的视力，宝宝的眼光总喜欢追随着可爱的小物体，如果挂的位置一成不变，可能会造成宝宝眼睛斜视。所以宝宝床头的玩具要挂对位置。
很多妈妈喜欢在小宝宝的床栏中间系一根绳，上面悬挂一些可爱的小玩具。因为宝宝多是远视眼，如果把玩具放得特别近，要使劲调节眼睛才能看得见，这样时间久了，宝宝的眼睛较长时间地向中间旋转，就有可能发展成内斜视。

家中电视的X线
当家里的电视机开着时，显像管会发出一定量的X线，宝宝对X线特别敏感，如果大人抱着宝宝看电视，使宝宝吸收过多的X线，宝宝则会出现乏力、食欲不振、营养不良、白细胞减少、发育迟缓等现象。
为了防止无形的X射线，宝宝看电视要分月龄分时间，越小月龄的宝宝每天看电视的时间越要缩短，1岁左右的宝宝每天只能看10分钟，那么随着宝宝的长大，电视时间可以逐渐延长到半小时。
电子用品、Ipad
现在电子产品充斥着市场，宝宝经常会盯着手机或者ipad一直不眨眼，眼睛累着了、视力也差了。
长时间、近距离地用眼，会导致宝宝的视力直线下降。在此期间，要特别注意限制宝宝近距离作业的时间。一般每次不应超过30分钟。经常带宝宝向远处眺望，引导宝宝努力辨认远处的一个目标，这样有利于眼部肌肉的放松，预防近视眼。爸爸妈妈们，千万不要因为宝宝用一双水亮水亮的眼睛对着你就心软，这事情是绝对不可以有丝毫的退让。
各式各样的玩具
人的眼球部分暴露在眼眶的外面，易遭受外界各种致伤因素而损伤，由于宝宝的自我保护能力差，受眼外伤的机会相对较多，不要给宝宝玩任何带有锐角的玩具。
会走路的宝宝要小心预防眼外伤。不要让宝宝拿刀、剪、针等尖锐物体，带锐角的家具也最好能把角包上，以免因走路不稳摔倒而让锐器刺伤眼球。另外厨房里的开水、热油、火苗，家里的宠物，节假日的鞭炮都可能给宝宝的眼睛造成损伤。

【爱大爱眼睛养护】春天时刻眼睛干燥需“补水”

五月 18, 2022

&nbsp; &nbsp; &nbsp;&nbsp;干眼症是一种很常见的眼科病，正常人的眼睛表面有一层很薄的泪液膜，它可以润滑眼球、保持角膜光滑透亮，“水汪汪”就是对这层保护膜的生动形容。
&nbsp; &nbsp; &nbsp; 眼科专家把眼泪誉为“眼球表面的血液替代品”，眼泪不足时，可导致角膜上皮角化，严重缺泪时，还会致角膜溃疡及感染发炎。如果长时间过度用眼或眨眼次数少，眼球表面的泪液就会不断蒸发，使泪膜脂层变薄，引起眼睛干燥。

&nbsp; &nbsp; &nbsp;&nbsp;如果遇上到春寒或者干燥春季，干眼症的症状就会更加恶化。此外，干眼症和嗜辛好辣、烟酒刺激等不良饮食习惯以及女性更年期、内分泌失调等疾病也有密切的关系。
&nbsp; &nbsp; &nbsp;&nbsp;干眼病是可以预防的，多刻意地眨眨眼、适当使用眼药水和补充维生素都可以提高眼睛的调节能力。在高强度用眼时，要注意别长时间“目不转睛”，刻意地眨眨眼可以使泪水均匀地涂在角膜和结膜表面，减少眼球水分蒸发过快；空调房间要采取一定的保湿措施，如定期喷喷水，减少烟雾粉尘刺激；同时可以适当使用一定量的人工眼药水滋润眼睛；在饮食方面可以多吃一些富含维生素的绿色蔬菜、蛋黄、胡萝卜、葡萄等，摄取充足的叶黄素。

&nbsp; &nbsp; &nbsp; 常见的症状是眼部干涩和异物感，其他症状有烧灼感、痒感、畏光、红痛、视物模糊易普疲劳、粘丝状分泌物等。如有上述症状，则应仔细询问病史，寻找病因。对于严重的眼干，应询问是否伴有口干、关节痛，以排除SS。
&nbsp; &nbsp; &nbsp; 干眼症症状的个体差异很大。大多数病人抱怨眼部异物感，烧灼感和一般的眼部不适。这些不适被典型地描述为刮擦感，眼干，疼痛，沙粒感，刺痛感或烧灼感。这些不适感往往是干眼症的证明，因为角膜的表面遍布感觉神经末梢。有相当比例的病人有畏光和间歇性模糊或其他的视力问题。
&nbsp; &nbsp; &nbsp; 干眼症的患者常常诉说自己的眼睛易于疲劳，使得读书和看电视发生困难。发生这些困难的原因是当工作需要注意力集中时，瞬目的频率明显减少。当瞬目的频率明显减少时，泪膜蒸发的时间将会明显延长。如果瞬目不足，角结膜暴露的时间将会超过but，从而导致在角膜表面形成一个或多个干点。
&nbsp; &nbsp; &nbsp;&nbsp;调查证实，每天在电脑前工作3小时以上的人中，90%的人都患有干眼症。而在未来5年中，干眼症患者人数还将以每年10%以上的速度上升。
&nbsp; &nbsp; &nbsp; 长时间戴隐形眼镜也容易得干眼症，这是由于隐形眼镜很容易吸附水分，从而导致眼球表面脱水。
&nbsp; &nbsp; &nbsp;&nbsp;干眼症目前还无法根治，临床上大多采用人工泪液缓解症状。人工泪液有水剂和膏体两种，症状轻重不同，使用的人工泪液也不相同。一旦停用，症状往往又会复发。

&nbsp; &nbsp; &nbsp; 保持良好的工作、生活习惯是预防干眼症的有效手段。首先要避免长时间操作电脑，注意中间休息。通常连续操作1小时，休息5至10分钟。休息时可以看看远处或做做眼保健操。其次要调整好显示器与眼睛的距离和位置，建议距离50至70厘米，位置略低于眼水平线10至20厘米，显示器的亮度也不要太亮，调节到最大亮度的一半就可以了，以看得清楚内容但比周围物体稍暗为宜。另外，长时间使用电脑时，最好不要戴隐形眼镜，以免加剧眼睛的干燥程度。
&nbsp; &nbsp; &nbsp; 此外，长期从事电脑操作的人，要注意饮食调理，应多吃豆制品、鱼、牛奶、核桃、青菜、大白菜、空心菜、西红柿及新鲜水果等。

如何确定孩子是弱视？

五月 18, 2022

也有家长说了，孩子才3岁，不会认视力表，可以查弱视吗？
&nbsp; &nbsp; &nbsp;&nbsp;当然可以。因为就算孩子不会认视力表，眼科医生也可以通过很多仪器来排查孩子有没有引起弱视的高危因素。有个4岁的小孩，还不会认视力表，但是验光发现有600度的远视，基本可以确定他90%扥可能性有弱视。

发现弱视了，该如何处理治疗呢？
弱视治疗的黄金时间是6~8岁，8岁以后开始治疗弱视，疗效就明显下降。如果在12岁之后才发现弱视并开始治疗，那么就非常难治愈了。
弱视治疗的办法有三种：验光配镜、遮盖、视觉训练。
（1）验光配镜
&nbsp; &nbsp; &nbsp; 大多数孩子是因为屈光不正、屈光参差引起的弱视，占到了70%。这类型的弱视需要验光配镜，那些由于双眼度数差别较大，戴框架眼镜不适应的孩子，也可以尝试戴隐形眼镜。
（2）遮盖
&nbsp; &nbsp; &nbsp; 在戴眼镜的基础上，把好的眼睛先遮挡起来，逼迫患儿用差的眼睛去看东西，这样迫使弱视眼的视觉系统得到更好的锻炼。单眼弱视，或者双眼弱视程度不一样的孩子，需要遮盖疗法。
（3）视觉训练
&nbsp; &nbsp; &nbsp; 视觉训练包括：串珠子、画图以及带各种视觉刺激功能的弱视训练仪，另外，还有3D/4D数字化弱视技术。
不同类型的弱视，解决办法是不一样的。
而形觉剥夺性弱视，如先天性白内障、上睑下垂，则需要通过手术先把这些病因去除掉，再处理弱视。
&nbsp; &nbsp; &nbsp;&nbsp;虽然弱视比近视可怕但也并不是什么绝症只要发现及时、治疗趁早绝大部分效果还是不错的

对于眼镜最好的保养是什么？

五月 18, 2022

&nbsp; &nbsp; &nbsp; 对健康双眼而言，眼泪是眼睛最好的护理液，因为泪液可以保持角膜湿润，本身对结膜囊就有清洁和保护作用。眼泪的化学成分非常复杂，除水和电解质外，还有含氮物质、糖类、有机酸、维生素、酶类、脂类等成分。其中溶菌酶等是天然抗菌物质，多种免疫球蛋白执行着非特异和特异的免疫功能。

&nbsp; &nbsp; &nbsp; 眼泪分泌同眨眼次数有关，两者呈正比关系。眨眼是种保护性神经反射，可以使泪水均匀地涂在角膜和结膜表面，以保持眼球润湿而不干燥。
&nbsp; &nbsp; &nbsp; 正常人每分钟眨眼约20次，倘若长时间睁眼凝视变动快速的屏幕，眨眼次数常减少至每分钟四到五次，眼泪分泌减少就会导致眼睛干燥酸涩，长期持续此状态就会引起干眼症，又称为干燥性角膜、结膜炎，症状除了眼睛干涩、痛痒外，还有灼热感和疲劳感，如乱用眼药水、乱用洗眼液、随意冲洗眼睛，非但保护不了眼睛，还极易对眼睛造成伤害！
&nbsp; &nbsp; &nbsp; 这是因为洗眼睛对环境、技术和药水要求都非常高，如果洗眼睛的器械消毒不严格，还会使眼睛受伤或传染结膜炎、沙眼等眼疾。据了解，目前市面上的洗眼液主要对眼睛起润湿作用，并附带消炎、清凉等功能。大多洗眼液是人工泪液加纤维素和一些维生素组成的，有的会添加些具有营养成分、杀菌消炎成分和防腐成分的东西，其成分组成与天然泪液相差甚远，它们的PH值、酸碱度和渗透压等也都不相同。
&nbsp; &nbsp; &nbsp;&nbsp;经常洗眼，不但会造成药物依赖，眼睛的自洁功能下降，还会对眼角膜、结膜造成伤害。用含抗菌成分的洗眼液还可能使眼内细菌产生抗药性，眼睛一旦出现感染，治起来也很麻烦。因为过度洗眼可能使原本轻微的角膜炎症因未对症下药而转成角膜溃疡。
&nbsp; &nbsp; &nbsp; 另外，为稳定药物性质或较长时间不被污染，绝大多数洗眼液都含防腐剂，如果含有的防腐剂过多，就可能改变眼球表面微环境，破坏眼球表面结构，损害原本紧密连接的角膜上皮细胞结构，出现角膜上皮脱落、缺损、糜烂，严重的甚至发生角膜溃疡、角膜溶解、穿孔以致失明。

近视，到底要怎样才能预防和延缓

五月 18, 2022

近视，到底要怎样才能预防和延缓
怎样预防近视
1.注意用眼卫生
2.坚持做眼保健操
3.劳逸结合，睡眠充足
4.注意营养，加强锻炼，增强体质
5.定期到专业眼镜公司检查视力，发现减退应及时矫正，防止近视加深。
保证充足的睡眠
保证充足的睡眠时间是消除疲劳、恢复工作能力的重要因素。因为眼睛在睡眠状态下，肌肉放松最充分，最易消除疲劳，也是保证身体健康所必不可少的。同时，在睡眠时内分泌激素增多，这对儿童青少年的生长发育也很重要。所以，家长和教师都应当重视学生睡眠，保证学生有充足的休息时间。

眼保健操
眼保健操是根据祖国医学的经络和推拿学说的原理，结合医疗体育而成的一种保护眼睛的自我按摩法。通过按摩眼睛四周的穴位以增强眼眶的血液循环，改善神经营养，消除眼内的过渡充血，达到解除眼疲劳的目的。实践表明，坚持做好眼保健操，对保护视力，预防近视具有一定的意义。每天可做数次，选择在学习中间的休息时间。
饮食结构合理
研究发现，饮食中增加蛋白质，减少碳水化合物供应，可使有遗传背景而发生近视的青少年减少或中止近视度数的增加。因此，为避免发生近视，少吃糖果和高糖食品。食糖过多，会使血液中产生大量酸性物质，酸与肌体内的食盐，特别是钙相结合，造成了血钙减少，这就会影响眼球壁的坚韧性，使眼轴易于伸长，助长了近视发生和发展。研究表面，预防近视可补充蛋白质、钙质、磷质、胡萝卜、豆芽、橘子、广柑、红枣等蔬菜水果也对预防近视有益。
一、平时注意营养，多吃新蔬菜，不要偏食。
二、看书或看报时，眼睛与书面保持30公分，45度角的距离，并避免长时间近距离使用眼睛。

六、每天做3—4次眼保健美操。

近视不带眼镜可以吗？

五月 18, 2022

“近视是眼轴距(眼球角膜到视网膜的距离)过长引起的，就像长高的个子一样不可逆转，只能控制。不少家长觉得戴眼镜很丑，不想让孩子戴。其实眼镜是控制孩子视力的主要手段，眼镜就像挑水的扁担，可以分担睫状肌的压力。

近视度数比较高的患者可以佩戴两副眼镜，一副眼镜是充分矫正视力的眼镜，看远时用，另一副可低300度在看近处时用，以减少调节力的使用，防止加深度数。但是高度近视患者，如果不佩戴眼镜，则会由于经常近距离看东西，会引起近视加深。而且长期不戴眼镜，会因为视功能得不到锻炼，而发生退化，形成弱视和斜视可能。
近视眼镜的作用
1.近视眼镜可以矫正视力。
近视眼因为远方的光线不能在视网膜上聚焦，造成视远物不清楚，而配戴了近视镜后，就可以获得清晰的物像，从而使视力得以矫正。
2.近视眼镜可以减轻视疲劳。
近视而不戴镜，必然导致眼睛极易疲劳，其结果只能是促使度数日益加深。正常戴镜后，视疲劳现象就会大大减少。
3.近视眼镜可以防治外斜。
近视眼视近时，眼的调节作用减弱，时间一长眼外直肌作用超过内直肌，就会引起眼位外斜。当然，近视伴外斜，仍可通过近视镜矫正。
4.近视眼镜可以防止眼球突出。
青少年由于眼球仍处于发育期，调节性近视极易发展为轴性近视。特别是高度近视眼，眼球前后径显著延长，外表即表现为眼球突出，如果近视一开始就正常戴镜矫正，这种情况就会有所减轻。
5.近视眼镜可以防止弱视。
高度近视而没有及时戴镜，往往造成屈光不正性弱视，只要配戴合适眼镜，经过较长时间治疗，视力大多会逐渐提高。
6.近视眼镜可预防高度近视引起的合并症
如视网膜脱落、玻璃体混浊、白内障、青光眼、眼球震颤等。
由此可见，近视了戴眼镜几乎就是不能避免的，不戴眼镜只会加重度数。这时候，近视眼睛就显得尤为重要，我们一定要选择正规的配镜店来为我们配眼镜。

爱大爱解析:近视的人数不断增加原因

五月 18, 2022

&nbsp; &nbsp; &nbsp; 目前全世界大概有20亿人佩戴眼镜，注意！是近视眼。中国是世界上近视发生概率最高的国家之一，全国近视人数占比总人口数的一半，也就是说，每2个中国人就有1个是近视。在我的学生时代，我模糊记得身边有6成同学都有佩戴眼镜的习惯。其中因为近视而佩戴的人也占其中的百分之四十。这么多人近视真的令人深思，到底是什么原因导致近视的呢？
认为，孩子近视也跟父母对他们的护眼工作没有做好有一定的关系，毕竟孩子的视力发育是非常重要的。如果孩子近视了，家长们该怎么做呢？

误区一：孩子视力不好，就是近视。
&nbsp; &nbsp; &nbsp; 有些家长一旦发现孩子的视力下降或是孩子在体检时被查出视力下降，就武断地认为这孩子是近视了，然后就到处寻医帮助孩子治疗近视。事实证明，这种做法是错误的。其实，能影响孩子视力的眼病有很多，除了近视以外，还有先天性眼底病变、远视、散光、斜视、弱视等。因此，家长们发现孩子的视力下降时，应先带孩子去正规的医院进行确诊。
误区二：近视后不戴眼镜能慢慢恢复。
&nbsp; &nbsp; &nbsp; 为了不影响孩子的学习，一些家长在孩子患了近视后，总是喜欢在寒、暑假期间为其治疗近视。许多不良商家也正是利用家长们的这种心理，声称某种方法或仪器可在一个月甚至十几天内治好近视，导致许多家长上当受骗，就目前来说，还没有一种仪器和设备能通过物理的方法彻底治疗近视或降低度数。反而，配戴合适的眼镜却能够控制和延缓度数增长。
误区三：戴眼镜早，以后近视会加深更快。
&nbsp; &nbsp; &nbsp; 不少父母认为“越早戴眼镜，以后度数就会越高”，因此拒绝为孩子配镜矫正，这样是不对的，应该根据验光的结果，结合视力要求、眼位、眼肌运动、是否有弱视等综合条件考虑配镜处方。在配镜之前，第一件事是接受验光，只有明确眼的屈光状态，才能给予合适的矫正。
误区四：成年后的近视患者视力不会再下降
&nbsp; &nbsp; &nbsp; 一般来说，成年人视力会逐渐趋于稳定，不过也并不是所有都会稳定。成人两三年内增长50度—75度都属于正常范畴，而成年人造成近视程度不断加深的原因通常有两种。
&nbsp; &nbsp; &nbsp; 1、一种是病理性近视。这是眼睛内部发生的病变，具有遗传性，表现为近视持续加深，并且增加急速。如果一年内近视度数增长超过100度，甚至200度以上，或是累计达到2000度的高度近视，就很可能是病理性近视。
&nbsp; &nbsp; &nbsp; 2、另一种是不注意用眼卫生，用眼习惯不正确。比如，照明过亮或过暗，用眼时间太长，出现倒睫、上睑下垂、角膜炎等眼疾不及时诊治等，都会加重近视。
误区五：近视是个小问题，不用在意。
&nbsp; &nbsp; &nbsp; 近视如果不合理的矫正，很容易变成高度近视，引起一系列如视网膜剥离、黄斑变性等严重并发症。近视还可能会影响孩子将来的专业选择。
误区六：孩子小，近视就是假性近视。
&nbsp; &nbsp; &nbsp; 很多孩子近视后，家长都怀疑自己孩子属于假性近视。其实，假性近视在青少年近视中的一个非常小的比例，它只是一种“近视”现象，本质上还不是近视眼。假性近视跟年龄大小关系不大。
专家表示：半数以上的弱视与斜视有关，从症状上来看，斜视为眼位异常，弱视是视力异常。如果父母发现孩子有斜视、屈光不正等症状，首先应带领孩子去医院进行全面检查。千万不可把弱视错当成近视，而耽误了治疗。
