【爱大爱眼睛保健】保护视力的方法

五月 18, 2022

&nbsp; &nbsp; &nbsp; 现在由于电子产品的普及,基本上每一个人每天面对着电子屏幕的时间都在加长,尤其是手机平板这些,很多人都喜欢睡前躺在床上玩,其实这样子对视力的伤害是非常大的,长久的话就会导致出现近视眼的情况,有些严重的还会伴随着头晕等症状,所以养成一个良好的习惯对保护视力非常有帮助。
&nbsp; &nbsp; &nbsp;&nbsp;另外现在中小学生开始忽略了看书写字时应该注意的正确姿势，导致现在越来越多的中小学生小小年纪就已经戴上眼镜了，有些度数深的镜片像啤酒瓶底一样厚，那么保护视力的方法有吗?

小编告诉你两个相对来说最容易的方法：
&nbsp; &nbsp; &nbsp;&nbsp;1.找一处10米以外的草地或绿树：绿色由于波长较短，成像在视网膜之前，促使眼部调节放松、眼睫状肌松弛，减轻眼疲劳。不要眯眼，也不要总眨眼，排除杂念、集中精力、全神贯注的凝视25秒，辨认草叶或树叶的轮廓。接着把左手掌略高于眼睛前方30厘米处，逐一从头到尾看清掌纹，大约5秒。看完掌纹后再凝视远方的草地或树叶25秒，然后再看掌纹。10分钟时间反复20次，一天做三回，视力下降厉害的要增加训练次数。
&nbsp; &nbsp; &nbsp; 2.或者选一安静场所，或坐或站，全身放松，清除杂念，二目睁开，头颈不动，独转眼球。先将眼睛凝视正下方，缓慢转至左方，再转至凝视正上方，至右方，最后回到凝视正下方，这样，先顺时针转9圈。再让眼睛由凝视下方，转至右方，至上方，至左方，再回到下方，这样，再逆时针方向转6圈。总共做4次。每次转动，眼球都应尽可能地达到极限。这种转眼法可以锻炼眼肌，改善营养，使眼灵活自如，炯炯有神。
&nbsp; &nbsp; &nbsp; 预防近视保护视力最有效的方法还是需要一个正确的读书写字的习惯，长时间的伏案工作了之后应该适当的眺望一下远处，这样子可以有效的帮助缓解眼疲劳，同时还能够帮助改善视力。在日常的生活中的时候也可以多吃一些富含维生素的食物，有利于提高视力。

寒暑假矫正孩子视力？远远不够！

五月 18, 2022

&nbsp; &nbsp; &nbsp; 家长都希望能利用假期时间为孩子检查、矫正视力，所以每年的寒暑假都是学生患流的高峰期，但保护视力单靠假期这个短暂的时间是远远不够的，家长应该从日常的学习行为进行干预，以保证孩子们的视力正常及眼部的健康。

&nbsp; &nbsp; &nbsp; 随着电脑的日益普及，给使用者造成狭小的工作环境，其辐射对人眼和大脑的危害程度尚难估计和预料，已引起有关学者的重视。使用电脑时间不应太长，用眼45分钟左右应休息10分钟左右并看远，使眼睛调节松弛。眼保健操是由教育部和卫生部定为在校学生的健眼操，可现在有很多学校都不做眼保健操了，但只要能够按照规定认真去做，相信对眼的健康有益无害。最起码做操期间是使眼睛休息，适当的按摩增加了眼局部的血液循环。
介绍，电脑族常有的症状为“眼干涩不舒，甚至干燥、畏光、视物模糊”，他们的症状并不是病理变化所引起，而是由于用眼不当所致。尤其是目不转睛的时候，害处最多。在神情专注的时候，眨眼次数减少，这就使眼表面的泪液蒸发过多，而不能得到及时的补充，久之则引起干眼的症状。这样的病人，早期检查结果往往查不出什么病，但长此以往，便可能失去健康的眼睛。对于青少年来说，长期近距离使用电脑会引起调节痉挛而看不清远处，继而造成近视等。
&nbsp; &nbsp; &nbsp; 专家提醒家长，要想让孩子预防或消除眼部疾病，保护视力，日常应注意对眼睛的呵护。首先，要定时休息。如每看书或使用电脑1小时要休息10-15分钟，此时可远眺窗外景观，不要集中在近距离用眼。其次，注意滋润眼睛，多眨眼。用电脑时，最好保持10至15度的向下视角，这样有助于减少眼球暴露的面积，以减少眼球表面水分蒸发。避免座位在空调出风口处，还可以在座位附近放置茶水，增加周围湿度。再次要经常锻炼。建议进行一些球类活动，如乒乓球、羽毛球等，当眼球追随目标时，睫状肌不断地放松与收缩，以及眼外肌的协同作用，可以提高眼的血液灌注量，促进眼部新陈代谢，从而减轻眼疲劳。
&nbsp; &nbsp; &nbsp; 由于现在学生们的压力很大，除了来自于学业上的，还有来自于学校、老师、家长、社会等多重压力，导致他们学习时间过长，用眼过度。一部分家长，他们很“心疼”孩子，认为孩子平时学习很辛苦，在假期就应该好好放松，导致学生在假期长时间使用电脑、看电视，虽然这是家长们的一番好意，却导致孩子过度用眼，从而发生眼部不适的症状。那么在此时，建议家长不要让孩子们长时间持续的中近距离用眼，比如长时间上网、看电视等，都可能导致眼睛疲劳、眼干、不能集中视力等症状的发生，甚至眼睛变红。尤其是大、中学生容易出现这些症状。应该注意：持续使用电脑时，要经常有意识的眨眨眼睛，防止眼睛干涩。
&nbsp; &nbsp; &nbsp;&nbsp;“学生们应该注意营养均衡，多吃坚果类食物。”爱大爱眼科专家建议，还要多吃富含钙、蛋白质的食物，避免偏食，控制甜食。饮食要有度，应注意质量、数量、进食时间和速度。在对何种食物对眼睛有益尚存在争议时，均衡营养切勿偏食是最保险的做法。还应保证起居有规律。平时应养成良好的生活习惯，早睡早起，每日保持充足睡眠时间，劳逸结合，脑体结合。

书桌和近视有哪些关系?

五月 18, 2022

小编为大家整理了一下“书桌和近视有哪些关系”，以便有需要的朋友有个参考，能更好的享受生活，下面就跟随爱大爱小编一起了解一下吧。
&nbsp; &nbsp; &nbsp; 即使经济不景气的时期，眼镜店永远不缺生意。一副普通镜框卖至上千已是常事，不禁令人担忧。而近视大军中，青少年更是主力军!根据相关研究机构的报告，有90%的青少年在生长发育阶段出现不同程度的脊柱侧弯、斜肩、驼背、近视等因坐姿不正产生的种种问题，我国中小学生近视、脊柱侧弯的发生率呈明显上升趋势。最近的调查显示，我国小学生近视眼发病率为22.78%，中学生为55.22%，更令人震惊的是，有份调查报告称，国内因高度近视致盲者已达30多成人。近视，脊柱侧弯以及心理问题已经构成青少年健康发育的障碍。

&nbsp; &nbsp; &nbsp;&nbsp;导致近视的诱因之一竟然是家中最不起眼的书桌。为何普通书桌成为青少年近视的“催化剂”呢?且听小编细细道来。
&nbsp; &nbsp; &nbsp; 1、传统书桌椅子的高度是固定的，孩子两脚悬空，双脚没有支点，上身很容易往前趴，另外，由于小腿重力缘故，大腿会与椅子座面产生挤压，大腿容易发麻，孩子就会无意识的经常动来动去。
&nbsp; &nbsp; &nbsp;&nbsp;2、传统书桌的桌沿是直的，孩子学习时两个肘关节会露在桌沿外面，小手臂会与桌沿产生挤压，小手臂容易酸痛，孩子就会本能的把两个肘关节放到桌面上，上身自然就会趴在桌面上。
&nbsp; &nbsp; &nbsp; 3、传统书桌的桌面是平的，孩子读写时被迫低头，导致孩子颈椎容易酸痛。
&nbsp; &nbsp; &nbsp; 4、传统椅子的坐面过大，孩子读写时腰背无法靠在椅背，容易引发腰酸背痛。
&nbsp; &nbsp; &nbsp; 知道了普通书桌的不足，在购买书桌时更应注意以下几点，避免青少年继续保持不正确的姿势。
&nbsp; &nbsp; &nbsp; 1、书桌椅的尺寸要与使用者的高度、年龄以及体型相结合。一般单人书桌的尺寸规格的表面在75cm130cm，其高度在75cm以下，双人书桌的表面的尺寸规格一般在75cm200cm，高度和单人书桌类似。假如不想频繁更换孩子的书桌，那就选择能调节高度的升降式儿童书桌吧，这样就可以随时根据孩子的成长进行调整，让他们的每一个学习阶段都舒舒适服的。
&nbsp; &nbsp; &nbsp; 2、选购书桌椅，安全性是首先要考虑的。书桌椅的线条应该圆滑流畅，圆形或弧形收边为最佳，另外还要有细腻的表面处理。购买的时候可以用力摇晃几下，结构松动、感觉摇摇摆摆的家具绝对不可要。
&nbsp; &nbsp; &nbsp; 3、用料要环保，表面的涂层应该具有不褪色和不易刮伤的特点，而且一定要选择使用塑料贴面或其他无害涂料的书桌椅。
&nbsp; &nbsp; &nbsp; 4、作为书房的一部分，书桌椅的选择要与房间搭调。选购书桌的时候，无论是材质还是风格，务必与其他家具相互统一，切不可因一件家具而产生一种突兀的感觉。

注意近视眼的三大“信号”【爱大爱眼睛保健】

五月 18, 2022

很多人都认为，近视是潜移默化慢慢的形成的，因此眼睛近视前应该没什么征兆。其实，近视眼的发生是有预兆、有信号的，下面小编就做一下介绍，希望引起注意。

近视眼的三大“信号”：
在发生眼疲劳的同时，许多人还伴有眼睛灼热、发痒、千涩、胀痈，重者疼痛向眼眶深部扩散，甚至引起偏头痛，亦可引起枕部、颈项肩背部的酸痛，这是由于眼部的感觉神经发生疲劳性知觉过敏所致。
有些高年级的小学生或中学生，看书时间一长，字迹就会重叠串行，抬头再看面前的物体，有若即若离、浮动不稳的感觉。有些人在望远久后再将视力移向近处物体，或望近久后再移向远处物体，眼前会出现短暂的模糊不清现象。
这些都是眼睛睫状肌调节失灵表现，是由眼疲劳所致。另外，有的少年儿童会反复发生霰粒肿、麦粒肿或睑缘炎．这些儿童视力虽然可达到1.0以上，其实已经“奏响”了近视眼的前奏曲。
原来成绩好的小朋友对学习会产生厌烦情绪，听课时注意力不够集中，反应也有些迟钝，脾气变得急躁，对原来喜爱的东西也缺乏兴趣，学习成绩下降。晚上睡眠时多梦、多汗，身体容易倦怠，且有眩晕、食欲不振等症状。这些变化也是即将发生近视的信息。
从以上分析可以看出近视前首先出现的并非视力下降，而是神经系统方面的症状。近视眼也并不单单是眼睛的问题，而是与全身变化息息相关。因此，当发现以上近视信号或先兆时，一定要引起注意，做好预防。（爱大爱,眼睛养护中心）

&nbsp; &nbsp; &nbsp;&nbsp;随着电脑、智能手机的广泛应用，干眼症在年轻人当中的发病率非常高，甚至因此来眼科就诊的小学生也不少见。“干眼症”顾名思义就是眼睛干涩。人的眼睛表面有一层“泪膜”，使眼睛表面保持湿润。正常情况下，眼睛会自动眨眼，每眨1次眼，就会重新形成一层泪膜，保证眼睛在泪膜的保护下，减少刺激。如果泪液分泌不足，不足以润滑角膜、结膜，眼球表面就会变得干燥，就会出现眼睛刺痛、怕光、烧灼感、异物感，有时还会出现刺激性流泪。“这种情况往往出现在长时间看电脑，看手机的人或职业驾驶员身上。”
&nbsp; &nbsp; &nbsp; “干眼症”是办公室的常见病，也叫蒸发过强型的干眼症。解释，干眼症的防治很大一部分是要讲究用眼卫生，特别是一些长期使用电脑的人员。首先要注意室内环境保持一定湿度、注意通风；另外，电脑使用的距离要适度（一般在60-70厘米）、室内亮度也不能过暗或过高，最重要的使用电脑时间，一般一次不宜超过一小时，建议用眼50分钟，要起身看看远处或闭眼休息10分钟。
&nbsp; &nbsp; &nbsp;&nbsp;对于青少年出现干眼症状，建议，要减少青少年看手机、玩游戏的时间，多参加球类活动，如乒乓球、羽毛球、足球等。当眼球追逐目标时，眼部睫状肌不断地放松与收缩，加上眼外肌的协同作用，可提高眼部的血液的灌注量，促进眼部新陈代谢，减轻眼睛疲劳和不适。
&nbsp; &nbsp; &nbsp; 有人说眼睛干涩，回去点眼药效果也不好。提醒大家，眼药可不是随便点的。平时在药店能买到的眼药主要是消炎眼液和缓解视疲劳等药物。但干眼症需要补充的是人工泪液。这就需要到医院眼科就诊，通过医生检查确定是否为干眼症，及干眼症的程度，来确定应用哪种人工泪液较为合适。

【爱大爱】隐形眼镜的常见感染问题

五月 18, 2022

&nbsp; &nbsp; &nbsp; 我们的眼角膜非常脆弱，而隐形眼镜离眼角膜太近，所以很难避免其对眼睛造成的伤害。戴隐形眼镜的人眼睛感染的风险更高，持续戴隐形眼镜（白天、晚上或者游泳洗澡）更是会增加这种风险。虽然大多数细菌感染可以用口服或局部涂抹抗生素来治疗好，但由于个体差异，科学家并不十分确定不同的感染情况都需要多长时间才开始对眼睛造成伤害，所以最好的办法是一旦感觉眼睛不对劲，就应立即停止佩戴隐形眼镜并就医。
&nbsp; &nbsp; &nbsp;&nbsp;隐形眼镜其实还算安全，但如果不合理佩戴隐形眼镜就会引发一系列眼部感染问题，部分感染甚至会导致包括失明在内的严重后果。这与几个风险因素有关，如戴隐形眼镜的时间太长，会阻碍眼睛“呼吸”。眼角膜也需要呼吸，即使是高透氧性的镜片覆盖在角膜上也会阻碍眼睛“吸入”氧气。长时间佩戴隐形眼镜会导致角膜缺氧，如果此时恰好有微生物存在，就有可能引发角膜炎，甚至角膜溃疡，后者是佩戴隐形眼镜的严重并发症之一。

提醒：如不能正确清洁，或没有按照眼科医生的指导及时更换隐形眼镜，也会带来眼部感染的风险。美国疾病控制和预防中心的数据显示，40%至90%的隐形眼镜佩戴者没有按照护理说明清洁和佩戴隐形眼镜。不遵守卫生习惯的隐形眼镜佩戴者，其隐形眼镜病例中的细菌污染会增加。66%的隐形眼镜病例细菌或真菌污染检测呈阳性。在近40%的污染病例中，有多种微生物被分离出来。所以用肥皂洗手、让隐形眼镜盒自然风干、使用隐形眼镜盒和消毒液都是减少隐形眼镜污染的关键步骤。
&nbsp; &nbsp; &nbsp;&nbsp;虽然大多数人可以通过简单的预防措施安全地佩戴隐形眼镜，但冒着风险夜间佩戴或在游泳洗澡时佩戴隐形眼镜，其实就像坐在一辆没有安全带的汽车里，隐患会一直存在。隐形眼镜很好，但必须记住它仍然是塑料的，人类不需要它每时每刻都待在我们的眼睛里。

【爱大爱解析】眼睛发育的黄金时期

五月 18, 2022

&nbsp; &nbsp; &nbsp; 眼球的发育过程是由小变大的，眼球的前后直径也是由短变长的。从光学原理上说，人在出生的时候是高度远视。随着年龄增长，眼球在变大，眼轴在不断变长。正常人的视力，到18～20岁，会逐渐从远视状态变成正视状态。

专家介绍，眼睛有两个重要的快速发展阶段。第一阶段是3～4岁，第二阶段是青春期，12～15岁。这两个阶段被称为视力发育的“黄金期”。成人视力的好坏，与这两个“黄金期”有密切关系。眼睛在发育中，年龄越小，发展的速度越快。眼睛能否健康发育与所受到的刺激成正比，刺激越小发育越好，反之亦然。
&nbsp; &nbsp; &nbsp;&nbsp;爱大爱专家指出，近视的成因主要在于近距离长时间的刺激会加速眼睛从远视状态向正视状态改变。在第一个“黄金期”，孩子3～4岁时，尽管处于远视阶段，但如果给予长时间近距离的刺激，如看电视、玩电子游戏、看书、识字等，就必然会加快眼球变长的速度。同样，孩子在第二个“黄金期”也就是青春期，过重的课业负担、过度使用眼睛，也会加快眼轴的变长。
&nbsp; &nbsp; &nbsp; 这样使得本来应该18～20岁时发育成正视的眼睛，被迫提前几年或十几年变成正视。但此时眼球还未发育成熟，它在成长中不断拉长眼轴，逐渐形成近视。如果孩子在第一个“黄金期”已经发展到正视或轻度近视，那么在第二个“黄金期”近视的度数就会快速发展，从而形成高度近视。
&nbsp; &nbsp; &nbsp;&nbsp;养成良好的爱眼卫生习惯，应该注意以下几点：在第一个“黄金期”尽量不要让孩子读书识字，多带孩子参加户外活动，参加唱歌、跳舞、游泳之类的体能学习班。在第二个“黄金期”，建议患近视的孩子看黑板时戴眼镜，看书写字时不要戴着眼镜。

有时候眼睛胀痛什么原因

五月 18, 2022

表示，一些不科学的用眼可以导致眼疲劳的出现，另外还有一些疾病也可以出现眼疲劳。下面我们就一起来了解一下导致眼疲劳的几种疾病。
1、眼发育异常
&nbsp; &nbsp; &nbsp; 如两眼瞳距较大，两眼集合困难，易产生眼疲劳。
2、近视眼
&nbsp; &nbsp; &nbsp; 近视眼未得到矫正时，由于阅读距离太近而引起过度集合，过度集合又可伴随过度调节，使近视程度增加，阅读距离更近。如此发生恶性循环，以致产生眼疲劳。

3、眼睛疾病
患有角膜云翳、晶状体混浊以及其他眼疾引起的视物不清，也易引起眼疲劳。不宜眼外肌疾患引直的平衡失调，注视目标时，双眼单视困难，造成眼疲劳。
&nbsp; &nbsp; &nbsp; 此外，除了这些疾病可以导致眼疲劳外，还有一些其他的因素也可以导致眼疲劳，比如年龄因素，或者是体质问题，比如缺乏锻炼、营养不良、经常失眠、生活没有规律、烟酒过度、不注意用眼卫生等，均容易发生眼疲劳。

爱大爱解析：为何视力会下降的原因

五月 18, 2022

据调查发现，除了先天遗传和外伤导致视力下降、丧失以外，疾病也是影响视力的一大因素，比如青光眼就是其中的一种。
&nbsp; &nbsp; &nbsp; 它导致的视力损伤是不可逆转的，且早期并不容易被发现，所以给患者造成了极大伤害。由于青光眼造成的视力损伤不可逆转，所以它的影响超过了白内障，占据了第一的位置。且这并不属于老年疾病，无论你处于哪一年龄段都有可能患病。
如果出现以下四个症状，那很可能是青光眼在偷走你的视力！
1、视力下降明显
&nbsp; &nbsp; &nbsp; 如果发现近期近视度数在逐渐上涨，眼光视力矫正不到1.0，特别是高度近视患者，即便是佩戴舒适的眼镜，仍然有一定头痛、眼胀痛感传来，就需要提高警惕，很可能是青光眼到来的信号。由于高度近视患者的巩膜边长，弹性也相对下降，所以受疾病影响出现高眼压后，患者无法感知症状，或疏忽了细微变化，但你的视力却在悄悄溜走。
2、面部不适
&nbsp; &nbsp; &nbsp;&nbsp;患者早早上起床后看书或对着电脑时，明显感觉到眼部吃力，容易出现鼻梁根部酸痛和眼眶前额胀痛问题。这和眼压变化同样有关，正常人的眼压有昼夜波动规律，通常清晨眼压偏高，但到了夜间之后眼压就会逐渐平稳。而青光眼患者24小时的眼压波动浮动更大，早上起床后眼压过高，从而出现了不适症状。
3、闭角型青光眼
&nbsp; &nbsp; &nbsp; 如果在情绪波动过大或者是暗处停留时间过长，比如看电影、暗室工作之后，突然出现眼睛胀痛、头痛、视物模糊，且总感觉眼前有云雾缭绕感的话，这就是典型闭角型青光眼症状。如果多次反复出现，但却没有重视的话，那就可能会进入急性大发作期，给患者视力造成更大的伤害。
4、无法对着灯光
&nbsp; &nbsp; &nbsp; 如果在夜间看灯光时，明显感觉灯光下出现了五彩缤纷的晕圈，就如同天上的彩虹一般，这就是典型的“虹视”症状。同样和眼压上升有关，由于眼压过高，导致角膜水肿，从而造成了角膜折光改变，引发彩虹光圈出现。
&nbsp; &nbsp; &nbsp; &nbsp;以上四点都和青光眼疾病有关，如果不重视起来的话，视力就会越来越差，甚至有一定失明风险，最主要的是青光眼造成的视力损伤无法恢复，所以一旦出现以上表现，及时就医很有必要。

【爱大爱眼睛养护】为什么“小眼镜”会越来越多

五月 18, 2022

&nbsp; &nbsp; &nbsp; 眼睛是心灵的窗户，这足以说明眼睛的重要性。然而现在我们身边的“小眼镜”却越来越多，视力不良已直接影响到青少年的身心健康和未来职业的选择，应引起学校、家长乃至全社会的共同关注。
“小眼镜”越来越多
专家表示，五六年级学生是近视眼的“重灾区”，近视的提前发生跟“独生子女”这一社会现象密切相关。
&nbsp; &nbsp; &nbsp; “现在的孩子不是说他们读书有多用功才把视力弄坏的，我相信更多患近视眼的学生都是平常打游戏所致。现在的互联网这么发达，这种情况不单单出现在一些都市孩子中，在农村也很普遍，我所教的班级就有很多‘小四眼’，而且数量每学期都在增加。”这是爱大爱与该学校的一位班级主任交谈的说法。

多种因素影响孩子视力
&nbsp; &nbsp; &nbsp;&nbsp;近视眼形成的原因是很复杂的。学生的升学压力越来越大，导致学习负担不断加重;电视、游戏机和计算机在学生生活中的出现等因素，都是导致学生近视的原因。
&nbsp; &nbsp; &nbsp; 爱大爱眼科专家指出，前去配镜的“小近视”中，每天接触电视、电脑的时间普遍在两三个小时以上，而这一“用眼不卫生”问题的背后，是独生子女的孤独感。
&nbsp; &nbsp; &nbsp; 目前大部分家庭仅有一个孩子，父母上班后，孩子在家只能通过看电视、用电脑等方式自娱自乐。此外，孩子独自在家时，长时间用手机、看漫画等习惯，也造成用眼过度，直接引起近视。
预防近视迫在眉睫
那么，该怎样预防孩子患上近视呢?
&nbsp; &nbsp; &nbsp; 专家认为，家长应该让孩子多出去活动，而不应该长时间地看书或看电视、玩电脑等；台灯的配备很重要，家长应给孩子购买40瓦可调节的白炽台灯，灯罩为磨砂玻璃，台灯光线顺着灯罩斜射到孩子的鼻子底下。书、作业本的纸白些时，灯光就调暗些，纸黄些即可调亮些，以不刺眼为好。另外注意孩子的坐姿，眼睛离书面的距离为33厘米。
&nbsp; &nbsp; &nbsp;&nbsp;另外，眼保健操是预防近视的一个很好的手段，做眼保健操应成为孩子们自觉、自动的行为，家长最好能在晚上再督促孩子认真做一次眼保健操。
