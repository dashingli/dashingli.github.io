三个方法控制和减缓青少年近视的发展

七月 28, 2020 本文总阅读量次

&nbsp; &nbsp; &nbsp; 近视孩子家长，最怕的就是检查时孩子近视的度数又增长了！爱大爱根据调查数据表示，孩子近视后，每年度数会呈75-150度增长，更有甚者增长200-300度，这无疑给孩子眼睛的健康带来非常大的隐患。那么排除严重不健康用眼情况，近视度数基本在几岁左右能稳定下来呢？
&nbsp; &nbsp; &nbsp; 在很多人的印象里，似乎小孩子的度数会蹭蹭地往上涨，不管是防控做得再好，也会出现缓慢增长的趋势，但到了成年后就不会有太大的变化。
&nbsp; &nbsp; &nbsp; 谁说成年人的近视度数不会涨? 所谓成年，是指几岁？18岁是法律上对成年人的定义，但是调查研究显示，大部分人到了18岁度数还在增长，一般到了22岁左右才基本能稳定下来，当然这里是指部分近视患者，不排除特殊情况患者；也不是指眼睛度数绝对性不会有变化。
&nbsp; &nbsp; &nbsp; 在特殊情况中，尤其是高度近视患者，可能一辈子度数都在涨。这是因为近视的成因复杂，除了基因，环境、用眼习惯等各种因素都掺杂其中，而高度近视的成因就更加复杂了。因此，定时眼健康体检是十分必要的。
&nbsp; &nbsp; &nbsp; 预防孩子近视必须上升到我们的日常生活，视力保健工作从小就应该做起。而针对已经近视的孩子来说，仅仅只是戴眼镜可能无法避免孩子将来成为高度近视的命运。这时，有效的防控，就显得非常重要了。
以下“三个方法”是目前可以有效控制和减缓青少年近视的发展！！！

  1. 户外活动，是目前预防近视最有效方法之一
    &nbsp; &nbsp; &nbsp; 户外活动是一项十分经济有效的近视防控手段，研究表明每周10小时-14小时的户外活动时间能够有效预防近视发生，对于已经近视的孩子也能延缓近视发展。然而，升学压力与课业负担正在压缩孩子们的户外活动时间，需要家庭、学校、社会共同努力。
  2. 科学使用低浓度阿托品，可有效防控近视
    &nbsp; &nbsp; &nbsp; 专家提醒，低浓度阿托品并不是对每个孩子都有“特效”，研究显示它对大约10%-15%的人无明显控制近视的作用。阿托品发挥防控近视的作用是需要长期规律使用的，它可以直接作用于视网膜和巩膜以延缓眼轴增长，但它同样具有扩瞳和睫状肌麻痹作用，因此有的小朋友会因为畏光和看近模糊等副作用不能坚持治疗。专家强调，请务必在正规医院专业医生的指导下使用低浓度阿托品，使用前需要进行眼科检查以确认是否适用。
  3. 规范使用角膜塑形镜，可有效干预近视增长
    &nbsp; &nbsp; &nbsp; 角膜塑形镜(也称OK镜)是具有控制近视进展作用的一种特殊类型的隐形眼镜。原理是睡前佩戴，通过夜间佩戴至少8小时，将角膜压平，改变眼球屈光度，早晨起床摘下，患者白天无需再佩戴近视眼镜。因角膜具有一定记忆性，患者只需夜间戴镜，白天无需戴镜也可获得清晰视觉。适用于年龄大于8岁、近视度数增长过快（>0.5D/年）的青少年，屈光度在-1.25D~-4.00D的患者效果更佳。
